1. 命题“$\forall x \in \mathbb{R}$，有 $3x^2 + 4x + 1 \leq 0$”的否定是（ ）
A. $\forall x \in \mathbb{R}$，有 $3x^2 + 4x + 1 > 0$
B. $\exists x \in \mathbb{R}$，有 $3x^2 + 4x + 1 \leq 0$
C. $\exists x \in \mathbb{R}$，有 $3x^2 + 4x + 1 > 0$
D. $\forall x \in \mathbb{R}$，有 $3x^2 + 4x + 1 \geq 0$
【答案：C】
A. $\forall x \in \mathbb{R}$，有 $3x^2 + 4x + 1 > 0$  
B. $\exists x \in \mathbb{R}$，有 $3x^2 + 4x + 1 \leq 0$  
C. $\exists x \in \mathbb{R}$，有 $3x^2 + 4x + 1 > 0$  
D. $\forall x \in \mathbb{R}$，有 $3x^2 + 4x + 1 \geq 0$

2. 设 $x \in \mathbb{R}$，则“$0 < x < 1$”是“$0 < x < 3$”的（ ）
A. 充分不必要条件  
B. 必要不充分条件  
C. 充要条件  
D. 既不充分也不必要条件
【答案：C】
A. 充分不必要条件  
B. 必要不充分条件  
C. 充要条件  
D. 既不充分也不必要条件

3. 下列各组函数是同一个函数的是（ ）
A. $f(x) = \sqrt{x+1} \cdot \sqrt{x-1}$ 与 $g(x) = \sqrt{x^2 - 1}$  
B. $f(x) = \sqrt{x^2}$ 与 $g(x) = (\sqrt{x})^2$  
C. $f(x) = \sqrt{(x+10)^2}$ 与 $g(x) = x + 10$  
D. $f(x) = \dfrac{x^3 + x}{x^2 + 1}$ 与 $g(x) = x$
【答案：D】
A. $f(x) = \sqrt{x+1} \cdot \sqrt{x-1}$ 与 $g(x) = \sqrt{x^2 - 1}$  
B. $f(x) = \sqrt{x^2}$ 与 $g(x) = (\sqrt{x})^2$  
C. $f(x) = \sqrt{(x+10)^2}$ 与 $g(x) = x + 10$  
D. $f(x) = \dfrac{x^3 + x}{x^2 + 1}$ 与 $g(x) = x$

4. 函数 $f(x) = \sqrt{x+4} + \dfrac{2}{x-1}$ 的定义域是（ ）
A. $\{x | x \geq -4\}$  
B. $\{x | x \neq 1\}$  
C. $\{x | x \geq -4 \text{ 且 } x \neq 1\}$  
D. $\{x | x > -4 \text{ 且 } x \neq 1\}$
【答案：C】
A. $\{x | x \geq -4\}$  
B. $\{x | x \neq 1\}$  
C. $\{x | x \geq -4 \text{ 且 } x \neq 1\}$  
D. $\{x | x > -4 \text{ 且 } x \neq 1\}$
