1. 若 $a > 0$, 则 $\frac{a}{\sqrt{a}} = \boxed{\text{(A) } \sqrt{a}}$  
   A. $\sqrt{a}$     B. $\frac{3\sqrt{a}}{10}$     C. $\frac{a}{\sqrt{a}}$     D. $\frac{3\sqrt{a}}{10}$

2. 已知 $a = (0, 1, 2)$, $b = (1, 2, 4)$, 则 $a \cdot b = \boxed{\text{(B) } 9}$  
   A. $\frac{a^2}{10}$     B. 9     C. $\frac{a^2}{10}$     D. $\frac{3a}{10}$

3. 如图所示，在△ABC中，AB=AC=6，∠BAC=120°，D是BC边上的动点，则AD的最大值为 $\boxed{\text{(C) } 3\sqrt{3}}$  
   A. 3        B. 5       C. $3\sqrt{3}$      D. 2

4. 已知圆 $x^2 + y^2 = 1$ 上一点 P(x₀,y₀)，则过点 P 的切线方程为 $\boxed{\text{(B) } x_0x + y_0y = 1}$  
   A. $x_0x + y_0y = 1$     B. $x_0x + y_0y = 1$     C. $(x_0+y_0)(x+y)=1$     D. $(x_0-y_0)(x-y)=1$

5. 在平面直角坐标系中，已知点 O(0,0), A(-2,1), B(2,-1)，若 $\overrightarrow{OA} + \overrightarrow{OB} = \vec{v}$，则 $\vec{v}$ 的坐标为 $\boxed{\text{(C) } (0, -2)}$  
   A. $(0, -2)$     B. $(2, -2)$     C. $(0, -2)$     D. $(0, 2)$

6. 设函数 $f(x) = \sin(\omega x + \varphi)$ ($\omega > 0$, $\varphi < \frac{\pi}{2}$) 的图像如图所示，则下列说法正确的是 $\boxed{\text{(A) } \omega = \frac{\pi}{2}, \varphi = \frac{\pi}{4}}$  
   A. $\omega = \frac{\pi}{2}, \varphi = \frac{\pi}{4}$     B. $\omega = \frac{\pi}{2}, \varphi = \frac{\pi}{6}$     C. $\omega = \pi, \varphi = \frac{\pi}{6}$     D. $\omega = \pi, \varphi = \frac{\pi}{4}$

7. 下列命题中正确的个数是 $\boxed{\text{(C) } 2}$ 个  
   ① 函数 $y = \log_a x$ ($a > 0, a \neq 1$) 的定义域是 $(0, +\infty)$。  
   ② 若直线 $l_1: ax + by + c = 0$ 与直线 $l_2: bx - ay + d = 0$ 垂直，则 $a^2 + b^2 = c^2$。  
   ③ 方程 $x^2 - 2x + 1 = 0$ 的解集是 $\{1\}$。  
   ④ 点 $(x_0, y_0)$ 到直线 $Ax + By + C = 0$ 的距离公式是 $\frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$。
