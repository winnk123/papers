1. 若集合 M = {x | 0 ≤ 4 - 2x < 4}，N = {x | x - 3 < -2}，则 M ∪ N = 
A. {x | x ≤ 2}
B. {x | 1 < x ≤ 2}
C. {x | 0 < x < 1}
D. {x | x > 0}

【答案：C】

2. 命题“∃x ∈ R, x² - 5 = -6”的否定是 D
A. ∃x ∈ R, x² - 5 ≠ -6
B. ∃x ∉ R, x² - 5 = -6
C. ∀x ∈ R, x² - 5 > -6
D. ∀x ∈ R, x² - 5 ≠ -6

【答案：D】

3. 已知正数 a, b 满足 a² + b² = 32，则 a + b 的最大值为 A
A. 4
B. 6
C. 8
D. 10

【答案：A】

4. 某投资方对某项目提出两个投资方案：方案一为一次性投资 1000 万元；方案二为第一年投资 200 万元，以后每年投资 30 万元。下列不等式表示“经过 n 年后，方案一的总投资不多于方案二的总投资”的是 
A. 30n + 170 ≤ 1000
B. 30n + 170 ≥ 1000
C. 30n + 200 ≥ 1000
D. 30n + 170 > 1000

【答案：C】

A. M = N
B. M > N
C. M < N
D. M, N 的大小关系不确定

【答案: D】
