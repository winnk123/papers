12. 已知集合 $A = \{-2, 0, 2, 4\}, B = \{x | x - 3 \leq m\}$，若不等式 $4x - 3y$ 的取值范围为 $-10 \leq 4x - 3y \leq 4$，则实数 $m$ 的取值范围是 ( )  
【答案：C】  
A. $\{-2, 0, 2, 4\}$     B. $\{x | x - 3 \leq m\}$     C. $\{x | x \geq -3\}$   D. $\{x | x \leq 3\}$
