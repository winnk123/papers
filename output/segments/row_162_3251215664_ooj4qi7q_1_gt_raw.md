2. 命题“∀ x ∈ R 都有 x² + x + 1 > 0”的否定是()
A. 不存在 x ∈ R, x² + x + 1 > 0
B. 存在 x₀ ∈ R, x₀² + x₀ + 1 ≤ 0
C. 存在 x₀ ∈ R, x₀² + x₀ + 1 > 0
D. 对任意的 x ∈ R, x² + x + 1 ≤ 0

【答案：B】

4. 已知集合 A = {x | (2 - x)/x ≥ 0}, 集合 B = {x | |x² - 1| < 1}, 则 A ∪ B = ()
