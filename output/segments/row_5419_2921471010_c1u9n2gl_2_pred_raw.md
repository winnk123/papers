13. 为测量某塔的高度，在塔的水平地面上共取三点 A, B, C，测得塔顶 P 的仰角分别为 $30^\circ$, $60^\circ$, $45^\circ$，且 $AB = BC = 50\,\text{m}$。则塔高 $OP = \frac{100\sqrt{3}}{3}\,\text{m}$。

14. 设 $y = m(x > 0)$ 与 $f(x) = \frac{1}{2}\sin(2x + \varphi)$ 图象的相邻两个交点之间的距离为 $\pi$，则 $m = 1$。

15. (本题满分 15 分) 已知 $\vec{a} = (\cos\alpha, \sin\alpha), \vec{b} = (\cos\beta, \sin\beta), 0 < \beta < \alpha < \pi$。
   (I) 若 $\vec{a} \cdot \vec{b} = \frac{\sqrt{3}}{2}, \vec{a} - \vec{b} = \left(\frac{\sqrt{3}}{2}, \frac{1}{2}\right)$，求 $\alpha, \beta$。
   (II) 若 $\vec{a} \cdot \vec{b} = \frac{1}{2}$，求 $\alpha - \beta$ 的最大值。

16. (本题满分 15 分) 如图，已知 △ABC 内角 A, B, C 所对的边分别为 a, b, c，△ABC 的面积为 $S = \frac{1}{2}bc\sin A$。
   (I) 求 $\tan B$ 的最小值。
   (II) 若 △ABC 的内角平分线 AD 与边 BC 相交于点 D，BD = $\frac{5}{7}a$, CD = $\frac{2}{7}a$，求 △ABC 的周长。

   (I) 求 $\frac{a^2 - c^2}{b^2}$ 的取值范围。
   (II) 求 $\frac{\sin A - \sin C}{\sin B}$ 的取值范围。
