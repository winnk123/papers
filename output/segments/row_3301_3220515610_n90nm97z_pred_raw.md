11. 已知 $x > 0, y > 0, x + y + xy = 3$，则 ( )  
A. $x + y \geq 2$  
B. $xy \leq 1$  
C. $x^2 + y^2 \geq 4$  
D. $\frac{1}{x} + \frac{1}{y} \geq \frac{4}{3}$  

【答案：A】

12. 若命题“$\forall x \in \mathbb{R}$, 都有 $mx^2 + nx + p > 0$”为真，则实数 $m$ 的取值范围是 ( )  
A. $m > 0$ 且判别式 $\Delta < 0$  
B. $m > 0$ 且判别式 $\Delta \leq 0$  
C. $m < 0$ 且判别式 $\Delta < 0$  
D. $m < 0$ 且判别式 $\Delta \leq 0$  

【答案：B】

14. 已知正数 $a, b$ 满足 $ab = a + b + 1$，则 $\frac{1}{a} + \frac{1}{b}$ 的最小值为 ( )  
A. $-2$  
B. $-\frac{1}{2}$  
C. $2$

【答案：C】
