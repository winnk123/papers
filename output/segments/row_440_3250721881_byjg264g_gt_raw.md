一、选择题

1. 已知集合 $U=\{1,2,3,4,5\}$, $A=\{1,3\}$, $B=\{1,2,4\}$, 则 $C_U B \cup A$ = ( )  
A. $\{1,3,5\}$     B. $\{1,3\}$     C. $\{1,2,4\}$     D. $\{1,2,4,5\}$  
【答案：A】

2. 命题“$\forall x \in \mathbb{R}$ 都有 $x^2 + x + 1 > 0$”的否定是 ( )  
A. 不存在 $x \in \mathbb{R}, x^2 + x + 1 > 0$  
B. 存在 $x_0 \in \mathbb{R}, x_0^2 + x_0 + 1 \leq 0$  
C. 存在 $x_0 \in \mathbb{R}, x_0^2 + x_0 + 1 > 0$  
D. 对任意的 $x \in \mathbb{R}, x^2 + x + 1 \leq 0$  
【答案：B】

3. 设 $U=\{1,2,3,4,5,6\}$, 集合 $A=\{1,2,4,6\}$, 集合 $B=\{2,3,5\}$, 则 Venn 图中阴影部分表示的集合的真子集个数是 ( )  
A. 1     B. 2     C. 3     D. 4  
![插图](../0fad217a-a6c9-45ff-86ea-f92eddf32a92.jpg)
【答案：C】

4. 已知集合 $A=\left\{x \mid \frac{2-x}{x} \geq 0\right\}$, 集合 $B=\{x \mid |x^2 - 1| < 1\}$, 则 $A \cup B$ = ( )  
A. $\{x \mid -\sqrt{2} < x < 0 \text{ 或 } 0 < x \leq 2\}$  
B. $\{x \mid -\sqrt{2} < x \leq 2\}$  
C. $\{x \mid -\sqrt{2} < x < 0 \text{ 或 } 0 < x < 2\}$  
D. $\{x \mid 0 < x < \sqrt{2}\}$  
【答案：B】

5. 若命题“$\exists x \in \mathbb{R}, ax^2 - ax + 1 \leq 0$”是假命题，则实数 $a$ 的取值范围为 ( )  
A. $\{a \mid 0 \leq a < 4\}$     B. $\{a \mid 0 < a < 4\}$     C. $\{a \mid 0 \leq a \leq 4\}$     D. $\{a \mid a \leq 0 \text{ 或 } a > 4\}$  
【答案：C】

6. 已知集合 $A=\{x \mid -1 < x \le 4\}$, $B=\{x \mid a-1 \leq x \leq a+2\}$, 若集合 $A \cap B$ 中恰好只有两个整数，则实数 $a$ 的取值范围是 ( )  
A. $[-1,0) \cup (2,3]$     B. $(-1,0) \cup (2,3)$     C. $(-2,-1] \cup [3,4)$     D. $(-2,-1) \cup (3,4)$  
【答案：D】

7. 已知 $2 < x < 4$, $-3 < y < -1$, 则 $\frac{x}{x-2y}$ 的取值范围是 ( )  
A. $\left\{x \mid \frac{1}{10} < x < \frac{1}{4}\right\}$     B. $\left\{x \mid \frac{2}{3} < x < 2\right\}$     C. $\left\{x \mid \frac{1}{5} < x < 1\right\}$     D. $\left\{x \mid \frac{1}{4} < x < \frac{2}{3}\right\}$  
【答案：D】

8. 定义：(1) $\min\{a,b\}$ 表示 $a,b$ 中的最小值；(2) $\lfloor x \rfloor$ 表示不超过 $x$ 的最大整数。设 $a,b,c$ 为实数，则 $\min\left\{\left\lfloor \frac{a+b}{c} \right\rfloor + \left\lfloor \frac{b+c}{a} \right\rfloor + \left\lfloor \frac{c+a}{b} \right\rfloor\right\}$ = ( )  
A. 0     B. 2     C. 3     D. 4  
【答案：D】

11. 已知集合 $A=\{x \mid x=3m, m \in \mathbb{N}^*\}$, $B=\{x \mid x=3m+1, m \in \mathbb{N}\}$, $C=\{x \mid x=3m+2, m \in \mathbb{N}\}$，若 $a \in A$, $b \in B$, $c \in C$，则下列结论中不可能成立的是 ( )  
A. $2025 = a + b + c$     B. $2025 = abc$     C. $2025 = a+bc$     D. $2025 = a(b-c)$

【答案：$\pm 1$】
