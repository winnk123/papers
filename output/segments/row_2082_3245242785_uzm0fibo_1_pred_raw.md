[条件缺失]

1. 设集合 $A=\{x\in \mathbb{N}|-2<x<2\}, B=\{-1,0,1,2\}$，则 $A\cap B=$ ( )
A. $\{0, 1\}$   B. $(0, 2)$   C. $\{0, 1\}$   D. $\{0, 1, 2\}$
【答案：C】
A. $\{0, 1\}$   B. $(0, 2)$   C. $\{0, 1\}$   D. $\{0, 1, 2\}$

2. 若命题“$\exists x<2023, x^2>0$”为假命题，则实数 $x$ 的取值范围是 ( )
A. $[2023, +\infty)$   B. $(2023, +\infty)$   C. $(-\infty, 2023)$   D. $(-\infty, 2023]$
【答案：D】

3. 若命题 $p: \forall x\in \mathbb{R}, \frac{1}{x-2}<0$，则 $p$ 表述准确的是 ( )
A. $\exists x\in \mathbb{R}, \frac{1}{x-2}\geq 0$ 或 $x=2$   B. $\forall x\in \mathbb{R}, \frac{1}{x-2}\geq 0$ 或 $x=2$
C. $\exists x\in \mathbb{R}, \frac{1}{x-2}>0$ 或 $x=2$   D. $\forall x\in \mathbb{R}, \frac{1}{x-2}>0$ 或 $x=2$
【答案：C】

4. 已知集合 $A=\{x|ax^2+ax+2=0\}$，若集合 $A$ 为空集，则实数 $a$ 的取值范围是 ( )
A. $\{a|0<a<8\}$   B. $\{a|0\leq a<8\}$   C. $\{a|a>8\}$   D. $\{a|a\leq 0$ 或 $a>8\}$
【答案：B】

5. 数 $a, b$ 满足 $0<b<a<8$，则下列不等式一定成立的是 ( )
A. $a+b<ab+1$   B. $ab<b^2$   C. $a+b>ab+1$   D. $a+b<ab$
【答案：C】

6. 不等式 $x^2-(m+3)x+3m<0$ 的解集中恰有 3 个正整数，则实数 $m$ 的取值范围为 ( )
A. $(3,4)$   B. $(3,4]$   C. $(5,6]$   D. $(6,7]$
【答案：C】

7. 若不等式 $x^2+bx+c<0$ 的解集为 $\{x|-1<x<3\}$，则下列结论正确的是 ( )
A. $c>0$   B. $b>0$   C. $c<0$   D. $b<0$
【答案：A】

8. 若集合 $A=\left\{x|\frac{x+2}{x+1}\geq 0\right\}, B=\{x||ax+1|\leq 0\}$，若 $B\subseteq A$，则实数 $a$ 的取值范围是 ( )
A. $[-\frac{1}{3}, 0]$   B. $[-\frac{1}{3}, 0)\cup(0,+\infty)$   C. $[-\frac{1}{3}, 0)$   D. $[-\frac{1}{3}, 0]\cup(0,1)$
【答案：C】

9. 下列结论正确的有 ( )
A. 当 $x<\frac{4}{3}$ 时，$\sqrt{x}+\frac{1}{\sqrt{x}}\geq 2$   B. 当 $x>2$ 时，$z+\frac{1}{z}\geq 2$
C. 当 $x>0$ 时，$\sqrt{x}+\frac{1}{\sqrt{x}}\geq 2$   D. 当 $x>0$ 时，$z+\frac{1}{z}\geq 2$
【答案：C】

10. 图中阴影部分所表示的集合是 ( )

A. $N\cap C_U M$   B. $M\cap C_U N$   C. $C_U(M\cap N)$   D. $(C_U M)\cap(C_U N)$
【答案：C】

11. 下列命题为假命题的是 ( )
A. “若 $A\cap B\neq\emptyset$，则 $A\subseteq B$” 是必要不充分条件
B. 若 $P:\{y|x^2+y^2=1\}, Q:\{y|x^2+y^2=4\}$，则 $P\subseteq Q$
C. 方程 $x^2+x+a=0$ 有两个负根的充要条件是 $a>0$
D. 已知集合 $A=\{x|x^2-4<0$ 且 $x\in \mathbb{N}\}$，则集合 $A$ 的真子集数是 7
【答案：C】
