8. 已知关于 $x$ 的方程 $x^2 + 2(m-2)x + m^2 - 3m + 3 = 0$ 有两个不相等的实数根 $x_1, x_2$，则 $\frac{mx_1}{x_1-1} + \frac{mx_2}{x_2-1}$ 的最大值为 ( )  
A. $2\sqrt{2}$     B. $-2\sqrt{2}$   C. 2      D. $-4$  
【答案：C】  

9. 已知实数 $a > 0, b > 0$，则下列说法正确的是 ( )  
A. 若 $a > b$，则 $a^2 > b^2$  
B. 若 $a > b$，则 $\frac{1}{a} < \frac{1}{b}$  
C. 若 $a + b = 2$，则 $\frac{ab}{a+b} \leq 1$  
D. 若 $a + b = 2$，则 $\frac{1}{a} + \frac{1}{b} \geq 2$  
【答案：AB】  

10. 已知关于 $x$ 的方程 $x^2 + (m-3)x + m = 0$，则下列结论正确的是 ( )  
A. 方程有一个正根一个负根的充要条件是 $m < 0$  
B. 方程有两个正根的充要条件是 $0 < m \leq 1$  
C. 方程无实数根的一个必要条件是 $m > 1$  
D. 当 $m=3$ 时，方程的两个实数根之和为 0  
【答案：AB】  

11. 已知 $x > 0, y > 0$，且 $x + y + xy = 3$，则下列结论正确的是 ( )  
A. $xy$ 的取值范围是 $(0,1]$  
B. $x+y$ 的取值范围是 $[2,3]$  
C. $x+2y$ 的最小值是 $4\sqrt{2}-3$  
D. $x+5y$ 的最小值为 $4\sqrt{5}-6$  
【答案：CD】
