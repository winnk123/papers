一、选择题

9. 下列选项不正确的是 ( )  
A. 集合 $\{x \in \mathbb{N} \mid x^3 = x\}$ 用列举法表示为 $\{0,1,-1\}$  
B. 空集是任何集合的子集  
C. 任何集合至少有两个子集  
D. 满足方程组 $\begin{cases} x + y = 0 \\ x - y = -1 \end{cases}$ 的解集为 $\left\{ x = -\frac{1}{2}, y = \frac{1}{2} \right\}$  
【答案：ACD】

10. 下列各组函数中是同一个函数的是 ( )  
A. $f(x) = \sqrt{(x-1)(x+1)}$, $g(x) = \sqrt{x-1} \cdot \sqrt{x+1}$  
B. $f(x) = \frac{x^2 - 9}{x + 3}$, $g(x) = (\sqrt{x})^2 - 3$  
C. $f(x) = x^2 + (x-1)^0$, $g(x) = \frac{x^3 - x^2 + x - 1}{x - 1}$  
D. $f(x) = \sqrt{x} + \frac{1}{x}$, $g(t) = \sqrt{t} + \frac{1}{t}$  
【答案：D】

11. 已知 $a > 0$, $b > 0$, $ab + 2(a + b) = 14$, 则下列正确的是 ( )  
A. $ab$ 的最大值为 $11 - 6\sqrt{2}$  
B. $\frac{3}{a+2} + \frac{3}{b+2}$ 的最小值为 $\sqrt{2}$  
C. $(a+1)b$ 最大值为 8  
D. $2a + b$ 的最大值为 6  
【答案：BC】

12. 已知命题：$\exists x \in \mathbb{R}, ax^2 + ax + 1 \leq 0$。若此命题是假命题，则实数 $a$ 的取值集合是 ______。  
【答案：$a=0$】

13. 设矩形 $ABCD$ ($AB > CD$) 的周长为 $8\,\mathrm{cm}$，把 $\triangle ABC$ 沿 $AC$ 向 $\triangle ADC$ 折叠，$AB$ 折至与 $DC$ 于点 $P$，则 $\triangle ADP$ 面积的最大值为 ______。
