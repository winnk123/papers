1. 已知集合 $A=\{x\in \mathbb{Z} \mid -1\le x<2\}, B=\{x\in \mathbb{N} \mid 0\le x<3\}$, 则 $A\cap B=$ ( ) 【答案：C】
A. $$\{-1,0,1,2\}$$     B. $$[0,1,2]$$     C. $$\{0,1\}$$   D. $$[1,2]$$

2. 集合 $M=\left\{x \mid x=\frac{k\pi}{2}+\frac{\pi}{4}, k\in \mathbb{Z}\right\}$, $N=\left\{x \mid x=\frac{k\pi}{4}+\frac{\pi}{4}, k\in \mathbb{Z}\right\}$, 则 ( ) 【答案：B】
A. $$M=N$$     B. $$M\subseteq N$$     C. $$N\subseteq M$$   D. $$M\cap N=\varnothing$$

3. 设 $a$, $b$ 是实数，则“$a>b$”是“$\frac{1}{a}<\frac{1}{b}$”的 ( ) 【答案：D】
A. 充分而不必要条件     B. 必要而不充分条件     C. 充分必要条件   D. 既不充分也不必要条件

4. 设 $x>0$, $y>0$, $\frac{1}{x}+2y=2$, 则 $x+\frac{1}{y}$ 的最小值为 ( ) 【答案：C】
A. $$\frac{3}{2}$$     B. $$2\sqrt{2}$$     C. $$\frac{3}{2}+\sqrt{2}$$   D. $$3$$

5. 若 $-2<a<b<2, 2<c<3$, 则下列不等式不正确的是 ( ) 【答案：D】
A. $$ac<bc$$     B. $$-12<(a-b)c<0$$     C. $$-4<a-b<0$$   D. $$ab<ac$$

6. 设正实数 $x$, $y$, $z$ 满足 $x^2-xy+y^2-z=0$, 则 $\frac{xy}{z}$ 的最大值为 ( ) 【答案：D】
A. $$4$$     B. $$2$$     C. $$3$$   D. $$1$$

7. 已知命题“$\forall x\in \{x \mid -3\le x\le -2\}, mx>12$”是假命题的一个充分不必要条件是 ( ) 【答案：B】
A. $$m\le -4$$     B. $$m\ge -4$$     C. $$m<-6$$   D. $$m\ge -6$$

8. 已知 $a>0, b>0$, 且 $\frac{1}{a}+\frac{2}{b}=1$, 则 $\frac{2}{a-1}+\frac{1}{b-2}$ 的最小值为 ( ) 【答案：A】
A. $$2$$     B. $$2\sqrt{2}$$     C. $$\frac{3\sqrt{2}}{2}$$   D. $$1+\frac{3\sqrt{2}}{4}$$

9. 若实数 $a$, $b$ 满足 $\begin{cases} 1\leq a+b\leq 5 \\ -1\leq a-b\leq 3 \end{cases}$, 则下列叙述正确的是 ( ) 【答案：ABC】

10. 下列说法正确的是 ( ) 【答案：CD】
A. $a>b$ 的一个必要条件是 $a-1>b$     B. 若集合 $A=\{x \mid ax^2+x+1=0\}$ 中只有一个元素，则 $a=4$     C. “$ac<0$”是“一元二次方程 $ax^2+bx+c=0$ 有一正一根负根”的充要条件   D. 已知集合 $M=\{0,1\}$, 则满足条件 $M\cup N=M$ 的集合 $N$ 的个数为 4
