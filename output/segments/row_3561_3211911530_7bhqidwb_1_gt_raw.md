三、解答题

15. (13分) 已知集合 $A = \{ x | 3 \leq x < 10 \}$，$B = \{ x | 1 < 3x - 5 < 16 \}$。
(1) 求 $A \cup B$；
(2) 求 $(C_R A) \cap B$。

16. (15分) 已知集合 $A = \{ x | ax^2 + 2x + 1 = 0,\ a \in R \}$。
(1) 若 $1 \in A$，用列举法表示 $A$；
(2) 当集合 $A$ 中有且只有一个元素时，求 $a$ 的值并表示出 $A$。

17. (15分) 已知集合 $A = \{ x | x^2 + 4x = 0 \}$，$B = \{ x | x^2 + 2x - k = 0 \}$。
(1) 若 $B \neq \varnothing$，求实数 $k$ 的取值范围；
(2) 若 $B \subseteq A$，求实数 $k$ 的取值范围。

18. (17分) 已知集合 $U = R$，二次函数 $y = x^2 - 2tx + t^2 - 4$（$t$ 为实数），集合 $A = \{ x | y \le 0 \}$，集合 $B = \{ m | m = y_{\min}, 0 \leq x \leq 4 \}$（$y_{\min}$ 表示函数在 $0 \le x \le 4$ 范围内的最小值），集合 $C = \{ x | -3 < x - t < 3 \}$。
(1) 求集合 $A$（用含 $t$ 的式子表示），并判断 $t=2$ 时，$A$ 与 $C$ 的关系；
(2) 求集合 $B$ 中 $m$ 的取值（用含 $t$ 的式子表示，需分情况讨论二次函数在 $0 \leq x \leq 4$ 内的最小值）。
