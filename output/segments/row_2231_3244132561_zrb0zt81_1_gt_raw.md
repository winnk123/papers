二、解答题

15. (13分)  
(1) 已知角 $\alpha$ 终边上一点 $P(-2, 3)$，求 $\frac{\cos(\frac{\pi}{2} + \alpha)\sin(\pi + a)}{\cos(\pi - \alpha)\sin(3\pi + \alpha)}$ 的值。  
(2) 已知 $\frac{1 - 2\sin \alpha \cos \alpha}{\cos^2 \alpha - \sin^2 \alpha} = \frac{1}{2}$，求 $\tan \alpha$ 的值。

16. (15分) 已知函数 $f(x) = x\sin x + \cos x, x \in (0, 2\pi)$。  
(1) 求函数 $f(x)$ 在 $x = \pi$ 处的切线方程；  
(2) 求函数 $f(x)$ 的极值。

17. (15分) 已知函数 $f(x) = \sin x \cos\left(x - \frac{\pi}{3}\right)$。  
(1) 求函数 $f(x)$ 的最小正周期；  
(2) 在锐角 $\triangle ABC$ 中，若 $f(A) = \frac{\sqrt{3}}{2}, AC = \sqrt{2}, BC = \sqrt{3}$，求 $\triangle ABC$ 的面积。  


18. (17分) 设锐角 $\triangle ABC$ 的内角 $A, B, C$ 所对的边分别为 $a, b, c$，已知 $a = b\cos A - a\cos B$。  
(1) 求证：$B = 2A$；  
(2) 求 $\frac{b}{a}$ 的取值范围；  
(3) 若 $c = 1$，求 $\triangle ABC$ 面积的取值范围。
