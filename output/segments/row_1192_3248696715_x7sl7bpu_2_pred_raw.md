[严重污损]

15. 已知集合 $A = \{x | 1 \le x \le 3\}$, $B = \{x | ax - 1 \ge 0, a \ne 0\}$; 若 $A \subseteq B$, 求实数 $a$ 的取值范围.
(1) 当 $a = 2$ 时, 求 $A \cap B$ 与 $C_R(A \cup B)$.
(2) 若 $A \subseteq B$, 求实数 $a$ 的取值范围.

【答案：】
(1) $A \cap B = \left\{x \mid 1 \le x \le 3\right\}, C_R(A \cup B) = \left\{x \mid x < 1 \text{ 或 } x > 3\right\}$
(2) $a \le -\frac{1}{3}$ 或 $a > 0$

16. 已知命题 $p: \exists x \in R, \frac{3}{x^2} + tx - 2 \ge 0$, 命题 $q: \forall x \in R, x + 2kx + 3k + 4 = 0$. 若命题 $p$ 为假命题时，求实数 $k$ 的取值范围；若命题 $p$ 和 $q$ 中有且仅有一个是假命题，求实数 $k$ 的取值范围.

【答案：】
(1) $k < -8$ 或 $k > 0$
(2) $-8 < k \le 0$

17. 设全集 $U = R$, 集合 $A = \left\{x \left| \frac{x - 5}{x - 1} \le 0\right.\right\}, B = \left\{x \left| -8 < x < 2a - 1\right.\right\}$, 其中 $a \in R$.
(1) 若“$x \notin A$”是“$x \in B$”的必要而不充分条件，求实数 $a$ 的取值范围.
(2) 若命题“$\exists x \in A, 使得 x \notin B$”是假命题，求实数 $a$ 的取值范围.

【答案：】
(1) $a \ge 5$
(2) $a \le 2$
