9. 下列结论错误的是 ( )
A. 若圆 C₁: x² + y² + 2x + 3y + 1 = 0 ，圆 C₂: x² + y² + 4x + 3y + 2 = 0 ，则圆 C₁ 与圆 C₂ 的公共弦所在直线方程是 x = -(1)/(2) 。
B. 若圆 x² + y² = r² 上且仅有 2 个点到直线 l: x - y + √(2) = 0 的距离等于 1，则 0 < r < 2 。
C. 曲线 C₁: x² + y² + 2x = 0 与曲线 C₂: x² + y² - 4x - 8y + m = 0 恰有三条公切线，则 m = 4 。
D. 已知圆 C: x² + y² = 2 ， P 为直线 x + y + 2√(3) = 0 上一动点，过点 P 向圆 C 引条切线 PA ，其长度最小值为 4。
