9. 若集合 $M = \{x | x = 2m - 1, m \in \mathbb{Z}\}, N = \{x | x = 2n + 1, n \in \mathbb{Z}\}$，则下列结论正确的是 ( )
A. $M = P$
B. $M = N$, $x < 0$ 或 $x > 1$
C. $N \cap P = N$
D. $N \cap P = P$

【答案：B】

A. $(-\infty, 0)$
B. $(-3, -1]$
C. $(3, +\infty)$
D. $[2, 3]$

11. 设 $P$ 是一个数集，且至少含有两个数，若对任意 $a, b \in P$，都有 $a + b, a - b, ab, \frac{a}{b} \in P$，则集合 $M$ 可以是 ( )
A. $\{a, b\}$
B. $\{-3, -1\}$
C. $(3, +\infty)$
D. $[2, 3]$

【答案：B】

12. 若命题“$\exists a < 0, a + - > b$”是假命题，则实数 $b$ 的取值范围为 __________。
【答案：$b \leq 0$】

13. 已知集合 $U = \{1, 2, 3, 4, 5, 6, 7\}$，$A, B \subseteq U$ 且 $A \subsetneq B$，满足：$A \cap B = \{1, 4\}$，则 $A = \underline{\quad}$，$S$ 的所有非空子集的元素之和为 128，则 $A = \underline{\quad}$。
【答案：$\{1, 4\}$】

14. 设集合 $S = \{a_1, a_2, a_3, a_4\}$，其中 $a_i \in \mathbb{Z}(i=1,2,3,4)$，若 $a_1 + a_2 + a_3 + a_4 = 128$，则 $a_1 + a_2 + a_3 + a_4 = \underline{\quad}$。
【答案：$128$】
