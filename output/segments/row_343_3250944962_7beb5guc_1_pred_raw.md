2. 已知命题 $p: \exists x_0 \geq 1$，使 $|x_0^2 - 1| < 0$，其否定命题 $\neg p$ 为 ( ) 【答案：B】
A. $\neg p: \exists x_0 > 1$，使 $|x_0^2 - 1| \geq 0$
B. $\neg p: \forall x > 1$，使 $|x^2 - 1| \geq 0$
C. $\neg p: \forall x \leq 1$，使 $|x_0^2 - 1| \geq 0$
D. $\neg p: \exists x_0 \leq 1$，使 $|x_0^2 - 1| \geq 0$

3. 下列命题为真命题的是 ( ) 【答案：B】
A. 若 $a > b > 0$，则 $ac^2 > bc^2$   B. 若 $a > b > 0$，则 $a^2 > b^2$
C. 若 $a < b < 0$，则 $a^2 < b^2$   D. 若 $a < b < 0$，则 $\frac{1}{a} < \frac{1}{b}$

4. 如图，$I$ 是全集，$M$、$P$、$S$ 是 $I$ 的 3 个子集，则阴影部分所表示的集合是 ( ) 【答案：C】

B. $(M \cap P) \cup S$
C. $(M \cap P) \cap C_I S$
D. $(M \cap P) \cup C_I S

5. 不等式 $\frac{1-2x}{3x+1} \geq 1$ 的解集是 ( ) 【答案：B】

A. $(1,2)$   B. $(1,4)$   C. $[1,2]$   D. $[1,4]$
