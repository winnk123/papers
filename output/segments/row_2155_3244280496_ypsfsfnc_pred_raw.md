2. 给出下列关于空间向量的命题，其中正确的结论是 ( ) 【答案：C】
A. 若 $\vec{a}$ 与 $\vec{b}$ 所在的直线是异面直线，则 $\vec{a}$ 与 $\vec{b}$ 一定不共面
B. 非零向量 $\vec{a}$，$\vec{b}$，$\vec{c}$ 满足 $\vec{a}$ 与 $\vec{b}$，$\vec{b}$ 与 $\vec{c}$，$\vec{c}$ 与 $\vec{a}$ 都是共面向量，则 $\vec{a}$，$\vec{b}$，$\vec{c}$ 必共面
C. 两个非零向量 $\vec{a}$，$\vec{b}$ 与任何一个向量 $\vec{c}$ 都不能构成空间的一个基底，则 $\vec{a} // \vec{b}$
D. $\vec{a}$ 与 $\vec{b}$ 是平面 $\alpha$ 上互不平行的向量，点 $A \notin \alpha$，点 $B \notin \alpha$，则 $\overline{AB}$ 与 $\vec{a}$、$\vec{b}$ 一定不共面

D. 所有直线 $l$ 组成的平面区域可覆盖整个直角坐标平面
