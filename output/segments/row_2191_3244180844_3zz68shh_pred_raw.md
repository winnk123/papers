       A. ∃x ≤ 2, (x+1)(x-2) ≤ 0
       B. ∃x > 2, (x+1)(x-2) ≤ 0
       C. ∀x > 2, (x+1)(x-2) < 0
       D. ∀x ≤ 2, (x+1)(x-2) ≥ 0
