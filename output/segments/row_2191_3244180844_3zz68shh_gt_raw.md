**一、单选题：本题共 8 小题，每小题 5 分，共 40 分。在每小题给出的选项中，只有一项是符合题目要求的。**

1. 已知命题 p: ∀x > 2, (x+1)(x-2) > 0，则 ¬p 是（ ）

   A. ∃x ≤ 2, (x+1)(x-2) ≤ 0  
   B. ∃x > 2, (x+1)(x-2) ≤ 0  
   C. ∃x > 2, (x+1)(x-2) < 0  
   D. ∃x ≤ 2, (x+1)(x-2) < 0  

2. 下列命题中，为真命题的是（ ）

   A. ∀a,b∈R, \(\frac{a^2 + b^2}{2} \geq (\frac{a+b}{2})^2\)  
   B. 若 a > b，则 ac² > bc²  
   C. 若 1/a > 1/b, 则 a < b  
   D. 若 a > b，c > d, 则 a-c > b-d

3. 已知集合 A = {x | x = 3k + 1, k ∈ Z}, B = {x | x = 6k + 1, k ∈ Z}, C = {x | x = 6k - 2, k ∈ Z}，则下列选项错误的是（ ）

   A. B = C  
   B. A = B ∪ C  
   C. B ∩ C = ∅  
   D. B ⊆ A  

4. 已知全集 U = {2,3,4,5,6,7,8}, A,B 是 U 的两个子集，且 (∁ᵤA) ∪ (∁ᵤB) = {2,3,4,6,7,8}，则 A ∩ (∁ᵤB) = {2，3，6}, 则 (∁ᵤA) ∪ B =

   A. {4,7,8}  
   B. {4,5,7,8}  
   C. {2,3,5,6}  
   D. {3,5,6}  

5. 已知 |x| ∈ {-x, 1, 0}，则实数 x 的取值构成的集合为（ ）

   A. {x | x < 0}  
   B. {x | x < 0 或 x = 1}  
   C. {x | x < -1 或 -1 < x < 0 或 x = 1}  
   D. {x | x < -1 或 -1 < x < 0}  

6. 我校高一（1）班甲、乙两位男同学 22:25 同时从教室沿同一路线跑到寝室，甲有一半时间以速度 m 行进，另一半路程以时间 n 行进；乙有一半路程以速度 m 行进，另一半路程以速度 n 行进，若 m ≠ n，则甲、乙两人到达寝室的情况是（ ）

   A. 甲先到  
   B. 乙先到  
   C. 甲乙同时到  
   D. 不能确定  

7. 已知集合 A = {x | 1 ≤ x < 5}, B = {x | a - 1 < x ≤ a + 1}，则使 “A ∩ B ≠ ∅” 成立的充分不必要条件的是（D）

   A. 0 < a ≤ 6  
   B. 0 ≤ a < 6  
   C. 0 ≤ a ≤ 6  
   D. 0 < a < 6  

8. 已知对任意 x ∈ {x | x² - 5x + 6 ≤ 0}, y ∈ {y | y² - 11y + 24 ≤ 0}, mx² - 4xy + y² ≥ 0 恒成立，则实数 m 的最小值是（D）

   A. 0  
   B. 2  
   C. 2√2  
   D. 4  

**二、多选题：本题共 3 小题，共 18 分。在每小题给出的选项中，有多项符合题目要求。全部选对的得 6 分，部分选对的得部分分，有选错的得 0 分。**

9. 若不等式 ax² - bx + c > 0 的解集是 {x | -1 < x < 2}，则（ABC）
   A. a < 0  
   B. 不等式的 (ax+c)/(x-1)  ≥ 0 的解集为 {x | -1 < x ≤ 2}  
   C. 不等式 bx² + ax - c ≥ 0 的解集为 ∅  
   D. 不等式 ax² - cx + b < 0 的解集为 R

10. 下列说法正确的是（ACD）

A. 若 $P$ 是 $q$ 的必要不充分条件，$P$ 是 $r$ 的充要条件，则 $q$ 是 $r$ 的充分不必要条件

B. 若 $\forall x \in \mathbb{R}, \sqrt{kx^2 - 4kx + 3}$ 恒有意义，则实数 $k$ 的取值范围是 $0 < k ≤ 1 $

C. 若关于 $x$ 的不等式 $x^2 - kx + k - 2 < 0$ 的解集为 $A$，集合 $B = \{x \mid 1 < x < 2\}$，满足 $A \cap B = B$，则实数 $k$ 的取值范围是 $k \geq 2$

D. “一元二次不等式 $ax^2 + bx + c < 0$ 在 $\mathbb{R}$ 上有解”是“$\begin{cases} a > 0 \\ b^2 - 4ac > 0 \end{cases}$”的必要不充分条件

11. 已知 $a > 0, b > 0, a + b = 2$，则下列结论成立的是（AB）

A. $ab$ 的最大值为 $1$

B. $\frac{b^2 + 2}{ab}$ 的最小值为 $\sqrt{3} + 1$

C. $\frac{b^2 - 6b + 18}{a + 1}$ 的最小值为 $6$

D. $a - \frac{1}{b}$ 的最小值为 $0$
