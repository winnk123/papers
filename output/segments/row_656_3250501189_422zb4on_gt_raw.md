一、选择题
1. (5分) 过点 $A(-1,\sqrt{3})$, $B(2,0)$ 两点的直线倾斜角为 ( )
A. $\frac{\pi}{3}$     B. $\frac{3\pi}{4}$     C. $\frac{2\pi}{3}$     D. $\frac{5\pi}{6}$
【答案：D】

2. 在空间直角坐标系 $Oxyz$ 中，已知点 $A(1,1,1)$, $B(2,-1,0)$，若点 $P$ 与点 $A$ 关于 $Oxy$ 平面对称，则 $\overrightarrow{BP} =$ ( )
A. $(-3,2,1)$     B. $(-1,0,1)$     C. $(-1,0,-1)$     D. $(3,-2,-1)$
【答案：A】

3. 已知 $\vec{a}=(1,0), \vec{b}=(0,1), \vec{p}=a\cdot\vec{a}+2\vec{b}-\vec{c}$，则 $\vec{p}=$ ( )
A. $-1$     B. $1$     C. $0$     D. $-2$
【答案：A】

4. 若两平行直线 $x+my+2=0$ 与 $2x+4y+n=0(n>0)$ 之间的距离是 $\sqrt{5}$，则 $m+n=$ ( )
A. $-2$     B. $12$     C. $14$     D. $-2$
【答案：C】

5. 在平行六面体 $ABCD-A_1B_1C_1D_1$ 中，点 $E$ 为棱 $AA_1$ 的中点，点 $F$ 为棱 $CC_1$ 上靠近 $C$ 的三等分点。若 $\overrightarrow{EF}=x\overrightarrow{AB}+y\overrightarrow{AD}+z\overrightarrow{AA_1}$，$x,y,z\in\mathbb{R}$，则 $x+y+z$ 的值为 ( )
A. $\frac{1}{6}$     B. $\frac{11}{6}$     C. $\frac{17}{6}$     D. $\frac{1}{6}$
【答案：B】

6. 过点 $P(0,-1)$ 作直线 $l$，若直线 $l$ 与线段 $A(-2,1)$, $B(2\sqrt{3},1)$ 两点的线段有公共点，则直线 $l$ 的倾斜角范围为 ( )
A. $\left[\frac{\pi}{6},\frac{\pi}{4}\right]$     B. $\left[\frac{\pi}{6},\frac{3\pi}{4}\right]$     C. $\left[0,\frac{\pi}{6}\right]\cup\left[\frac{3\pi}{4},\pi\right)$     D. $\left[\frac{\pi}{6},\frac{\pi}{2}\right]\cup\left[\frac{3\pi}{4},\pi\right)$
【答案：B】

7. 已知点 $A(1,2)$，直线 $l_1: (λ+2)x+(1-λ)y+2λ+7=0(λ\in\mathbb{R})$，则 $A$ 到 $l_1$ 的距离的最大值为 ( )
A. $3$     B. $\sqrt{10}$     C. $\frac{5\sqrt{2}}{2}$     D. $5$
【答案：D】

8. 三棱锥 $A-BCD$ 中，底面是边长为 2 的正三角形，$AC\perp BC, AD\perp BD$，直线 $AC$ 与 $BD$ 所成角为 $45^\circ$，则三棱锥 $A-BCD$ 的外接球表面积为 ( )
A. $6\pi$     B. $\frac{19\pi}{3}$     C. $8\pi$     D. $\frac{25\pi}{3}$
【答案：B】

9. 已知 $\overrightarrow{AB}=(-2,1,4), \overrightarrow{AC}=(4,2,0), \overrightarrow{AP}=(1,-2,1), \overrightarrow{AQ}=(0,4,4)$，则下列说法正确的是 ( )
A. $\overrightarrow{AP}$ 是平面 $ABC$ 的一个法向量
B. $A,B,C,Q$ 四点共面
C. $\overrightarrow{PQ}\parallel\overrightarrow{BC}$
D. $BC=\sqrt{53}$
【答案：AD】

10. 已知直线 $l_1: x+y+2=0$，直线 $l_2: x+py-2=0$，下列说法正确的是 ( )
A. 直线 $l_1$ 在 $y$ 轴上的截距等于直线 $l_2$ 在 $x$ 轴上的截距
B. 若点 $P(-2,2)$ 在直线 $l_1$ 上，则点 $P'(-2,2)$ 也在直线 $l_2$ 上
C. 若 $l_1\parallel l_2$，则 $p=1$
D. 若 $l_1\perp l_2$，则 $p=1$
【答案：BD】

11. 在棱长为 2 的正方体 $ABCD-A_1B_1C_1D_1$ 中，点 $P$ 满足 $\overrightarrow{BP} = \lambda \overrightarrow{BC} + \mu \overrightarrow{BB_1}$，且 $0 \leq \lambda \leq 1, 0 \leq \mu \leq 1$，则下列说法正确的是 ( )

A. 若 $\lambda + \mu = 1$，则 $AP // $ 面 $AC_1D$

B. 若 $\lambda + \mu = 1$，则 $AP \perp BC_1$

C. 若 $\lambda = \mu = \frac{1}{2}$，则 $P$ 到平面 $ABD$ 的距离为 $\frac{2}{3}\sqrt{3}$

D. 若 $\lambda = 1, 0 \leq \mu \leq 1$ 时，直线 $DP$ 与平面 $ABD$ 所成角为 $\theta$，则 $\sin\theta \in \left[\frac{\sqrt{3}}{3}, \frac{\sqrt{6}}{3}\right]$
【答案：ACD】

三、填空题

12. 若 $\vec{a}=(1,0,1), \vec{b}=(0,2,2)$，则 $\sin\langle\vec{a},\vec{b}\rangle=$ ______
【答案：$\frac{\sqrt{2}}{2}$】
