6. 对任意实数 $x$，不等式 $(a-2)x^2 + 2(a-2)x - 4 < 0$ 恒成立，则 $a$ 的取值范围是 ( )
A. $(-2,2]$     B. $[-2,2]$     C. $(-\infty,-2) \cup [2,+\infty)$     D. $(-\infty,-2] \cup [2,+\infty)$
【答案：C】

7. 若 $a,b > 0$，且 $ab = a+b+3$，则 $a+b$ 的取值范围是 ( )
A. $\{x | x \leq 2\}$     B. $\{x | x \geq 6\}$     C. $\{x | 2 \leq x \leq 6\}$     D. $\{x | 2 < x < 6\}$
【答案：B】

8. 已知关于 $x$ 的不等式 $1 \leq ax^2 + bx + c \leq 3$ 的解集为 $\{x | 1 \leq x \leq 3\}$，则下列说法正确的是 ( )
A. $-8 \leq b < 8$     B. $-5 \leq c < 9$     C. $a+b \leq 6$     D. $b+c > 0$
【答案：C】

9. 集合 $A=\{0,1,2\}$，$B=\{x | ax-3=0\}$，$A \cap B = B$，则 $a$ 的值可以是 ( )
A. $0$     B. $1$     C. $\frac{3}{2}$     D. $3$
【答案：D】

10. 对于实数 $a,b,c$，下列命题是真命题的为 ( )
A. 若 $a>b>0$，则 $\frac{1}{a} < \frac{1}{b} < \frac{1}{c} < \frac{1}{2}$
B. 若 $a>b$，则 $ac^2 \geq bc^2$
C. 若 $a>0>b$，则 $a^2 < -ab$
D. 若 $c>a>b>0$，则 $\frac{a}{c-a} > \frac{b}{c-b}$
【答案：D】

11. 已知关于 $x$ 的不等式 $ax^2 + 4x + 2b \leq 0$（$a \neq 0$）的解集为 $\left\{-\frac{2}{a}\right\}$，下列选项中正确的有 ( )
A. $a > 0$     B. $ab = -4$     C. $a+2b$ 的最小值为 $4 \left(\frac{4}{a} - \frac{8}{a} + 2b\right)$     D. 若 $a>b$，则 $\frac{a^2+b^2}{a-b}$ 的最小值为 $\frac{2}{3}$
【答案：A,C,D】

12. 已知 $x > 0$，则 $\frac{x^2-x+4}{x}$ 的最小值是 ______。
【答案：$3$】

13. 函数 $f(x) = \sqrt{\frac{2-x}{3x-1}}$ 的定义域是 ______。
【答案：$\left\{x \mid \frac{1}{3} < x \leq 2\right\}$】
