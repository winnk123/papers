2. 圆 $C_1:(x-2)^2+(y+1)^2=3$ 与圆 $C_2:x^2+(y-1)^2=3$ 的位置关系是 ( ) 【答案：A】

3. 若直线 $3x-4y-2=0$ 与直线 $6x+my+5=0$ 平行, 则这两条直线间的距离为 ( ) 【答案：B】
