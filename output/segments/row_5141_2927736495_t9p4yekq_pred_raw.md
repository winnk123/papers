[条件缺失]

1. $\cos\left(-\dfrac{5\pi}{3}\right)$ 的值为 ( )
A. $\dfrac{1}{2}$   B. $-\dfrac{1}{2}$   C. $\dfrac{\sqrt{3}}{2}$   D. $-\dfrac{\sqrt{3}}{2}$
【答案：C】

2. 已知 $\tan\alpha=-2$，则 $\sin(\pi-\alpha)-\sin\left(\dfrac{3\pi}{2}-\alpha\right)$ 的值为 ( )
A. $\dfrac{2}{3}$   B. $-\dfrac{2}{3}$   C. $\dfrac{1}{3}$   D. $-\dfrac{1}{3}$
【答案：B】

3. 已知点 $P(m-1)$ 在角 $\alpha$ 的终边上，且 $\cos\alpha=\dfrac{3\sqrt{10}}{10}$，则 $\tan\alpha=$ ( )
A. $\dfrac{1}{3}$   B. $-\dfrac{1}{3}$   C. $-\dfrac{3}{10}$   D. $\dfrac{3}{10}$
【答案：B】

4. 设角 $\theta$ 是第一象限角，且满足 $\cos\dfrac{\theta}{2}=-\cos\dfrac{\theta}{2}$，则 $\theta$ 的终边所在的象限是 ( )
A. 第一象限   B. 第二象限   C. 第三象限   D. 第四象限
【答案：D】

5. 已知扇形的面积是 $4\text{cm}^2$，当扇形周长最小时，扇形的圆心角的大小为 (单位：rad)
A. $\dfrac{\pi}{4}$   B. $\dfrac{\pi}{2}$   C. $\pi$   D. $2\pi$
【答案：C】

6. 函数 $f(x)=\sin(2x+\dfrac{\pi}{6})$ 的最小正周期为 $\pi$ ( )
A. 正确   B. 错误   C. 无法确定   D. 条件不足
【答案：B】

7. 已知函数 $f(x)=\tan(2x-\dfrac{\pi}{3})$，下列说法正确的是 ( )
A. 定义域为 $\left\{x|x\in R,x\neq \dfrac{k\pi}{2}+\dfrac{\pi}{6},k\in Z\right\}$
B. 值域为 $\mathbb{R}$
C. 单调递增区间为 $\left[\dfrac{k\pi}{2}-\dfrac{\pi}{12},\dfrac{k\pi}{2}+\dfrac{5\pi}{12}\right],k\in Z$
D. 图像所有对称中心为 $\left(\dfrac{k\pi}{2}+\dfrac{\pi}{6},0\right),k\in Z$
【答案：D】

8. 若点 $P$ 为 $\triangle ABC$ 所在平面内一点，且满足 $\overrightarrow{PA}+\overrightarrow{PB}+\overrightarrow{PC}=2\overrightarrow{BC}$，则 $\dfrac{S_{\triangle PBC}}{S_{\triangle ABC}}$ 的值为 ( )
A. $\dfrac{1}{2}$   B. $\dfrac{1}{4}$   C. $\dfrac{2}{3}$   D. $\dfrac{1}{3}$
【答案：B】

9. 已知函数 $f(x)=\sin x + \cos x$，则下列说法正确的是 ( )
A. 最小正周期为 $2\pi$   B. 最大值为 $\sqrt{2}$   C. 图像关于直线 $x=0$ 对称   D. 在区间 $(0,\dfrac{\pi}{2})$ 上单调递减
【答案：B】

10. 若 $a,b,c$ 为实数，且 $a+b+c=0$，则下列不等式一定成立的是 ( )
A. $ab+bc+ca\leq0$   B. $a^2+b^2+c^2\geq0$   C. $abc\geq0$   D. $|a|+|b|+|c|\geq0$
【答案：D】
