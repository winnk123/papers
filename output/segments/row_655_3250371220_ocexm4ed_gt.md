一、选择题

1. 已知集合 M={1, 2, 3}, N={2, 3, 4} , 则 ( )
A. M ⊆ N B. N ⊆ M C. M ∩ N = {2, 3} D. M ∪ N = {1, 4}

2. 命题 “ ∀ x ∈ R, |x| + √(x) ≥ 0 ” 的否定是 ( )
A. ∀ x ∈ R, |x| + √(x) < 0 B. ∃ x ∈ R, |x| + √(x) ≥ 0
C. ∀ x ∈ R, |x| + √(x) ≤ 0 D. ∃ x ∈ R, |x| + √(x) < 0

3. 已知 p: x₁=2 且 x₂=3 , q: x₁+x₂=5 , 则 p 是 q 的 ( )
A. 充分不必要条件 B. 必要不充分条件
C. 充要条件 D. 既不充分也不必要条件

4. 已知集合 A={0, 1, 2, 4, 6}, B={n ∈ N^* | 2ⁿ < 33} , 则集合 A ∩ B 的子集个数为 ( )
A. 8 B. 7 C. 6 D. 4

5. 已知 -2 ≤ x < 3, -1 < y ≤ 5 , 则 2x - y 的取值范围是 ( )
A. (-3, 1) B. [-3, 1C. (-9, 7) D. [-9, 7)

6. 已知 a, b, c > 0 , 且 a + b + c = 1 , 则 (2)/(a) + (2)/(b + c) 的最小值为 ( )
A. 3 B. 4 C. 8 D. 9

7. 设集合 A = {x | x² + x - 6 = 0}, B = {x | mx - 2 = 0} , 则 B 是 A 的真子集的一个充分不必要条件是 ( )
A. m ∈ {0, (2)/(3)} B. m ∈ {0, -(2)/(3)} C. m ∈ {0, -(2)/(3), 1} D. m ∈ {0, (2)/(3), 1}
