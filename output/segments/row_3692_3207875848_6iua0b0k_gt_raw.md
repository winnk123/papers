1. 设复数 $z$ 满足 $z = (1+i)i$，则 $|z|=$ ( ) 【答案：A】
A. $\sqrt{2}$     B. $2$     C. $\sqrt{5}$     D. $1$

2. 已知 $\vec{a} = (\lambda, -2)$，$\vec{b} = (1, 2)$，$\vec{a} \perp \vec{b}$，则 $\lambda =$ ( ) 【答案：B】
A. $-4$     B. $4$     C. $1$     D. $-1$

3. 已知函数 $f(x) = \sin x - \cos x$，则下列选项中是 $f(x)$ 的一个单调递增区间的是 ( ) 【答案：B】
A. $\left[-\frac{\pi}{2}, \frac{\pi}{2}\right]$     B. $\left[-\frac{\pi}{4}, \frac{3\pi}{4}\right]$     C. $\left[-\frac{3\pi}{4}, \frac{\pi}{4}\right]$     D. $\left[\frac{3\pi}{4}, \frac{7\pi}{4}\right]$

4. 已知全集 $U = \{x \mid x < 10, x \in \mathbb{N}^*\}$，集合 $A, B$ 是 $U$ 的子集，若 $(\complement_U A) \cap B = \{5, 7, 9\}$，$A \cap B = \{2\}$，$(\complement_U A) \cap (\complement_U B) = \{6, 8\}$，则集合 $A =$ ( ) 【答案：D】
A. $\{2, 3, 4\}$     B. $\{1, 2, 4\}$     C. $\{1, 2, 3\}$     D. $\{1, 2, 3, 4\}$

5. 已知平面 $\alpha, \beta$，直线 $a, b$，则下列结论正确的是 ( ) 【答案：A】
A. 若 $a \subset \alpha, b \parallel a$，则 $b \parallel \alpha$     
B. 若 $a \parallel \beta, a \subset \alpha, b \subset \beta$，则 $a \parallel b$     
C. 若 $a \parallel \alpha, b \perp \alpha$，则 $a \perp b$     
D. 若 $a \parallel \beta, a \parallel \alpha$，则 $\alpha \parallel \beta$

6. 已知首项为 $1$ 的数列 $\{a_n\}$，其前 $n$ 项积是公差为 $3$ 的等差数列，则 $a_3 =$ ( ) 【答案：C】
A. $4$     B. $3$     C. $\frac{7}{4}$     D. $\frac{10}{7}$

7. 已知甲、乙、丙、丁四位老师参加青年教师教学大赛，问其比赛结果，他们回答如下：
甲：丙第一，乙第二；乙：丙第二，丁第三；丙：丁最后，甲第二。
如果每个人的两个回答中，都恰有一个是正确的，而且没有并列名次，那么这次比赛获得第一、二、三、四名依次是 ( ) 【答案：A】
A. 丙、甲、丁、乙     B. 丙、甲、乙、丁     C. 甲、乙、丙、丁     D. 甲、乙、丁、丙

8. $f(x) = 2x^3 - 3x^2 - 12x$，已知 $b < 0$，若 “$f(x) \geq a$” 的充要条件是 “$x \geq b$”，则实数 $b$ 的最大值为 ( ) 【答案：D】
A. $-2$     B. $-\frac{5}{2}$     C. $-1$     D. $-\frac{1}{2}$
