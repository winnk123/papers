15. (本小题满分 13 分)
已知 $m \in \mathbb{R}$, 复数 $z = 2m + 3 + (m - 1)i$.
(1) 若 $z$ 在复平面内对应的点位于第四象限，求 $m$ 的取值范围；
(2) 若 $z$ 满足 $z + 3\bar{z} = n + 4i$, $n \in \mathbb{R}$, 求 $\frac{n + mi}{3 + 4i}$ 的值。

16. (本小题满分 15 分)
在 $\triangle ABC$ 中，内角 $A,B,C$ 的对边分别为 $a,b,c$，且 $a = 3a\cos B + 3b\cos A$.
(1) 求 $\frac{a^2 + c^2}{ac}$ 的值；
(2) 若 $B = \frac{2\pi}{3}$, $\triangle ABC$ 的面积为 $\frac{15\sqrt{3}}{4}$, 求 $AC$ 边上的高。

17. (本小题满分 15 分)
已知二次函数 $f(x)$ 满足 $f(0) = 2$，函数 $g(x)$ 满足 $g(x-1) = 4x - 7$，且不等式 $f(x) + g(x) < 0$ 的解集为 $(-1, \frac{1}{2})$.
(1) 求 $f(x)$ 的解析式；
(2) 若关于 $x$ 的不等式 $f(x^3) \geq (2m - 1)x^3 + 9$ 对任意的 $x \in \mathbb{R}$ 恒成立，求实数 $m$ 的取值范围。

如图，在梯形 $ABCD$ 中，$AB // CD$, $AB \perp AD$, $AB = 2CD = 4$, $E,F$ 分别为 $DC,CB$ 的中点，且 $\overrightarrow{AC} \cdot \overrightarrow{EF} = 2$. $P$ 是线段 $AB$ 上的一个动点。

19. (本小题满分 17 分)
定义：若非零向量 $\overrightarrow{OM} = (a,b)$，函数 $f(x)$ 的解析式满足 $f(x) = a\sin x + b\cos x$，则称 $f(x)$ 为 $\overrightarrow{OM}$ 的伴随函数，$\overrightarrow{OM}$ 为 $f(x)$ 的伴随向量。
(1) 若向量 $\overrightarrow{OM}$ 为函数 $f(x) = 2\sin\left(x + \frac{\pi}{6}\right) + 4\sin\left(x - \frac{\pi}{2}\right)$ 的伴随向量，求 $|\overrightarrow{OM}|$；
(2) 若函数 $f(x)$ 为向量 $\overrightarrow{OM} = (\sqrt{3}, -1)$ 的伴随函数，在 $\triangle ABC$ 中，$BC = 2\sqrt{3}$，$f(A) = 1$，且 $\cos B\cos C = -\frac{1}{8}$，求 $\overrightarrow{AB} + \overrightarrow{AC}$ 的值；
(3) 若函数 $f(x)$ 为向量 $\overrightarrow{OM} = (2,1)$ 的伴随函数，关于 $x$ 的方程 $f(x) = m + 2\cos\left(\frac{x}{2} - \frac{2}{3}\cos x\right)$ 在 $[0, 2\pi]$ 上且仅有四个不相等的实数根，求实数 $m$ 的取值范围。
