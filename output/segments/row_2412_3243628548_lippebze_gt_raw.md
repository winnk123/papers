1. 已知集合 $U=\{1,2,3,4,5\}, A=\{1,3\}, B=\{1,2,4\}$, 则 $\complement_U B \cup A =$ ( ) 【答案：A】
A. $$\{1,3,5\}$$     B. $$\{1,3\}$$     C. $$\{1,2,4\}$$   D. $$\{1,2,4,5\}$$

2. 命题“$\forall x \in \mathbf{R}, x - |x| \geqslant 0$”的否定是 ( ) 【答案：C】
A. $$\exists x \in \mathbf{R}, x - |x| \geqslant 0$$     B. $$\forall x \in \mathbf{R}, x - |x| \geqslant 0$$
C. $$\exists x \in \mathbf{R}, x - |x| < 0$$     D. $$\forall x \in \mathbf{R}, x - |x| < 0$$

3. 使命题“$\forall x \in \{x | 1 \leqslant x \leqslant 2\}, x^2 - a \leqslant 0$”为真命题的一个充分不必要条件是 ( ) 【答案：B】
A. $$a \leqslant 4$$     B. $$a \geqslant 4$$     C. $$a \leqslant 5$$   D. $$a \geqslant 5$$

4. 已知 $a = \sqrt{c+3} + \sqrt{c+4}, b = \sqrt{c+2} + \sqrt{c+5}$, 则 ( ) 【答案：A】
A. $$a > b > 1$$     B. $$b > a > 1$$     C. $$a > 1 > b$$   D. $$b > 1 > a$$

5. 购买同一种物品，可以用两种不同的策略，第一种是不考虑物品价格的升降，每次购买这种物品的数量一定；第二种是不考虑物品价格的升降，每次购买这种物品所花的钱一定。假设连续两天购买该物品，第一天物品的价格为 $p_1$, 第二天物品的价格为 $p_2$, 且 $p_1 \ne p_2$, 则以下说法正确的为 ( ) 【答案：D】
A. 第一种方式购买物品的平均价格为 $\frac{p_1 p_2}{p_1 + p_2}$
B. 第二种方式购买物品的平均价格为 $\frac{p_1 + p_2}{2}$
C. 第一种方式购买物品所用平均价格更低
D. 第二种方式购买物品所用平均价格更低

D. $$y = \sqrt{x^2 + 5} + \frac{4}{\sqrt{x^2 + 5}}$$
