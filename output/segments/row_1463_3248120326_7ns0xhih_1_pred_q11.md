11. 在信道内传输0,1信号，信号的传输相互独立。发送0时，收到1的概率为 α(0 < α < 1) ，概率为 1-α ；发送1时，收到0的概率为 β(0 < β < 1) ，收到1的概率为 1-β 。考虑两种传输：单次传输是指每个信号只发送1次；三次传输是指每个信号重复发送3次。
信号需要译码，译码规则如下：
- 单次传输时，收到的信号即为译码；
- 三次传输时，收到的信号中若依次收到1,0,1，则译码为1；
- 多的即为译码（例如，若依次收到1,0,1，则译码为1）。
A. 采用单次传输方案，若依次发送1,0,1，则依次收到1,0,1的概率为 (1-α)(1-β)²
B. 采用三次传输方案，若发送1，则译码为1的概率为 β(1-β)²
C. 采用三次传输方案，若发送1，则译码为1的概率为 β(1-β)²
D. 当 0 < α < 0.5 时，若发送0，则采用三次传输方案译码为0的概率大于采用单次传输方案译码为0的概率
