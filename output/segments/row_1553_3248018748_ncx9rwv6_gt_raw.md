一、选择题

1. 直线 $\sqrt{3}x - y + 2 = 0$ 的倾斜角为 ( ) 【答案：D】
A. $$\frac{\pi}{6}$$     B. $$\frac{\pi}{3}$$     C. $$\frac{2\pi}{3}$$     D. $$\frac{3\pi}{4}$$

2. 已知点 $A(2,3)$, $B(4,1)$, 则线段 $AB$ 的垂直平分线的方程为 ( ) 【答案：B】
A. $$x - y - 1 = 0$$     B. $$x - y + 1 = 0$$     C. $$x + y - 5 = 0$$     D. $$x + y - 2 = 0$$

3. 已知圆 $C_1: x^2 + (y+1)^2 = 9$ 与圆 $C_2: (x+2\sqrt{3})^2 + (y-1)^2 = 1$ 的位置关系为 ( ) 【答案：C】
A. 相离     B. 相交     C. 外切     D. 内切

4. 若直线 $x + y - m = 0$ 被圆 $C: (x-1)^2 + (y+1)^2 = 4$ 截得的弦长为 $2\sqrt{2}$, 则 $m =$ ( ) 【答案：A】
A. $$\pm 2$$     B. $$\sqrt{2}$$     C. $$2$$     D. $$2\sqrt{2}$$

5. 如果椭圆 $\frac{x^2}{36} + \frac{y^2}{9} = 1$ 的弦被点 $(4,2)$ 平分，则这条弦所在的直线方程是 ( ) 【答案：B】
A. $$x - 2y = 0$$     B. $$x + 2y - 8 = 0$$     C. $$2x + 3y - 14 = 0$$     D. $$x + 2y - 4 = 0$$

6. 已知圆 $C: x^2 + y^2 + 2x - 2my - 4 - 4m = 0 (m \in \mathbb{R})$, 则当圆 $C$ 的面积最小时，圆上的点到坐标原点的距离的最大值为 ( ) 【答案：D】
A. $$\sqrt{5}$$     B. $$6$$     C. $$\sqrt{5} - 1$$     D. $$\sqrt{5} + 1$$

7. 在正四面体 $ABCD$ 中，$3\overrightarrow{AE} = \overrightarrow{AB}$, 则直线 $CE$ 与直线 $AD$ 所成角的余弦值为 ( ) 【答案：D】
A. $$\frac{2}{3}$$     B. $$\frac{\sqrt{2}}{3}$$     C. $$\frac{1}{7}$$     D. $$\frac{\sqrt{7}}{7}$$

8. 已知椭圆 $C: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 (a > b > 0)$ 的左、右焦点分别为 $F_1, F_2$, 过点 $F_1$ 且斜率为 $k (k > 0)$ 的直线 $l$ 与椭圆 $C$ 相交于 $A, B$ 两点，若 $\overrightarrow{AF_1} = 3\overrightarrow{F_1B}$, 且 $\angle AF_2B = 90^\circ$, 则椭圆 $C$ 的离心率为 ( ) 【答案：B】
A. $$\frac{\sqrt{2}}{2}$$     B. $$\frac{\sqrt{3}}{2}$$     C. $$\sqrt{3} - \sqrt{7}$$     D. $$\sqrt{5} - \sqrt{7}$$

9. 已知空间向量 $\vec{a} = (2, -1, 3)$, $\vec{b} = (-4, 2, x)$, 下列说法正确的是 ( ) 【答案：C】
A. 若 $\vec{a} \perp \vec{b}$, 则 $x = \frac{10}{3}$     B. 若 $3\vec{a} + \vec{b} = (2, -1, 10)$, 则 $x = 1$     C. 若 $\vec{a}$ 在 $\vec{b}$ 上的投影向量为 $\frac{1}{3}\vec{b}$, 则 $x = 4$     D. 若 $\vec{a}$ 与 $\vec{b}$ 夹角为锐角，则 $x \in \left(\frac{10}{3}, +\infty\right)$

10. 椭圆 $C: \frac{x^2}{4} + y^2 = 1$ 的左、右焦点分别为 $F_1$ 和 $F_2$, $P$ 为椭圆上一点，则下列说法正确的有 ( ) 【答案：AB】
A. 过点 $F_2$ 的直线与椭圆 $C$ 交于 $A, B$ 两点，则 $\triangle ABF_1$ 的周长为 16     B. 若 $PF_1 \perp PF_2$, 则 $\triangle F_1PF_2$ 的面积为 $\frac{\sqrt{3}}{2}$     C. 椭圆 $C$ 上存在点 $P$, 使得 $PF_1 \perp PF_2$     D. $|PF_1|$ 的取值范围是 $\left[2 - \sqrt{3}, 2 + \sqrt{3}\right]$

11. 如图，在棱长为 2 的正方体 $ABCD - A_1B_1C_1D_1$ 中，$M$ 为 $B_1C_1$ 的中点，$P$ 为线段 $B_1D_1$ 上动点（包括端点），则下列说法中正确的是 ( ) 【答案：B】

二、填空题

12. 若两条直线 $l_1: x + 2y - 6 = 0$ 与 $l_2: x + ay - 5 = 0$ 平行，则 $l_1$ 与 $l_2$ 间的距离是 ______ 【答案：$\frac{1}{\sqrt{5}}$】

13. 已知直线 $l: x - y - 6 = 0$ 和圆 $C: x^2 + y^2 + 2x + 2y - 6 = 0$. 若 $P$ 是直线 $l$ 上的动点，过 $P$ 作圆 $C$ 的一条切线，切点为 $M$，则 $|PM|$ 的最小值为 ______ 【答案：$\sqrt{14}$】

14. 若 $P$ 为椭圆 $\frac{x^2}{16} + \frac{y^2}{15} = 1$ 上任意一点，$EF$ 为圆 $N: (x-1)^2 + y^2 = 4$ 的任意一条直径，则 $\overrightarrow{PE} \cdot \overrightarrow{PF}$ 的取值范围是 ______ 【答案：$[-1, 3]$】
