8. 关于 $x$ 的方程 $ax^2 - (a+1)x - 2a + 3 = 0$ 有两个不相等的实数根 $x_1, x_2$，若 $-2 < x_1 < x_2 < 3$，则实数 $a$ 的取值范围是 ( )
A. $\{a | a > 1\}$
B. $\left\{a \left| a > 1 \text{ 或 } a < -\frac{5}{4}\right.\right\}$
C. $\left\{a \left| a > \frac{1}{5} \text{ 或 } a < -\frac{1}{9}\right.\right\}$
D. $\left\{a \left| a > 1 \text{ 或 } a < -\frac{5}{4}\right.\right\}$

【答案：D】
