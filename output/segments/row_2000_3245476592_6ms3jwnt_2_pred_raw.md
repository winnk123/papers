(2) 若复数 $z$ 是方程 $x^2 + mx + n = 0$ 的一个根，求实数 $m,n$ 的值.
$\frac{5-12i}{1+4i} = \frac{(5-12i)(1-4i)}{(1+4i)(1-4i)} = \frac{5 - 20i - 12i + 48i^2}{1+16} = \frac{-43 - 32i}{17} = -\frac{43}{17} - \frac{32}{17}i$

$z = -\frac{43}{17} - \frac{32}{17}i$, $z^2 = \left(-\frac{43}{17}\right)^2 + 2\cdot\left(-\frac{43}{17}\right)\cdot\left(-\frac{32}{17}\right)i + \left(-\frac{32}{17}\right)^2 i^2 = \frac{1849}{289} + \frac{2752}{289}i - \frac{1024}{289} = \frac{825}{289} + \frac{2752}{289}i$

由韦达定理得：$z_1 + z_2 = -m$, $z_1 z_2 = n$

所以 $m = -\left(-\frac{43}{17} - \frac{32}{17}i - \text{共轭根}\right) = \frac{86}{17}$

$n = \left(-\frac{43}{17} - \frac{32}{17}i\right)\cdot\left(-\frac{43}{17} + \frac{32}{17}i\right) = \left(\frac{43}{17}\right)^2 + \left(\frac{32}{17}\right)^2 = \frac{1849 + 1024}{289} = \frac{2873}{289} = 10$
