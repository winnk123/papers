1. 已知集合 $A=\left\{x\left|\frac{4}{x+2}\geq1\right.\right\}$, $B=\{x|x^2-3x<0\}$, 则 $A\cup B=$ ( ) 【答案：C】
A. $\{x|x\leq2\text{ 或 }x\geq3\}$     B. $\{x|-2<x<3\}$     C. $\{x|0<x\leq2\}$     D. $\{x|x\leq-2\text{ 或 }x\geq3\}$

2. 已知 $z=2-i$, 则 $z(\overline{z}+i)=$ ( ) 【答案：C】
A. $6-2i$     B. $4-2i$     C. $6+2i$     D. $4+2i$

3. 已知命题 $p$: “$\exists x\leq0,\ x^2+3x-4=0$”, 则命题 $p$ 的真假及命题 $p$ 的否定分别为 ( ) 【答案：B】
A. 真命题, $\exists x\geq0,\ x^2+3x-4=0$     B. 真命题, $\forall x\leq0,\ x^2+3x-4\neq0$  
C. 假命题, $\exists x\geq0,\ x^2+3x-4=0$     D. 假命题, $\forall x\leq0,\ x^2+3x-4\neq0$

4. 已知 $\alpha$ 为锐角, 且 $\cos\left(\alpha+\frac{\pi}{4}\right)=-\frac{1}{2}$, 则 $\cos\left(\alpha+\frac{3\pi}{4}\right)=$ ( ) 【答案：C】
A. $-\frac{1}{2}$     B. $\frac{1}{2}$     C. $-\frac{\sqrt{3}}{2}$     D. $\frac{\sqrt{3}}{2}$

5. 已知向量 $\mathbf{a}=(1,1)$, $\mathbf{b}=(1,-1)$. 若 $(\mathbf{a}+\lambda\mathbf{b})\perp(\mathbf{a}+\mu\mathbf{b})$, 则 ( ) 【答案：D】
A. $\lambda+\mu=1$     B. $\lambda+\mu=-1$     C. $\lambda\mu=1$     D. $\lambda\mu=-1$
