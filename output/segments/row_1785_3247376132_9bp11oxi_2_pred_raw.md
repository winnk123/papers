15. (本小题满分 13 分)
已知集合 $A = \left\{x \mid x < -2 \text{ 或 } x > \frac{3}{2}\right\}, B = \left\{x \mid \frac{2}{x-2} < -1\right\}, C = \left\{x \mid |x - a| < 1\right\}$.
(1) 求 $A \cup B, \complement_{\mathbb{R}}(A \cap B)$;
(2) 若 $A \cup C = A$, 求实数 $a$ 的取值范围.

16. (本小题满分 15 分)
已知函数 $f(x) = x - 2, g(x) = e^x + \frac{ax}{2}$.
(1) 若 $a = e^2$, 求函数 $g(x)$ 在 $(0, g(0))$ 处的切线方程;
(2) 若 $a > -e$, 且函数 $h(x) = f(x) \cdot g(x)$, 讨论函数 $h(x)$ 的单调性.
