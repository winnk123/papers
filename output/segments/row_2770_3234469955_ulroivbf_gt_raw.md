一、选择题

1. 已知集合 $U = \{1,2,3,4,5\}, A = \{1,3\}, B = \{2,3,5\}$，则 $C_U(A \cup B) = $ ( )
A. $\{1,2,3,4\}$     B. $\{2,3,4\}$     C. $\{2,4\}$     D. $\{4\}$
【答案：D】

2. 已知全集 $U = \mathbb{R}$，集合 $A = \{x \mid x < -1 \text{ 或 } x > 4\}, B = \{x \mid -2 \leq x \leq 3\}$，那么阴影部分表示的集合为 ( )
![插图](../9dc8e3f5-fd4b-451d-a654-589cf4c1f002.jpg)
A. {x | -1 ≤ x < 3}
B. {x | -1 ≤ x ≤ 3}
C. {x | x ≤ 3 或 x ≥ 4}
D. {x | -2 ≤ x ≤ 4}
【答案：B】

3. 命题：“$\exists x > 0, x^2 - 2x > 0$”的否定是 ( )
A. $\forall x > 0, x^2 - 2x > 0$     B. $\forall x > 0, x^2 - 2x \leq 0$     C. $\exists x \leq 0, x^2 - 2x > 0$     D. $\exists x \leq 0, x^2 - 2x \leq 0$
【答案：B】

4. 已知非空集合 $P$ 满足：(1) $P \subseteq \{1,2,3,4,5\}$；(2) 若 $a \in P$，则 $6 - a \in P$。符合上述要求的集合 $P$ 的个数是 ( )
A. 4     B. 5     C. 7     D. 31
【答案：C】

5. 已知 $p$ 是 $q$ 的充分不必要条件，$q$ 是 $s$ 的充要条件，$s$ 是 $r$ 的充分不必要条件，$r$ 是 $q$ 的必要不充分条件，则 $p$ 是 $s$ 的 ( )
A. 充分不必要条件     B. 必要不充分条件     C. 充要条件     D. 既不充分也不必要条件
【答案：A】

6. 若 $p: a \in \mathbb{R}, |a| < 1, q:$ 关于 $x$ 的二次方程 $x^2 + (a+1)x + a - 2 = 0$ 的一个根大于零，另一个根小于零，则 $p$ 是 $q$ 的 ( )
A. 充分不必要条件     B. 必要不充分条件     C. 充要条件     D. 既不充分也不必要条件
【答案：A】

7. 已知命题 $p: \forall x \in [1,2], x^2 - a \geq 0$，命题 $q: \exists x \in \mathbb{R}, x^2 + 2ax + 2 - a = 0$。若命题 $p$ 和 $q$ 都是真命题，则实数 $a$ 的取值范围为 ( )
A. $a \leq -2$ 或 $a = 1$     B. $a \leq -2$ 或 $1 \leq a \leq 2$     C. $a \geq 1$     D. $-2 \leq a \leq 1$
【答案：A】

8. 某小学对小学生的课外活动进行了调查。调查结果显示：参加舞蹈课外活动的有63人，参加唱歌课外活动的有89人，参加体育课外活动的有47人，三种课外活动都参加的有24人，选择两种课外活动参加的有46人，不参加其中任何一种课外活动的有15人。则接受调查的小学生共有多少人
A. 120人 B. 144人 C. 177人 D. 192人
【答案：A】

9. 命题“∃x ∈ [-1, 2], (1/2)x² + x - 3/2 - a ≥ 0”为真命题的一个必要不充分条件是
A. a ≤ 0, B. a ≤ 1, C. a ≤ 2, D. a ≤ 3

10. 若集合 $A_1, A_2$ 满足 $A_1 \cup A_2 = A$，则称 $(A_1, A_2)$ 为集合 $A$ 的一个分拆，并规定：当且仅当 $A_1 = A_2$ 时，$(A_1, A_2)$ 与 $(A_2, A_1)$ 为同一种分拆。则集合 $A = \{0,1,2\}$ 的不同分拆种数为 ( )
A. 8     B. 16     C. 20     D. 27
【答案：D】

11. 设 $U$ 为全集，若 $A, B$ 是两个不相等的非空集合，且 $A \cup B = A$，则 ( )
A. B ⊆ A
B. A ⊆ B
C. A ∪ (CᵤB) = U
D. (CᵤA) ∪ B = U
【答案：AC】

12. 下列四个命题：其中正确的命题为（    ）
A. 已知集合 A = {y | y = x² + 1}，集合 B = {x | y = x² + 1}，则 A = B
B. 集合 {x ∈ N | 5/(x+1) ∈ N} 的非空真子集有 2 个
C. 已知集合 M = {1, m, m² + 3}，且 4 ∈ M，则 m 的取值构成的集合为 {-1, 1, 4}
D. 记 A = {x | x = 2k + 1, k ∈ Z}，B = {x | x = 4n + 1, n ∈ Z}，则 B ⊆ A
【答案：BD】

13. （多选）设命题 $p: \exists x \in \mathbb{R}, ax^2 + 2ax - 4 \geq 0$ 为假命题，则实数 $a$ 的值可以是 ( )
A. $-3$     B. $0$     C. $4$     D. $-2$
【答案：ABD】

14. 已知集合 M = {x | x = m + 1/2, m ∈ Z}，N = {x | x = 2n - 3/2, n ∈ Z}，P = {x | x = p/2, p ∈ Z}，则（    ）
A. N ⊆ P
B. P ⊆ M
C. N ⊆ M
D. M ⊆ N
【答案：ACD】
