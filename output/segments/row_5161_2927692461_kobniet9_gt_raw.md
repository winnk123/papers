一、选择题
1. 在下列选项中，能正确表示集合 $A=\{-1, 0, 1\}$ 和 $B=\{y|y=\sin x\}$ 关系的是 ( ) 【答案：A】
A. $$A=B$$     B. $$A \subseteq B$$     C. $$A \supseteq B$$     D. $$A \cap B = \varnothing$$

2. 将 $330^\circ$ 化为弧度为 ( ) 【答案：A】
A. $$\frac{11\pi}{6}$$     B. $$\frac{4\pi}{3}$$     C. $$\frac{5\pi}{3}$$     D. $$\frac{5\pi}{4}$$

3. “角 $\theta$ 为第三象限角” 是 “$\cos \theta < 0$” 的 ( ) 【答案：B】
A. 充分不必要条件     B. 必要不充分条件
C. 充要条件           D. 既不充分也不必要条件

4. 下列函数中，既是 $(0, \frac{\pi}{2})$ 上的减函数，又是以 $\pi$ 为周期的偶函数的是 ( ) 【答案：B】
A. $$y=-\tan x$$     B. $$y=\cos 2x$$     C. $$y=\sin 2x$$     D. $$y=2\cos x$$

5. 若 $a=\cos(-1)$, $b=\cos 2$, $c=\cos \pi$，则 $a$, $b$, $c$ 的大小关系为 ( ) 【答案：C】
A. $$a < b < c$$     B. $$b < c < a$$     C. $$c < a < b$$     D. $$c < b < a$$

6. 函数 $f(x)=ax^3 + b\sin x + 2$ 且 $f(2025)=4$，则 $f(-2025)$ 的值为 ( ) 【答案：A】
A. $$0$$     B. $$2$$     C. $$4$$     D. $$6$$

7. 已知$\theta$是第二象限角，$\tan\theta$, $\cos\theta$是方程 $2x^2 + ax + \sqrt{2} = 0$ 的两根，则 $a=$ ( ) 【答案：D】  
A. $\sqrt{3} + 1$  
B. $1 - \sqrt{3}$  
C. $\sqrt{2} - 2$  
D. $2 + \sqrt{2}$

8. 函数$f(x) = \left|\sin\left(x + \frac{\pi}{3}\right)\right| - 1, x \in [0, a]$，若$f(x)$恰好有5个零点，则$a$的取值范围是 ( ) 【答案：D】  
A. $\left[\dfrac{17\pi}{2}, \dfrac{21\pi}{2}\right)$  
B. $\left[\dfrac{16\pi}{3}, \dfrac{35\pi}{6}\right)$  
C. $\left[\dfrac{14\pi}{3}, \dfrac{17\pi}{3}\right)$  
D. $\left[\dfrac{25\pi}{6}, \dfrac{31\pi}{6}\right)$
