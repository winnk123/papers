一、选择题

1. 集合 $\{x \in \mathbb{N} \mid 2x + 5 \leq 11\}$ 用列举法表示为 ( ) 【答案：A】
A. $\{0,1,2,3\}$     B. $\{1,2,3\}$     C. $\{0,1,2,3,4\}$     D. $\{1,2,3,4\}$

2. 命题“$\exists x \in \mathbb{R}, x^5 - 2x < 0$”的否定为 ( ) 【答案：C】
A. $\exists x \in \mathbb{R}, x^5 - 2x \geq 0$     B. $\forall x \in \mathbb{R}, x^5 - 2x \geq 0$
C. $\forall x \in \mathbb{R}, x^5 - 2x \leq 0$     D. $\exists x \in \mathbb{R}, x^5 - 2x \geq 0$

3. 某投资方对某项目提出两个投资方案：方案一为一次性投资 1000 万元；方案二为第一年投资 200 万元，以后每年投资 30 万元。下列不等式表示“经过 $n$ 年后，方案一的总投资不多于方案二的总投资”的是 ( ) 【答案：B】
A. $30n + 170 \geq 1000$     B. $30n + 170 \geq 1000$     C. $30n + 200 \geq 1000$     D. $30n + 170 > 1000$

4. 设 $A, B$ 是两个集合，则“A $\cap$ B = $\varnothing$”是“A 与 B 之一为 $\varnothing$”的 ( ) 【答案：D】
A. 充要条件     B. 充分不必要条件     C. 必要不充分条件     D. 既不充分也不必要条件

5. 下列式子的值比 $(x+4)(x+7)$ 的值大的是 ( ) 【答案：D】
A. $(x+1)(x+10)$     B. $(x+2)(x+9)$     C. $(x+3)(x+8)$     D. $(x+5)(x+6)$

6. 定义一种新的集合运算 $\odot$: $A \odot B = \{x \mid x \in A \text{ 且 } x \notin B\}$，若集合 $A = \{x \mid x^2 - 7x + 6 < 0\}$，$B = \{x \mid 3 \leq x \leq 8\}$，则 $B \odot A =$ ( ) 【答案：D】
A. $\{x \mid 3 \leq x < 6\}$     B. $\{x \mid 1 < x < 3\}$     C. $\{x \mid 6 < x \leq 8\}$     D. $\{x \mid 6 \leq x \leq 8\}$

7. 若 $\forall x \in \mathbb{R}, mx^2 + 2mx + 5 \geq 0$，则 $m$ 的取值集合为 ( ) 【答案：C】
A. $\{m \mid 0 < m < 5\}$     B. $\{m \mid 0 \le m \leq 5\}$     C. $\{m \mid 0 \leq m \leq 5\}$     D. $\{m \mid m \geq 5\}$

8. 某班级共 40 位同学暑期去 A 馆、B 馆、C 馆三个馆打卡的情况如下：每位同学至少去其中一个馆打卡，既去了 A 馆打卡又去了 B 馆打卡的人数为 8，既去了 B 馆打卡又去了 C 馆打卡的人数为 10，既去了 A 馆打卡又去了 C 馆打卡的人数为 9，三个馆都去打卡的人数为 7，则仅去了其中一个馆打卡的人数为 ( ) 【答案：A】
A. 13     B. 34     C. 20     D. 27

9. 已知集合 $\{0, -2, 2a\} = \{a-2, -|a|, a+2\}$，则 $a$ 的值可能为 ( ) 【答案：A】
A. 2     B. 0     C. -2     D. 4

10. 设正数 $m, n$ 满足 $m + 3n = 6$，则 ( ) 【答案：D】
A. $mn \geq 2$     B. $mn \leq 3$     C. $\sqrt{m} + \sqrt{3n} \leq 2\sqrt{3}$     D. $m^2 + 9n^2 \geq 18$

11. 已知 $a, b, c$ 为三个互不相等的正整数，命题 $p: a - b < 0$，命题 $q: b - c < 0$，命题 $s: c - a < 0$。若 $a, b, c$ 只需满足三个命题 $p, q, s$ 中仅有两个是真命题，则 $(a, b, c) \in A$。若 $(x_1, x_2, x_3) \in A, (x_3, x_4, x_1) \in A$，则下列结论一定成立的是 ( ) 【答案：B】
A. $(x_2, x_4, x_1) \in A$     B. $(x_4, x_1, x_3) \notin A$     C. $(x_3, x_2, x_4) \notin A$     D. $(x_3, x_1, x_2) \in A$

二、填空题

12. 已知 $2 < a < 8, 2 < b < 8$，设 $t = \frac{a}{b}$，则 $t$ 的取值集合是 ______ 【答案：$\left\{ t \mid \frac{1}{4} < t < 4 \right\}$】

13. 已知正数 $x, y$ 满足 $\frac{7}{x} + \frac{1}{y} = 2$，则 $x + 7y$ 的最小值是 ______ 【答案：$4$】

14. 设集合 $A = \{x \in \mathbb{N} \mid x^3 < 150\}$，$B = \{x \mid -x^2 + (4a+2)x - 3a^2 - 6a > 0\}$，则 $A$ 的真子集个数为 ______，若集合 $A \cap B$ 中只有 2 个元素，则 $a$ 的取值集合是 ______ 【答案：$31$；$\left\{ a \mid -\frac{1}{3} < a < \frac{1}{3} \right\}$】

三、解答题

15. (13 分) 已知集合 $A = \{x \mid 2 - m < x < 2 + m\}$，$B = \{x \mid x \leq 1 \text{ 或 } x \geq 6\}$。
(1) 当 $m = 3$ 时，求 $A \cap B, A \cup B$；
(2) 若 $A \cap (\complement_{\mathbb{R}} B) = \varnothing$，证明：$m \leq 0$。
