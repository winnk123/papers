[条件缺失]

5. 若二次函数 $y = -x^2 + 2x + m$ 在区间 $(1, +\infty)$ 上有且仅有一个零点，则实数 $m$ 的取值范围是 ( )
A. $(-1, +\infty)$   B. $[-1, +\infty)$   C. $(-\infty, -1]$   D. $(-\infty, -1]$
【答案：C】
A. $(-1, +\infty)$   B. $[-1, +\infty)$   C. $(-\infty, -1]$   D. $(-\infty, -1]$

6. “$\forall x \in \mathbb{R}, x^2 + 2x - 3 - a > 0$” 的一个必要不充分条件是 ( )
A. $a < -4$   B. $a > -4$   C. $a < -3$   D. $a < -5$
【答案：D】
A. $a < -4$   B. $a > -4$   C. $a < -3$   D. $a < -5$

7. 已知正数 $a, b$ 满足 $a + b = 2$，则 ( )
A. $ab \geq 1$   B. $a^2 + b^2 \geq 2$   C. $\sqrt{a} + \sqrt{b} \geq 2$   D. $\frac{1}{a} + \frac{1}{b} \geq 2$
【答案：B】
A. $ab \geq 1$   B. $a^2 + b^2 \geq 2$   C. $\sqrt{a} + \sqrt{b} \geq 2$   D. $\frac{1}{a} + \frac{1}{b} \geq 2$

8. 若存在 $x \in \mathbb{R}$ 使得不等式 $ax^2 - |x| + 4a \leq 0$ 成立，则实数 $a$ 的最大值为 ( )
A. $\frac{1}{4}$   B. $\frac{1}{2}$   C. 2   D. 4
【答案：A】
A. $\frac{1}{4}$   B. $\frac{1}{2}$   C. 2   D. 4

9. 已知 $M = \{x | x^2 - 4x + 3 = 0\}$, $N = \{x | ax - 1 = 0\}$，若 $N \subseteq M$，则 $a$ 的值可能是 ( )
A. -1   B. 0   C. $\frac{1}{3}$   D. 1
【答案：C】
A. -1   B. 0   C. $\frac{1}{3}$   D. 1

10. 设 $a, b, c \in \mathbb{R}$，若关于 $x$ 的不等式 $ax^2 + bx + c > 0$ 的解集为 $\{x | -1 < x < 3\}$，则 ( )
A. $a < 0$   B. 关于 $x$ 的不等式 $bx + c > 0$ 的解集为 $(\frac{3}{2}, +\infty)$   C. 关于 $x$ 的不等式 $bx^2 - cx - a < 0$ 的解集为 $(\frac{1}{2}, 1)$   D. $\frac{3b^2 - c^2 + 1}{2a}$ 的最大值为 $-\sqrt{3}$
【答案：B】
A. $a < 0$   B. 关于 $x$ 的不等式 $bx + c > 0$ 的解集为 $(\frac{3}{2}, +\infty)$   C. 关于 $x$ 的不等式 $bx^2 - cx - a < 0$ 的解集为 $(\frac{1}{2}, 1)$   D. $\frac{3b^2 - c^2 + 1}{2a}$ 的最大值为 $-\sqrt{3}$
