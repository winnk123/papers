一、选择题
1. 已知集合 $A=\{x|x^2-3x+2<0\}, B=\{x|x<a\}$, 则 $A \subseteq B$ 成立的充要条件是 ( ) 【答案：D】
A. $$a<1$$     B. $$a \leq 1$$     C. $$a>2$$     D. $$a \geq 2$$

2. 已知复数 $z_1=(1-i)(2+i), z_2=1+2i$, 则 $z_2 - z_1 =$ ( ) 【答案：C】
A. $$-3i$$     B. $$4+i$$     C. $$-2+3i$$     D. $$2-3i$$

3. 在空间中，下列命题正确的是 ( ) 【答案：D】
A. 若三条直线两两相交，则这三条直线一定共面
B. 若平面 $\alpha$ 与平面 $\beta$ 平行，直线 $l$ 在 $\alpha$ 内，直线 $m$ 在 $\beta$ 内，则 $l // m$
C. 若直线 $A_1B_1 \perp A_2B_2, B_1C_1 \perp B_2C_2$, 则 $\angle A_1B_1C_1$ 与 $\angle A_2B_2C_2$ 相等或互补
D. 若三个平面两两相交得三条交线，则这三条交线共点或互相平行

4. 某学校在读书节活动中，甲、乙、丙 3 个班各有 2 名同学获奖，现将这 6 人站成一排拍照，其中甲班的 2 名同学相邻，且乙班的 2 名同学不相邻的站法种数共有 ( ) 【答案：C】
A. 36 种     B. 72 种     C. 144 种     D. 288 种

5. 已知平面向量 $\vec{a}=(1,0), \vec{b}=(0,1), \lambda, \mu \in \mathbf{R}$, 若 $(\vec{a}+\lambda \vec{b}) \perp (\vec{a}+\mu \vec{b})$, 则 ( ) 【答案：B】
A. $(\vec{a}-\lambda \vec{b}) \perp (\mu \vec{a}+\vec{b})$
B. $(\vec{a}-\lambda \vec{b}) // (\mu \vec{a}+\vec{b})$
C. $(\vec{a}-\lambda \vec{b}) \perp (\mu \vec{a}-\vec{b})$
D. $(\vec{a}-\lambda \vec{b}) // (\mu \vec{a}-\vec{b})$

6. 设双曲线 $C: \frac{x^2}{a^2} - \frac{y^2}{b^2} = 1 (a>0, b>0)$ 的右焦点为 $F$, 左、右顶点分别为 $A_1, A_2$, $M$ 为 $C$ 上一点，且 $MF \perp x$ 轴，点 $N$ 在线段 $MF$ 上，直线 $A_1N$ 交 $y$ 轴于 $H$ 点，直线 $A_2N$ 交 $y$ 轴于 $G$ 点，$O$ 为坐标原点，若 $|OG|=2|OH|$, 则 $C$ 的离心率为 ( ) 【答案：A】
A. $$3$$     B. $$2$$     C. $$\sqrt{3}$$     D. $$\sqrt{2}$$
