一、选择题

1. 已知集合 $A = \{0, a\}$, $B = \{1, a^2 - 2, 1 - a\}$，若 $A \subseteq B$，则 $a$ 的值为 ( )
A. $$1$$     B. $$\frac{1}{2}$$     C. $$\pm\sqrt{2}$$     D. $$2\text{ 或 }-1$$
【答案：A】

2. 已知用 $\alpha$ 的顶点在坐标原点，始边与 $x$ 轴非负半轴重合，终边过点 $P\left(-\frac{1}{2}, \frac{\sqrt{3}}{2}\right)$，则 $\sin 2\alpha =$ ( )
A. $$\frac{\sqrt{3}}{2}$$     B. $$-\frac{\sqrt{3}}{2}$$     C. $$\frac{\sqrt{3}}{4}$$     D. $$-\frac{1}{2}$$
【答案：B】

3. 不等式 $\frac{x}{1 - x} \geq 1$ 的解集是 ( )
A. $$\left\{x \middle| \frac{1}{2} \leq x < 1\right\}$$     B. $$\left\{x \middle| x \leq \frac{1}{2} \text{ 或 } x > 1\right\}$$
C. $$\left\{x \middle| \frac{2}{3} \leq x < 1\right\}$$     D. $$\left\{x \middle| x \leq \frac{2}{3} \text{ 或 } x > 1\right\}$$
【答案：C】

4. 若 $\{e_1, e_2\}$ 是平面内的一个基底，则下列四组向量中能作为平面的基底的是 ( )
A. $$\left\{e_1 - \frac{1}{2}e_2, \frac{1}{2}e_2 - e_1\right\}$$     B. $$\left\{2e_1 - e_2, e_1 - \frac{1}{2}e_2\right\}$$
C. $$\left\{2e_1 - 3e_2, 6e_1 - 4e_2\right\}$$     D. $$\left\{e_1 + e_2, e_1 + 2e_2\right\}$$
【答案：D】

5. 已知函数 $f(x) = 2^x + x - 3/2$, $g(x) = \log_2 x + x - 3/2$, $h(x) = 2x - \frac{3}{2}$ 的零点分别为 $a, b, c$，则 ( )
A. $$a > b > c$$     B. $$b > c > a$$     C. $$c > a > b$$     D. $$b > a > c$$
【答案：C】

6. 已知 $D, E$ 分别是 $\triangle ABC$ 的边 $AB, AC$ 上的点，且满足 $\overrightarrow{AB} = 2\overrightarrow{AD}$, $\overrightarrow{AC} = 3\overrightarrow{EC}$, $BE$ 与 $CD$ 相交于点 $O$，连接 $AO$ 并延长交 $BC$ 于点 $F$，若 $\overrightarrow{AO} = \lambda \overrightarrow{AF}$，则实数 $\lambda$ 的值为 ( )

8. 设函数 $f(x)$ 的定义域为 $\mathbb{R}$, $f(x+1)$ 是偶函数，$f(x+2)$ 是奇函数，且当 $x \in [1,2]$ 时，$f(x) = ax^2 + b$，若 $f(0) + f(3) = 4$，则 $f\left(\frac{17}{2}\right) =$ ( )
A. $$-3$$     B. $$-\frac{7}{3}$$     C. $$\frac{7}{3}$$     D. $$3$$
【答案：B】

9. 下列说法正确的是 ( )
A. 函数 $f(x) = a^{2x-2} - 3$ ($a > 0$ 且 $a \neq 1$) 的图象恒过点 $(1, -2)$
B. 函数 $f(x) = \log_2 2^x$ 与 $g(x) = \frac{x^2}{x}$ 表示同一个函数
C. 函数 $f(x) = \sqrt{x^2 + 3} + \frac{1}{\sqrt{x^2 + 3}} + 1$ 的最小值为 3
D. 若关于 $x$ 的不等式 $ax^2 + bx + c < 0$ 的解集为 $\{x | x < -2 \text{ 或 } x > 1\}$，则 $abc > 0$
【答案：A、D】

10. 已知点 $M$ 为 $\triangle ABC$ 所在平面内一点，则下列说法正确的是 ( )
A. 若 $\angle ABC = \frac{\pi}{6}$, $AB = 4$, $BC = 5$，则 $\overrightarrow{AB}$ 在 $\overrightarrow{BC}$ 上的投影向量为 $\frac{2\sqrt{3}}{5}\overrightarrow{BC}$
B. 若 $\overrightarrow{MA}, \overrightarrow{MB}, \overrightarrow{MC}$ 两两的夹角相等，且 $|\overrightarrow{MA}| = 1$, $|\overrightarrow{MB}| = 1$, $|\overrightarrow{MC}| = 3$，则 $|\overrightarrow{MA} + \overrightarrow{MB} + \overrightarrow{MC}| = 2$
C. 若 $\left(\frac{\overrightarrow{AB}}{|\overrightarrow{AB}|} + \frac{\overrightarrow{AC}}{|\overrightarrow{AC}|}\right) \cdot \overrightarrow{BC} = 0$，且 $\left|\frac{\overrightarrow{AB}}{|\overrightarrow{AB}|} \cdot \frac{\overrightarrow{AC}}{|\overrightarrow{AC}|}\right| = \frac{1}{2}$，则 $\triangle ABC$ 为等边三角形
D. 若 $\overrightarrow{AM} = x\overrightarrow{AB} + y\overrightarrow{AC}$，且 $x + y = \frac{1}{3}$，则 $\triangle MBC$ 的面积是 $\triangle ABC$ 面积的 $\frac{2}{3}$
【答案：A、C】
