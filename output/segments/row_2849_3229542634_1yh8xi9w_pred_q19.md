19. (17分) 问题：正数 a, b 满足 a + b = 1 ，求 (1)/(a) + (1)/(b) 的最小值。

有一种解法是： (1)/(a) + (1)/(b) = ((1)/(a) + (1)/(b))(a + b) = 1 + (b)/(a) + (a)/(b) + 1 ≥ 4 ，当且仅当 (b)/(a) = (a)/(b) 且 a + b = 1 时，即 a = (1)/(2) 且 b = (1)/(2) ， (1)/(a) + (1)/(b) 取得最小值 4。
