12. 在二项式 $\left(\frac{1}{x} - x\right)^5$ 的展开式中，含 $x^3$ 项的系数为 ______ 【答案：$-5$】

13. 已知数列 $\{a_n\}$ 满足 $a_1 = 1$, $a_n + a_{n+1} = 2^n$，则 $a_1 + a_2 + a_3 + \cdots + a_{2n-1} =$ ______ 【答案：$\frac{4^n - 1}{3}$】

14. 设 $a, b, c, d, e, f$ 为数字 $1, 2, 3, 4, 5, 6$ 的一个排列，记三位数 $m = \overline{abc}$, $n = \overline{def}$，其中 $\overline{abc} = 100a + 10b + c$，例 $\overline{321} = 100 \times 3 + 10 \times 2 + 1$，则 $|m - n|$ 的值大于 $400$ 的概率为 ______ 【答案：$\frac{2}{15}$】

15. (13分) 记 $\triangle ABC$ 的内角 $A, B, C$ 的对边分别为 $a, b, c$，已知 $\frac{c + b}{a} = \frac{\sqrt{3} \sin C - \sin A}{\sin C - \sin B}$。
(1) 求角 $B$ 的大小；
(2) 若点 $D$ 是 $BC$ 的中点，且 $AD = 2$, $AB = 2\sqrt{3}$，求 $\triangle ABC$ 的周长。


16. (15分) 已知正项数列 $\{a_n\}$ 满足 $a_1 = 99$, $a_{n+1} = a_n^2 + 2a_n$。
(1) 求证：$\{\lg(a_n + 1)\}$ 是等比数列。
(2) 设 $b_n = \frac{1}{\lg(a_n + 1) - 1}$，记数列 $\{b_n\}$ 的前 $n$ 项和为 $S_n$，求证：$S_n < 2$。
