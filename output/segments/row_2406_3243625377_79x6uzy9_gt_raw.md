一、选择题

1. 已知集合 $A = \{0,1,2\}, B = \{x \in \mathbb{N} \mid 0 < x < 4\}$，则 $A \cup B =$  
A. $\{1,2\}$  
B. $\{0,1,2,3\}$  
C. $\{0,1,2,3,4\}$  
D. $\{x \mid 0 \leq x < 4\}$  
【答案：B】

2. 若复数 $z$ 满足 $z \cdot \bar{z} + z = 1 - i$，则 $z$ 的虚部为  
A. $-1$  
B. $-\frac{1}{2}$  
C. $1$  
D. $2$  
【答案：B】

3. 已知等差数列 $\{a_n\}$ 的公差不为 $0$，a_1,a_2,a_3成等比数列，且 $a_2 = 2a_1 + 1$，则公差 $d =$  
A. $1$  
B. $2$  
C. $3$  
D. $4$  
【答案：C】

4. 已知平面内有四点 $O,A,B,C$，若 $\overrightarrow{OC} = x\overrightarrow{OA} + y\overrightarrow{OB}$，则“$A,B,C$ 三点共线”是“$x+y=1$”的  
A. 充分不必要条件  
B. 必要不充分条件  
C. 充要条件  
D. 既不充分也不必要条件  
【答案：B】

5. 若 $\alpha,\beta$ 是锐角，满足 $\sin\frac{\alpha}{2} = 1 + \cos\alpha$，则 $\tan(\alpha - 2\beta) = \tan[\alpha - (\pi - \beta)] = \tan\alpha = 0$  
A. $0$  
B. $\sqrt{3}/3$  
C. $\sqrt{3}$  
D. $\sqrt{5}$  
【答案：A】

6. 已知双曲线 $C: \frac{x^2}{a^2} - \frac{y^2}{b^2} = 1 (a>0, b>0)$ 的左顶点为 $A$，右焦点为 $F$，$P$ 为双曲线右支上的点，若双曲线的离心率为 $2$，且 $\angle PAF = 30^\circ$，则 $\angle PFA = \theta$。
A. $30^\circ$
B. $45^\circ$
C. $60^\circ$
D. $120^\circ$
【答案：D】

7. 甲、乙两人玩扑克牌游戏，他们拿出了 $2k$ 张牌 ($k \in \mathbb{N}^*$ 且 $3 \le k \le 10$)，其中“红心”花色的牌共有 4 张。将牌洗匀后合为一堆，两人依次从牌堆最上方拿一张牌（甲先拿牌），直到拿完所有牌为止。记事件 A 为“甲、乙二人手中各有两张‘红心’花色的牌”，事件 B 为“甲、乙二人其中一人手中只有一张‘红心’花色的牌”，则
A. 对任意满足题意的 $k$，都有 $P(A) > P(B)$
B. 对任意满足题意的 $k$，都有 $P(A) < P(B)$
C. 对任意满足题意的 $k$，都有 $P(A) = P(B)$
D. $P(A), P(B)$ 的大小关系与 $k$ 有关
【答案：D】

8. 设定义在 $\mathbb{R}$ 上的奇函数 $f(x)$ 的导函数为 $f'(x)$，且对任意的 $x \in \mathbb{R}, f'(x+4) = -f'(x), f(1)=2$，则 $f(2027)$ 的值为
A. $-2$
B. $0$
C. $2$
D. $2026$
【答案：C】

9. 已知函数 $f(x) = \sqrt{x^2 + 1} + x$，则下列结论正确的是
A. $\ln(f(x))$ 是奇函数
B. $f(x)$ 是单调递增函数
C. $f(x)$ 的图象关于点 $(0,1)$ 对称
D. 若 $(\sqrt{x^2 + 9} + x)(\sqrt{y^2 + 9} + y) = 9$，则 $x^2 = y^2$。
【答案：B、C】

10. 在数字通信中，信号是由数字 0 和 1 组成的序列。由于随机因素的干扰，发送的信号 0 或 1 有可能被错误地接收为 1 或 0。已知发送信号 0 时，接收为 0 和 1 的概率分别为 0.9 和 0.1；发送信号 1 时，接收为 1 和 0 的概率分别为 $p$ 和 $(1-p)$。假设每次发送信号 0 和 1 是等可能的且每次发送信号相互之间是独立的。当发送两次信号是“0”“1”时，接收到的信号也是“0”“1”的概率为 0.72，则下列结论正确的是
A. $p=0.8$
B. 当发送两次信号是“1”“1”时，恰有一次被正确接收的概率为 0.16
C. 一次发信后被接收为信号“1”的概率为 0.45
D. 若已知一次发信后被接收为信号“1”，则接收正确的概率为 $\frac{8}{9}$
【答案：A、C、D】

11. 已知动点序列 $M_n(n, \sqrt{n}) (n \in \mathbb{N}^*)$ 与圆 $C: (x-2)^2 + y^2 = 9$，设 $d_n$ 表示 $M_n$ 到圆 $C$ 上的点的最短距离。则下列结论正确的是
A. 当 $n \in \mathbb{N}^*$ 时，$d_{n+1} > d_n$
B. 当 $n \ge 5$ 时，$d_n = \sqrt{(n-2)^2 + n} - 3$
C. 当 $n \ge 6$ 时，$\triangle OM_n M_{n+1}$ 的面积 $S_n$ 满足 $S_n > S_{n+1}$
D. 当 $n \ge 5$ 时，记 $T_n = \sum_{i=5}^{n} d_i$，则 $T_n$ 满足 $n^2 - 8n + 16 < 2T_n < n^2 - 5n + 4$
【答案：B、D】
