一、选择题

1. 设复数 $z$ 满足 $z = (1+i)i$，则 $|z|=$ ( ) 【答案：B】
A. $\sqrt{2}$     B. $2$     C. $\sqrt{5}$     D. $1$

2. 已知 $\vec{a} = (\lambda, -2)$，$\vec{b} = (1, 2)$，$\vec{a} \perp \vec{b}$，则 $\lambda =$ ( ) 【答案：C】
A. $-4$     B. $4$     C. $1$     D. $-1$

3. 已知函数 $f(x) = \sin x - \cos x$，则下列选项中是 $f(x)$ 的一个单调递增区间的是 ( ) 【答案：D】
A. $\left[-\frac{\pi}{2}, \frac{\pi}{2}\right]$     B. $\left[-\frac{\pi}{4}, \frac{3\pi}{4}\right]$     C. $\left[-\frac{3\pi}{4}, \frac{\pi}{4}\right]$     D. $\left[\frac{3\pi}{4}, \frac{7\pi}{4}\right]$

4. 已知全集 $U = \{x | x < 10, x \in \mathbb{N}^*\}$，集合 $A, B$ 是 $U$ 的子集，若 $(\complement_U A) \cap B = \{5, 7, 9\}$，$A \cap B = \{2\}$，$(\complement_U A) \cap (\complement_U B) = \{6, 8\}$，则集合 $A =$ ( ) 【答案：D】
A. $\{2, 3, 4\}$     B. $\{1, 2, 4\}$     C. $\{1, 2, 3\}$     D. $\{1, 2, 3, 4\}$

5. 已知平面 $\alpha, \beta$，直线 $a, b$，则下列结论正确的是 ( ) 【答案：C】
A. 若 $a \subset \alpha, b \parallel a$，则 $b \parallel \alpha$     B. 若 $a \parallel \beta, a \subset \alpha, b \subset \beta$，则 $a \parallel b$     C. 若 $a \parallel \alpha, b \perp \alpha$，则 $a \perp b$     D. 若 $\alpha \parallel \beta, a \parallel \alpha$，则 $a \parallel \beta$

6. 已知首项为 $1$ 的数列 $\{a_n\}$，其前 $n$ 项积是公差为 $3$ 的等差数列，则 $a_5 =$ ( ) 【答案：A】
A. $4$     B. $3$     C. $\frac{7}{4}$     D. $\frac{10}{7}$

7. 已知甲、乙、丙、丁四位老师参加青年教师教学大赛，问其比赛结果，他们回答如下：
甲：丙第一，乙第二；
乙：丙第二，丁第三；
丙：丁最后，甲第二；
丁：如果每个人的两个回答中，都恰有一个是正确的，而且没有并列名次，那么这次比赛获得第一、二、三、四名依次是 ( ) 【答案：C】
A. 丙、甲、丁、乙     B. 丙、甲、乙、丁     C. 甲、乙、丙、丁     D. 甲、乙、丁、丙

8. $f(x) = 2x^3 - 3x^2 - 12x$，已知 $b < 0$。若 “$f(x) > a$” 的充要条件是 “$x > b$”，则实数 $b$ 的最大值为 ( ) 【答案：B】
A. $-2$     B. $-\frac{5}{2}$     C. $-1$     D. $-\frac{1}{2}$

9. 已知 $P$ 是圆 $C: (x-1)^2 + (y-2)^2 = 9$ 上的一个动点，过原点 $O$ 的动直线与圆 $C$ 交于 $M, N$ 两点，则下列说法正确的是 ( ) 【答案：B】
A. $|OP|$ 的最大值为 $3+\sqrt{5}$     B. $|OP|$ 的最小值为 $3-\sqrt{5}$     C. $|MN|$ 最大值为 $6$     D. $|MN|$ 最小值为 $2$

10. 某电商平台为了解用户对配送服务的满意度，从某地区随机抽取了 500 名用户进行问卷评分调查，将评分数据按 $[40,50), [50,60), \cdots, [90,100]$ 分组整理得到如下右方频率分布直方图，记该样本的平均数为 $\mu$，三个四分位数分别为 $a,b,c(a<b<c)$，则下列判断正确的是 ( ) 【答案：B】
A. $a-40 < b-a$     B. $c-b < 100-c$     C. $a,b,c$ 成等差数列     D. $\mu < b$

11. 已知函数 $f(x) = xe^x + (2-x)e^{2-x}$，则以下说法正确的是 ( ) 【答案：D】
