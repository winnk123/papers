二、填空题
12. 不等式 $|x - 1| + |x - 2| \le 3$ 的解集为 ______ 【答案：$[0,3]$】

13. 已知关于 $x$ 的方程 $x^2 - mx + 2m - 1 = 0$ 的两个不相等的实数根分别为 $x_1, x_2$，且 $x_1^2 + x_2^2 = 7$，则 $m =$ ______ 【答案：$-1$】

14. 若正实数 $a$、$b$ 满足 $2a + b = 2$，则 $\frac{2a^2}{a+1} + \frac{b^2 + 1}{b}$ 的最小值为 ______ 【答案：$4$】

三、解答题

15. (13分)  
已知集合 $A = \{x | x^2 + 2(a-1)x + a^2 - 1 = 0\}, B = \{x | x^2 - 4x = 0\}$。  
(1) 若 $0 \in A$，求实数 $a$ 的值；  
(2) 若 $C = \{x | mx + 2 = 0\}$，且 $C \subseteq B$，求 $m$ 的值；  
(3) 若 $(A \cup B) \subseteq B$，求实数 $a$ 的取值范围。

16. (15分)  
设集合 $P = \{x | -2 < x < 3\}, Q = \{x | 3a < x \le a + 1\}$。  
(1) 若 $C_R P \subseteq C_R Q$，求 $a$ 的取值范围；  
(2) 若 $P \cap Q = \varnothing$，求 $a$ 的取值范围。

17. (15分)  
已知正数 $a, b$ 满足 $a + 2b = ab$。  
(1) 求 $a + b$ 的最小值，并求此时 $a, b$ 的值；  
(2) 求 $\frac{2a}{a - 2} + \frac{8b}{b - 1}$ 的最小值，并求此时 $a, b$ 的值。
