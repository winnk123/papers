1. 已知命题 $P: \forall x>1, x^2+2x-3>0$，则 $\neg P$ 为 ( ) 【答案：B】
A. $$\exists x \leq 1, x^2+2x-3 \leq 0$$     B. $$\exists x > 1, x^2+2x-3 \leq 0$$
C. $$\forall x > 1, x^2+2x-3 < 0$$           D. $$\exists x > 1, x^2+2x-3 > 0$$

2. 设集合 $A = \left\{ x \middle| \frac{2-x}{x} \geq 0 \right\}$, $B = \left\{ x \middle| x^2 - 4x + 3 \leq 0 \right\}$，则 $A \cap (\complement_R B) =$ ( ) 【答案：D】
A. $$\{x | -1 < x < 2\}$$     B. $$\{x | 0 < x < 1\}$$     C. $$\{x | 0 < x \leq 2\}$$     D. $$\{x | 1 \leq x \leq 2\}$$

3. 下列两个函数是相同函数的有 ( ) 【答案：D】
A. $$f(x) = x - 1$$ 与 $$g(x) = \frac{x^2 - 1}{x + 1}$$     B. $$f(x) = \frac{(\sqrt{x})^2}{x}$$ 与 $$g(x) = 1$$
C. $$f(x) = x^0$$ 与 $$g(x) = 1$$                           D. $$f(x) = |x|$$ 与 $$g(x) = \sqrt{x^2}$$

4. 已知集合 $A = \{x | x^2 - 3x + 2 = 0, x \in \mathbb{R}\}$, $B = \{1,2,3,4,5\}$，则满足条件 $A \subseteq C$ 且 $C \subsetneqq B$ 的集合 $C$ 的个数为 ( ) 【答案：A】
A. 3     B. 4     C. 7     D. 8

5. 下列不等式中正确的是 ( ) 【答案：D】
A. $$x + \frac{1}{x} \geq 2$$     B. $$\frac{x^2 + 3}{\sqrt{x^2 + 2}} \geq 2$$     C. $$\frac{b}{a} + \frac{a}{b} \geq 2$$     D. $$x^2 + \frac{1}{x^2 + 1} \geq 1$$

6. 已知集合 $A = \{x | a + 1 < x < 2a - 3\}$, $B = \{x | -x^2 + 5x + 14 \leq 0\}$，则 $A \cap B = \varnothing$ 的一个必要不充分条件是 ( ) 【答案：A】
A. $$a \leq 5$$     B. $$a > 5$$     C. $$a \geq 7$$     D. $$a < 7$$

7. 若函数 $f(x)$ 的值域为 $(1,10)$，则函数 $g(x) = f(x) - 4\sqrt{f(x) - 1}$ 的值域为 ( ) 【答案：B】
A. $$[-3,1)$$     B. $$(-3,1)$$     C. $$[-3,-2)$$     D. $$(-3,-2)$$

8. 已知两个正数 $x,y$，满足 $x + 2y = 2$，则下列选项错误的是 ( ) 【答案：D】
A. $$xy$$ 的最大值为 $$\frac{1}{2}$$     B. $$\frac{2y}{x} + \frac{1}{y}$$ 的最小值 3
C. $$x^2 + 4y^2$$ 的最小值为 $$3$$     D. $$\left(x^2 + \frac{1}{5}\right)\left(4y^2 + \frac{1}{5}\right)$$ 的最小值为 $$\frac{21}{25}$$

9. 下列说法正确的是 ( ) 【答案：ABC】
A. 若 $a > b$，则 $\frac{1}{a} < \frac{1}{b}$     B. 若 $a > b > 0 > c$，则 $\frac{b}{a} < \frac{b - c}{a - c}$
C. 若 $a > b > c > d$，则 $a - c > d - b$     D. 若 $a < b < 0$，则 $\frac{1}{a} < \frac{1}{b}$
