[条件缺失]

1. 直线 $x + \sqrt{3}y - 2 = 0$ 的斜率为 ( )
A. $\sqrt{3}$     B. $-\sqrt{3}$     C. $-\frac{\sqrt{3}}{3}$     D. $\frac{\sqrt{3}}{3}$
【答案：D】

2. 点 $P(1,2,-3)$ 关于 $Ozx$ 平面对称的点的坐标是 ( )
A. $(1,2,3)$     B. $(1,-2,-3)$     C. $(-1,2,-3)$     D. $(1,-2,3)$
【答案：B】

3. 在平行六面体 $ABCD-A_1B_1C_1D_1$ 中，$\overrightarrow{AB} + \overrightarrow{AD} + \overrightarrow{AA_1} = ($ )
A. $\overrightarrow{CA_1}$     B. $\overrightarrow{AC_1}$     C. $\overrightarrow{DB_1}$     D. $\overrightarrow{BD_1}$
【答案：B】

4. 设 $x,y\in R$，向量 $\vec{a}=(0,1,2), \vec{b}=(2,y,2)$，若 $(\vec{a}-\vec{b})\cdot (\vec{a}+\vec{b}) = 0$，则 $|\vec{a}-\vec{b}|=$ ( )
A. $2\sqrt{2}$     B. $3$     C. $\sqrt{5}$     D. $2\sqrt{5}$
【答案：C】

5. 已知 $\vec{m}_1=(-1,9,1), \vec{m}_2=(m,-3,1)$，若 $(\vec{m}_1-\vec{m}_2)\perp (\vec{m}_1+\vec{m}_2)$，则 $m=$ ( )
A. $1$     B. $3$     C. $-1$     D. $-3$
【答案：B】

6. 已知 $a>0, b>0$，直线 $(a-1)x + 2y + 3 = 0$ 与直线 $x + by - 1 = 0$ 垂直，则 $\frac{1}{a} + \frac{1}{b}$ 的最小值是 ( )
A. $1$     B. $3$     C. $2 + \sqrt{2}$     D. $2 + 2\sqrt{2}$
【答案：C】

7. 若直线 $l$ 过定点 $P(1,0)$，且与以 $A(-1,2), B(2,\sqrt{3})$ 为端点的线段相交（包括端点），则直线 $l$ 的斜率范围是 ( )
A. $\left[\frac{\pi}{3}, \frac{3\pi}{4}\right]$     B. $\left[\frac{\pi}{6}, \frac{3\pi}{4}\right]$     C. $\left[0, \frac{\pi}{3}\right]\cup\left[\frac{3\pi}{4}, \pi\right]$     D. $\left[0, \frac{\pi}{3}\right]\cup\left[\frac{3\pi}{4}, \pi\right]$
【答案：D】

8. 在等腰直角三角形 $ABC$ 中，$AB=AC=4$，点 $P$ 是边 $AB$ 上异于 $A,B$ 的一点，光线从点 $P$ 出发经 $BC$ 反射后又回到点 $P$，若光线 $QR$ 经过 $\triangle ABC$ 的重心，则 $\triangle PQO$ 的周长等于 ( )
A. $\frac{4\sqrt{3}}{3}$     B. $\frac{8\sqrt{3}}{3}$     C. $\frac{4\sqrt{5}}{3}$     D. $\frac{8\sqrt{5}}{3}$
【答案：B】

9. 关于空间向量，以下说法正确的是 ( )
A. 若空间中任意一点 $O$，有 $\overrightarrow{OP}=\frac{1}{3}\overrightarrow{OA}+\frac{1}{3}\overrightarrow{OB}+\frac{1}{3}\overrightarrow{OC}$，则 $P,A,B,C$ 四点共面。
B. 若空间向量 $\vec{a}=(1,0,1), \vec{b}=(0,1,-1)$，则 $\vec{a}$ 与 $\vec{b}$ 夹角为钝角。
C. 若空间向量 $\vec{a}, \vec{b}$ 满足 $\vec{a}\cdot\vec{b}<0$，则 $\vec{a}$ 与 $\vec{b}$ 的一个方向向量为 $(3,-1)$。
D. 若 $l_1\parallel l_2$，则 $a=-1$ 或 $a=3$。
【答案：A】

10. 已知直线 $l_1: ax+3y+4=0, l_2: x+by+4=0$，若 $l_1\perp l_2$，则 $a=$ ( )
A. $1$     B. $-1$     C. $-3$     D. $3$
【答案：C】
