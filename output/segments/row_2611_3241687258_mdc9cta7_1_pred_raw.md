19. (17 分)
若 $a_i > 0$，$b_i > 0(i=1,2,\cdots,n)$，$m>0$，则不等式 $\frac{a_1^{m+1}}{b_1^m} + \frac{a_2^{m+1}}{b_2^m} + \cdots + \frac{a_n^{m+1}}{b_n^m} \geq \frac{a_1 + a_2 + \cdots + a_n}{b_1 + b_2 + \cdots + b_n}$ 成立，当且仅当 $\frac{a_1}{b_1} = \frac{a_2}{b_2} = \cdots = \frac{a_n}{b_n}$ 时，等号成立。这个不等式叫做权方和不等式，$m$ 称为该不等式的指数。
(1) 若 $a_1 > 0$，$b_1 > 0(i=1,2)$，证明二维形式的权方和不等式：$\frac{a_1^2}{b_1} + \frac{a_2^2}{b_2} \geq \frac{(a_1 + a_2)^2}{b_1 + b_2}$；
(2) 已知 $x_i > 0(i=1,2,\cdots,5)$，$x_1 + 2x_2 + 3x_3 + 4x_4 + 5x_5 = 30$，求 $x_1^2 + 2x_2^2 + 3x_3^2 + 4x_4^2 + 5x_5^2$ 的最小值；
(3) 某同学运用权方和不等式解决下列问题，指出这种解法是否正确，并说明理由：
已知正数 $x$，$y$ 满足 $x^2 + y^2 - 3x + 2 = 0$，求 $\frac{(x+2y)^2}{3x-2}$ 的最大值。
解：由权方和不等式得 $\frac{(x+2y)^2}{3x-2} = \frac{(x+2y)^2}{x^2 + y^2} \leq \frac{(x+2y)^2}{(x^2 + y^2)} = 5$，
所以 $\frac{(x+2y)^2}{3x-2}$ 的最大值是 5。
