1. 已知集合 $U=\{1,2,3,4,5\}, A=\{1,3\}, B=\{1,2,4\}$，则 $\complement_U B \cup A =$
A. $\{1,3,5\}$
B. $\{1,3\}$
C. $\{1,2,4\}$
D. $\{1,2,4,5\}$
【答案：A】

2. 命题“$\forall x \in \mathbb{R}, x - |x| \geq 0$”的否定是
A. $\exists x \in \mathbb{R}, x - |x| \geq 0$
B. $\forall x \in \mathbb{R}, x - |x| \geq 0$
C. $\exists x \in \mathbb{R}, x - |x| < 0$
D. $\forall x \in \mathbb{R}, x - |x| < 0$
【答案：C】

3. 使命题“$\forall x \in \{x \mid 1 \leq x \leq 2\}, x^2 - a \leq 0$”为真命题的一个充分不必要条件是
A. $a \leq 4$
B. $a \geq 4$
C. $a \leq 1$
D. $b > 1$
【答案：B】

4. 已知 $a = \sqrt{c+3} + \sqrt{c+4}, b = \sqrt{c+2} + \sqrt{c+5}$，则
A. $a > b > 1$
B. $b > a > 1$
C. $a > 1 > b$
D. $b > 1 > a$
【答案：A】

5. 购买同一种物品，可以用两种不同的策略，第一种是不考虑物品价格的升降，每次购买这种物品的数量一定；第二种是不考虑物品价格的升降，每次购买这种物品所花的钱一定。假设连续两天购买该物品，第一天物品的价格为 $p_1$，第二天物品的价格为 $p_2$，且 $p_1 \ne p_2$，则以下说法正确的是
A. 第一种方式购买物品的平均价格为 $\frac{p_1 p_2}{p_1 + p_2}$
B. 第二种方式购买物品的平均价格为 $\frac{p_1 + p_2}{2}$
C. 第一种方式购买物品所用平均价格更低
D. 第二种方式购买物品所用平均价格更低
【答案：C、D】

6. 下列函数中最小值为 4 的是
A. $y = 4x + \frac{1}{x} (x>0)$
B. $x > 0, y = \frac{x^2 + 2x + 5}{x + 1}$
C. $x < \frac{3}{2}, y = 2x - 1 + \frac{1}{2x - 3}$
D. $y = \sqrt{x^2 + 5} + \frac{4}{\sqrt{x^2 + 5}}$
【答案：D】

7. 已知不等式 $ax^2 - bx - 1 > 0$ 的解集是
A. $\{x \mid 2 < x < 3\}$
B. $\{x \mid x < 2 \text{ 或 } x > 3\}$
C. $\left|x\right| < \frac{1}{3} \text{ 或 } x > \frac{1}{2}$
D. $\left|x\right| < \frac{1}{3} \text{ 或 } x > \frac{1}{2}$

8. 在数学上称函数 $y = [x]$ 为高斯函数，如 $[1.5] = 1, [-2.3] = -3$，则下列结论成立的充分不必要条件是
A. $x \in \left\{x \left| \frac{1}{2} \leq x \leq \frac{5}{2} \right.\right\}$
B. $x \in \{x \mid 1 \leq x \leq 2\}$
C. $x \in \{x \mid 1 \leq x \leq 3\}$
D. $x \in \{x \mid 1 \leq x \leq 3\}$
【答案：B】

9. 下列命题中，是真命题的是
A. “$a > 1$ 且 $b > 1$”是“$ab > 1$”的充分条件
B. “$x > \frac{1}{2}$”是“$\frac{1}{x} < 2$”的充分条件
C. 命题“$\forall x < 1, x^2 < 1$”的否命题是“$\exists x \geq 1, x^2 \geq 1$”
D. “$a + b = 0$”的充要条件是“$a = b = 0$”
