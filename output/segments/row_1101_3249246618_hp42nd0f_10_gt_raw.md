一、选择题

9. 下列关系正确的是
A. $$\sqrt{2} \in \mathbb{R}$$     B. $$2 \subseteq \mathbb{Q}$$     C. $$\varnothing \subset \mathbb{Z}$$     D. $$\mathbb{N} \subseteq \mathbb{Z}$$ 【答案：ACD】

10. 已知 $p: x < 1$, $q: x < m$，要使 $q$ 为 $p$ 的充分条件，则 $m$ 的值可以是
A. $-1$     B. $0$     C. $1$     D. $2$ 【答案：ABC】

11. 小明的作业本被书包拉链划破了，一道数学题只剩下如下文字：“已知二次函数 y = ax² + bx + c (a < 0) 的图象过点 (1, 1)，△△△，求证：这个二次函数的图象关于直线 x = -1 对称。”其中“△△△”为被拉链划破处，且内容无法识别。根据现有信息，下列结论正确的是：
A. a + c ≤ 1
B. 直线 y = 1 被函数图象截得的线段的长度是 4
C. 函数的最大值为 1 - 4a
D. 不等式 ax² + bx + c > 0 的解集为 {x | -2 < x < 0}
【答案：BC】

二、填空题

12. 命题“$\exists x \in \mathbb{R}, x^2 - 1 < 0$”的否定是 ______ 【答案：$\forall x \in \mathbb{R}, x^2 - 1 \geq 0$】

13. 已知实数 $x,y$ 同时满足 $-2 \leq x + 2y \leq 1$ 和 $-3 \leq 2x - y \leq 0$，则 $8y - x$ 的最小值为 ______，最大值为 ______ 【答案：$-6$, $9$】

14. 已知集合 $A = \{x | x^2 + 2x - 3 > 0\}$，非空集合 $B = \{x | 2a - 1 \leq x < a + 2\}$。若 $x \in A$ 是 $x \in B$ 的必要不充分条件，则实数 $a$ 的取值范围是 ______ 【答案：$-1 \leq a < \leq -1$ 】
