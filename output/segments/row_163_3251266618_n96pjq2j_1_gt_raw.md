8. 沙市一中高一（8）班有54名学生，他们将从物理、化学、生物、政治、历史、地理这六科中，按选科规则，选学三门选修课，已知选物理有20人、选化学28人、选生物18人，同时选物理和化学有10人、选物理和生物有5人。问选物理或化学或生物最多有（ ）人。
A. 46    B. 48    C. 50    D. 51
【答案：A】

二、多选题：本题共3小题，每小题6分，共18分。在每小题给出的选项中，有多项符合题目要求。全部选对得6分，部分选对得部分分，有选错的得0分。

9. 设 $a < b < 0$，则下列不等式中正确的是（ ）
A. $\frac{2}{a} > \frac{2}{b}$    B. $ac < bc$    C. $|a| > -b$    D. $\sqrt{-a} > \sqrt{-b}$
【答案：AD】

10. 命题“$\exists x \in \mathbb{R},\ x^2 - x + 1 = 0$”是假命题
A. “$\exists x \in \mathbb{R},\ x + 1 \geq 0$”的否定是“$\forall x \in \mathbb{R},\ x + 1 < 0$”
B. “$x \in A$”是“$x \in B$”的必要条件
C. “$a > b$”是“$a^2 > b^2$”的充分不必要条件
D. “$x > 4$”是“$x > 2$”的充要条件
【答案：AD】

11. “$B \subseteq A$”是下列选项（ ）的充要条件
A. $A \cup B = A$    B. $\neg (x \in A)$    C. $(\complement_U A) \cap B = \varnothing$    D. 命题P：“$\exists x \in B$，使$x \notin A$”为假命题
【答案：AD】

三、填空题：本题共3小题，每小题5分，共15分。

12. 已知 $a > b > c > 0, A = ab + bc, B = ac + bc, C = a^2 + b^2$，则 $A, B, C$ 的大小关系是 ______。

13. 已知集合 $A = \{x \mid -1 \leq x \leq 1\}, B = \{x \mid a - 1 \leq x \leq 1 - 2a\}$，若 $B \subseteq A$，则实数 $a$ 的取值范围是 ______。

14. 定义集合的商集运算为 $\frac{B}{A} = \left\{ x \mid x = \frac{n}{m},\ m \in A,\ n \in B \right\}$，已知集合 $A = [2, 4], B = \{x \mid x = k/2 - 1, a \in \mathbb{N}\}$，则集合 $\frac{B}{A} \cup B$ 的真子集个数是 ______。
【答案：12】
