1. 已知复数 $z = \dfrac{3}{1 + 2i}$ ($i$ 是虚数单位)，则复数 $z =$ ( ) 【答案：A】

C. $\dfrac{3}{5} - \dfrac{6}{5}i$  
D. $\dfrac{3}{5} + \dfrac{6}{5}i$

2. 在 $\triangle ABC$ 中，内角 $A$, $B$, $C$ 所对边分别是 $a$, $b$, $c$，已知 $A = 45^\circ$, $a = 6$, $b = 3\sqrt{2}$，则 $B$ 的大小为 ( ) 【答案：A】  
A. $30^\circ$  
B. $60^\circ$  
C. $30^\circ$ 或 $150^\circ$  
D. $60^\circ$ 或 $120^\circ$

3. 如图，在 $\triangle ABC$ 中，$MC = \dfrac{1}{4} BC$，设 $\overrightarrow{AB} = \vec{a}$, $\overrightarrow{AC} = \vec{b}$，则 $\overrightarrow{AM} =$ ( ) 【答案：C】  
A. $\dfrac{1}{4}\vec{a} - \dfrac{3}{4}\vec{b}$  
B. $\dfrac{3}{4}\vec{a} - \dfrac{1}{4}\vec{b}$  
C. $\dfrac{1}{4}\vec{a} + \dfrac{3}{4}\vec{b}$  
D. $\dfrac{3}{4}\vec{a} + \dfrac{1}{4}\vec{b}$

4. 在复平面内，已知 $z = \dfrac{10i}{3 + i}$，则下列说法正确的是 ( ) 【答案：C】  
A. $z$ 的虚部为 $i$  
B. $z$ 在复平面内对应的点位于第三象限  
C. $|z| = \sqrt{10}$  
D. $z$ 的共轭复数 $\bar{z} = 3i - 1$

A. 充分不必要条件  
B. 必要不充分条件  
C. 充要条件  
D. 既不充分也不必要条件

6. 已知平面向量 $\vec{m}$ 与 $\vec{n}$ 之间的夹角为 $\dfrac{\pi}{3}$，$|\vec{m}| = 2$，$|\vec{n}| = 1$，则 $\vec{m}$ 与 $\vec{m} - \vec{n}$ 之间夹角的大小为 ( ) 【答案：B】  
A. $\dfrac{\pi}{4}$  
B. $\dfrac{\pi}{6}$

D. $\dfrac{2\pi}{3}$
