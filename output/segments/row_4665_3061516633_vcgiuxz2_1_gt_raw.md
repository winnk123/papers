一、选择题

8. 已知 $\triangle ABC$ 的内角 $A, B, C$ 的对边分别为 $a, b, c$。若 $a = \frac{1}{2}$，$c = 2\cos B$，$\cos C = c\sin A$ 则 $\triangle ABC$ 的面积为 ( )
A. $\frac{1}{20}$  
B. $\frac{3}{20}$  
C. $\frac{1}{4}$  
D. $\frac{\sqrt{2}}{4}$

9. 设 $z$ 是 $z$ 的共轭复数，则下列说法正确的是 ( ) 【答案：B】
A. $z - \overline{z}$ 是纯虚数  
B. $z + \overline{z}$ 是实数  
C. $z \cdot \overline{z}$ 是实数  
D. $\left| z - \overline{z} \right| \leq 2\left| \overline{z} \right|$  

10. 已知 $\vec{a} = (\cos \alpha, \sin \alpha)$，$\vec{b} = (\sqrt{2} \cos \beta, \sqrt{2} \sin \beta)$，若 $\vec{a} + \vec{b} = (1, \sqrt{3})$，则 ( )
A. $|\vec{a} + \vec{b}| = 2$  
B. $|\vec{a} - \vec{b}| = 2$  
C. $\cos(\alpha - \beta) = \frac{\sqrt{2}}{2}$  
D. $\vec{a} - \vec{b}$ 在 $\vec{a}$ 上的投影向量为 $\frac{\vec{a}}{2}$

11. 半径为 1 的球 $O'$ 完全在半径为 $R$ ($R > 1$) 的球 $O$ 的内部，且两球表面有唯一的公共点 $P$，球 $O$ 表面上三点 $A, B, C$ 确定的平面与球 $O'$ 相切，若 $PA = 3$，$\angle PAB = \angle PAC = \frac{\pi}{4}$，$\cos \angle CAB = \frac{4}{5}$，则 ( ) 【答案：B】
A. $O, O', P$ 三点共线  
B. $BC = \frac{8}{5}\sqrt{5}$  
C. 直线 $PA$ 与平面 $ABC$ 所成角小于 $\frac{\pi}{4}$  
D. 三棱锥 $P-ABC$ 的体积为 $\frac{18}{5}$  

二、填空题

12. 已知一组数据 $a+1, a+2, a+3, a+4, a+5$，则这组数据的方差为 ______ 【答案：$2$】

13. 已知四棱锥 $P-ABCD$ 的底面为平行四边形，过点 $A$ 的平面 $\alpha$ 与棱 $PB, PC, PD$ 分别交于 $E, F, G$。若三棱锥 $P-AGF$ 的体积是三棱锥 $P-EGF$ 体积的 $\frac{3}{2}$ 倍，则 $\frac{PE}{PB}$ 的值为 ______ 【答案：$\frac{3}{5}$】

14. 连续抛掷一颗质地均匀的正方体骰子两次（正方体六个面上的点数分别为 1,2,3,4,5,6），记录抛掷结果向上的点数。设事件 $A$：第一次点数为 1，事件 $B$：两次点数之和为 $t$($t\in N^{*}，2 \leq t \leq 12)。若事件 $A$ 与事件 $B$ 互斥，则 $t$ 的最小值为 ______；若事件 $A$ 与事件 $B$ 相互独立，则 $t$ 的值为 ______ 【答案：$2$；$7$】
