11. 定义：$\vec{m} \odot \vec{n}$ 是与向量 $\vec{m}, \vec{n}$ 在同一平面内，且与 $(\vec{m} + \vec{n})$ 线其起点逆时针旋转 $90^\circ$ 同向的向量，$|\vec{m} \odot \vec{n}| = |\vec{m}| \cdot |\vec{n}| \sin \langle \vec{m}, \vec{n} \rangle$ 为 $\vec{m}, \vec{n}$ 的夹角。规定：若 $\vec{m} = \vec{0}$ 或 $\vec{n} = \vec{0}$，则 $\vec{m} \odot \vec{n} = \vec{0}$。设 $\vec{a}, \vec{b}, \vec{c}$ 均为同一平面内的非零向量，则下列结论正确的是 ( ) 【答案：AB】
A. $(\vec{a} \odot \vec{b}) \cdot (\vec{a} + \vec{b}) = 0$  
B. $(\vec{a} \odot \vec{b}) + (\vec{a} + \vec{b}) = \vec{a} \odot \vec{b}$  
C. $|(\vec{a} + \vec{b}) \odot \vec{c}| = |\vec{a} \odot \vec{c}| + |\vec{b} \odot \vec{c}|$  
D. 若 $(\vec{a} \odot \vec{b}) \cdot \vec{c} = 0$，则 $\vec{a} // \vec{b}$
