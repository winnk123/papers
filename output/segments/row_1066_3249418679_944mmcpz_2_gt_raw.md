一、选择题

1. 函数 $f(x)=\sqrt{\frac{4-x}{2x+3}}$ 的定义域为 ( ) 【答案：C】
A. $$\left\{x \mid x \leq -\frac{3}{2} \text{ 或 } x \geq 4\right\}$$
B. $$\left\{x \mid x < -\frac{3}{2} \text{ 或 } x \geq 4\right\}$$
C. $$\left\{x \mid -\frac{3}{2} < x \leq 4\right\}$$
D. $$\left\{x \mid -\frac{3}{2} \leq x \leq 4\right\}$$

2. 已知 $m \in \mathbb{R}, n \in \mathbb{R}$，若集合 $\left\{m, \frac{n}{m}, 1\right\} = \left\{m^2, m+n, 0\right\}$，则 $m^{2025} + n^{2025} =$ ( ) 【答案：C】
A. 0
B. 1
C. -1
D. 1 或 -1

3. 设 $a_1, b_1, c_1, a_2, b_2, c_2$ 均为非零实数，方程 $a_1x^2 + b_1x + c_1 = 0$ 和 $a_2x^2 + b_2x + c_2 = 0$ 的解集分别为 $M$ 和 $N$，则 “$\frac{a_1}{a_2} = \frac{b_1}{b_2} = \frac{c_1}{c_2}$” 是 “$M=N$” 的 ( ) 【答案：C】
A. 充分不必要条件
B. 必要不充分条件
C. 充要条件
D. 既不充分也不必要条件

4. 对于任意集合 $M, N$，定义集合 $U = M \cup N$，则下列关系正确的是 ( ) 【答案：B】
A. $M \cup \complement_U N = M \cup N$
B. $\complement_U (M \cap N) = (\complement_U M) \cup (\complement_U N)$
C. $M \cap \complement_U N = M \cap N$
D. $\complement_U (M \cap N) = (\complement_U M) \cap (\complement_U N)$

5. 若不等式 $ax^2 + bx + 2 > 0$ 的解集为 $\left\{x \mid -\frac{1}{2} < x < \frac{1}{3}\right\}$，则 $a - b$ 的值是 ( ) 【答案：A】
A. -10
B. -14
C. 10
D. 14

6. 若 $a, b \in \mathbb{R}$，且 $ab > 0$，则下列不等式中，恒成立的是 ( ) 【答案：D】
A. $a^2 + b^2 > 2ab$
B. $a + b \geq 2\sqrt{ab}$
C. $\frac{1}{a} + \frac{1}{b} > \frac{2}{\sqrt{ab}}$
D. $\frac{b}{a} + \frac{a}{b} \geq 2$

7. 已知 $1 < a < 3$, $-5 < b < -2$，则下列结论错误的是 ( ) 【答案：A】
A. $-4 < a + b < 1$
B. $3 < a - b < 8$
C. $-15 < ab < -2$
D. $-\frac{3}{5} < \frac{a}{b} < -\frac{1}{2}$

8. 已知 $a, b, c > 0$，且 $b > c$，则 $\frac{2a^2 + b^2}{ab + c(b - c)}$ 的最小值为 ( ) 【答案：A】
A. 2
B. 3
C. $\sqrt{3} + 1$
D. $\sqrt{3} - 1$
