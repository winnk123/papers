[条件缺失]

1. 已知集合 $A=\{x|x^2-3x+2<0\}, B=\{x|x<a\}$, 若 $A \subseteq B$, 则实数 $a$ 的取值范围是 ( ) 【答案：C】
A. $(2, +\infty)$   B. $[2, +\infty)$   C. $(-\infty, 2]$   D. $(-\infty, 2)$

2. 已知 $x>0$, 则 $\frac{x}{x^2+1}$ 的最小值是 ( ) 【答案：A】
A. $\frac{1}{2}$   B. $1$   C. $\sqrt{2}$   D. $2$

3. 已知向量 $\vec{a}=(\sqrt{3}, 1), \vec{b}=(\cos\theta, \sin\theta)$, 若 $\vec{a} \cdot \vec{b} = \frac{\sqrt{3}}{2}$, 则 $\theta$ 的值为 ( ) 【答案：D】
A. $\frac{\pi}{6}$   B. $\frac{\pi}{3}$   C. $\frac{5\pi}{6}$   D. $\frac{7\pi}{6}$

4. 已知点 $P(x,y)$ 在曲线 $y=x^2$ 上，且满足 $x+y=1$，则 $|PQ|$ 的最小值为 ( ) 【答案：C】
A. $2$   B. $2\sqrt{3}$   C. $4$   D. $8$

5. 下列函数中，为周期函数，且在区间 $(\frac{\pi}{6}, \frac{\pi}{4})$ 上单调递减的是 ( ) 【答案：C】
A. $f(x)=\sin|x|+1$   B. $f(x)=|\sin 2x|+2$   C. $f(x)=\cos|3x|+3$   D. $f(x)=|\cos 4x|+4$
