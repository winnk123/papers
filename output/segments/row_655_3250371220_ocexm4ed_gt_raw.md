2. 命题 “$\forall x \in \mathbb{R},\ |x| + \sqrt{x} \geq 0$” 的否定是 ( ) 【答案：D】
A. $$\forall x \in \mathbb{R},\ |x| + \sqrt{x} < 0$$     B. $$\exists x \in \mathbb{R},\ |x| + \sqrt{x} \geq 0$$
C. $$\forall x \in \mathbb{R},\ |x| + \sqrt{x} \leq 0$$     D. $$\exists x \in \mathbb{R},\ |x| + \sqrt{x} < 0$$

3. 已知 $p: x_1=2$ 且 $x_2=3$, $q: x_1+x_2=5$, 则 $p$ 是 $q$ 的 ( ) 【答案：A】
A. 充分不必要条件     B. 必要不充分条件
C. 充要条件     D. 既不充分也不必要条件

4. 已知集合 $A=\{0, 1, 2, 4, 6\}, B=\{n \in \mathbb{N}^* \mid 2^n < 33\}$, 则集合 $A \cap B$ 的子集个数为 ( ) 【答案：A】
