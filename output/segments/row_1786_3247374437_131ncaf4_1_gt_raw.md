二、填空题

12. 不等式 $|x^2 - 5x| < 6$ 的解集为 ______
【答案：$(1,2) \cup (3,6)$】

13. 已知命题 $p: \exists m \in \mathbb{R}$ 且 $m + 1 \leq 0$，命题 $q: \forall x \in \mathbb{R}, x^2 + mx + 1 > 0$ 恒成立，若 $p$ 与 $q$ 不同时为真命题，则 $m$ 的取值范围是 ______
【答案：$(-\infty, -2] \cup (-1, +\infty)$】

14. 下列说法中正确的序号是 ______。
① 已知函数 $f(x)$ 是一次函数，满足 $f(f(x)) = 9x + 8$，则 $f(x)$ 的解析式可能为 $f(x) = -3x - 4$。
② $f(x) = \frac{|x|}{x}$ 与 $g(x) = \begin{cases} 1, & x > 0 \\ -1, & x \leq 0 \end{cases}$ 表示同一函数。
③ 函数 $f(x) = 2x + 4\sqrt{1 - x}$ 的值域为 $(-\infty, 4]$。
④ 定义在 $\mathbb{R}$ 上的函数 $f(x)$ 满足 $2f(x) - f(-x) = x + 1$，则 $f(x) = \frac{x}{3} + 1$。
⑤ 设 $a_1, a_2, b_1, b_2, c_1, c_2$ 都不为0，不等式 $a_1x^2 + b_1x + c_1 > 0$ 的解集为 $M$，不等式 $a_2x^2 + b_2x + c_2 > 0$ 的解集为 $N$，则 “$\frac{a_1}{a_2} = \frac{b_1}{b_2} = \frac{c_1}{c_2}$” 是 “$M = N$” 的充要条件。
