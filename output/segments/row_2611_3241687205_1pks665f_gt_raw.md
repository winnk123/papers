A. $$\{0\}=\varnothing$$     B. $$\{\{1,2\}\}=\{1,2\}$$     C. $$\{\varnothing\}=\varnothing$$     D. $$0\in\mathbb{N}$$

2. 已知命题 $p:\exists x_0>1$，使 $|x_0^2-1|<0$，其否定命题 $\neg p$ 为 ( ) 【答案：B】
A. $\neg p:\exists x_0>1$，使 $|x_0^2-1|\geq0$     B. $\neg p:\forall x>1$，使 $|x^2-1|\geq0$
C. $\neg p:\forall x\leq1$，使 $|x^2-1|\geq0$     D. $\neg p:\exists x_0\leq1$，使 $|x_0^2-1|\geq0$

3. 已知 $x,y$ 均为实数，则“$x^2+y^2\neq0$”是“$xy\neq0$”的 ( ) 【答案：C】
A. 充要条件     B. 充分不必要条件
C. 必要不充分条件     D. 既不充分也不必要条件

4. 已知 $a,b$ 均为正实数，若 $M=a^3+b^3,N=a^2b+ab^2$，则 ( ) 【答案：C】
A. $M<N$     B. $M\leq N$     C. $M>N$     D. $M\geq N$

6. 湘尖是湖南安化特产的黑茶紧压茶，属黑茶类“三尖”体系，由白沙溪茶厂生产，经拣青、杀青、揉捻等12道工序制成，具有松烟香明显、耐存放的特点 [2]。按原料嫩度分为湘尖一号至三号，则“A是湘尖一号”是“A是湘尖茶”的 ( ) 【答案：A】
A. 充分不必要条件     B. 必要不充分条件
C. 充要条件     D. 既不充分也不必要条件

7. 已知集合 $A=\{x|x\leq1\text{ 或 }x\geq2\},B=\{x|a-1<x<a+1\}$，若 $A\cup B=\mathbb{R}$，则实数 $a$ 的取值范围为 ( ) 
A. $1<a<2$     B. $1\leq a<2$     C. $1<a\leq2$     D. $1\leq a\leq2$

8. 甲乙丙丁四位同学在玩一个猜数字游戏，甲乙丙共同写出三个集合：$A=\left\{x\middle|0<\frac{2}{x}<2\right\},B=\left\{x\middle|-3\leq x\leq5\right\},C=\left\{x\middle|0<x<\frac{2}{3}\right\}$，然后他们三人各用一句话来正确的描述“$\Delta$”中的数字，让丁同该数字，以下是甲，乙，丙三位同学的描述，甲：此数为小于5的正整数；乙：$B$ 是 $A$ 成立的必要不充分条件；丙：$C$ 是 $A$ 成立的充分不必要条件。则“$\Delta$”中的数字可以是 ( ) 【答案：C】
A. 3或4     B. 2或3     C. 1或2     D. 1或3
