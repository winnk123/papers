1. 已知集合 $A = \{0,1,2,3,4,5\}$，$B = \{x \mid \ln x \leq 1\}$，则 $A \cap B = ($　　$)$
A. $\{1,2\}$
B. $\{1,2,3\}$
C. $\{0,1,2,3,4,5\}$
D. $\{0,1,2,3,4,5\}$
【答案：A】

2. 已知复数 $z = \frac{1+i}{2-i} - 2i$（$i$ 为虚数单位），则 $z$ 的虚部是 (　　)
A. $-\frac{9}{5}i$
B. $\frac{9}{5}$
C. $\frac{9}{5}i$
D. $-\frac{9}{5}$
【答案：D】

3. 函数 $f(x) = 5\sin\left(\frac{3}{4}x + \frac{\pi}{4}\right)$ 图象的一条对称轴可以为 (　　)
A. $x = \frac{\pi}{8}$
B. $x = \frac{\pi}{4}$
C. $x = \frac{5\pi}{4}$
D. $x = \frac{3\pi}{4}$
【答案：C】

4. 已知 $x$ 为实数，则 $x < 0$ 是 $\frac{x}{x-4} > 0$ 的 (　　)
A. 充分不必要条件
B. 必要不充分条件
C. 充要条件
D. 既不充分也不必要条件
【答案：B】

5. 已知向量 $\vec{m}, \vec{n}$ 满足 $\vec{m} - \vec{n} = (-3,4)$，$\vec{m} + 2\vec{n} = (3,4)$，则下列结论正确的是 (　　)
A. $\vec{m} \perp \vec{n}$
B. $\vec{m} \parallel \vec{n}$
C. $|\vec{m}| = 2|\vec{n}|$
D. $\vec{m} = (2,1)$
【答案：A】

6. 某乡镇计划对一荒山区域进行植树绿化，已知该区域到 2025 年年底植树绿化面积为 10 万亩，以此值为初始值 $b_0 (b_0=10)$，该区域经过 $a$ 年，到年底植树绿化面积为 $b$ 万亩，若 2025 年以后每年的增长率均为 $p$，满足关系式 $a = \frac{1}{p} \log_{b_0} \frac{b(30-b)}{(30-b_0)}$，则到 2030 年年底植树绿化面积为 (　　)
A. 20 万亩
B. 18 万亩
C. 15 万亩
D. 13 万亩
【答案：C】
