[条件缺失]

15. 设命题 $p$：对任意 $-1 \leqslant x \leqslant 1$，不等式 $x^2 - 2x - 5 + m < 0$ 恒成立；命题 $q$：存在 $-1 \leqslant x \leqslant 1$，使得不等式 $2x - 2 \geqslant m^2 - 3m$ 成立。若 $p, q$ 中至少有一个是假命题，则实数 $m$ 的取值范围为
A. $m < 2$
B. $0 \leqslant m \leqslant 3$
C. $0 \leqslant m < 2$
D. $m < 0$ 或 $m \geqslant 2$
【答案：C】

16. 已知正实数 $a, b$ 满足 $a + b = 2$，则 $\dfrac{a^2 + 1}{a} + \dfrac{b^2 + 1}{b} + 4$ 的最小值为
A. $\dfrac{11}{2}$
B. $\dfrac{23}{4}$
C. $\dfrac{25}{4}$
D. $\dfrac{13}{2}$
【答案：C】

17. 已知集合 $A = \{x \mid -1 < x < 1\}$，$B = \{x \mid |x - 1| < 2\}$，下列说法正确的是
A. $\forall x \in A, x \in B$
B. $\exists x \in B, x \in A$
C. “$x \in A$” 是 “$x \in B$” 的充分不必要条件
D. “$x \in B$” 是 “$x \notin A$” 的必要不充分条件
【答案：B】

18. 关于 $x$ 的不等式 $ax^2 - (a+1)x + 1 > 0$ 的解集可能是
A. $\mathbb{R}$
B. $\left\{x \mid \dfrac{1}{a} < x < 1\right\}$
C. $\left\{x \mid 1 < x < \dfrac{1}{a}\right\}$
D. $\left\{x \mid x < 1\right.$ 或 $\left.x > \dfrac{1}{a}\right\}$
【答案：B D】

19. 已知集合 $A = \{x \mid \dots\}$，下列结论正确的是
A. 若 $A \cap B = B$，
B. 若 $A \cup B = A$，
C. 若 $m < 4$，则 $A \cup B = \dots$
D. 若 $m > 4$，则 $A \cup B - \dots$
【答案：BC】

20. 已知正数 $x, y$ 满足 $2x + y = xy - 6$，下列结论正确的是
A. $x > 2$
B. $y > 2$
C. $2x + y$ 的最小值为 12
D. $x + 2y$ 的最小值为 14
【答案：BC】
