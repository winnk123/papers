一、选择题

1. 复数 $z$ 满足 $\frac{z + 3i}{z} = -i$（$i$ 是虚数单位），则 $z =$ ( ) 【答案：C】
A. $-\frac{3}{2} - \frac{3}{2}i$     B. $-\frac{3}{2} + \frac{3}{2}i$     C. $\frac{3}{2} - \frac{1}{2}i$     D. $\frac{3}{2} + \frac{1}{2}i$

2. 已知集合 $A = \{-6, 6\}, B = \{x \mid ax - 3 = 0\}, A \cup B = A$，则实数 $a$ 的取值构成的集合为 ( ) 【答案：D】
A. $\left\{-\frac{1}{2}\right\}$     B. $\left\{\frac{1}{2}\right\}$     C. $\left\{-\frac{1}{2}, \frac{1}{2}\right\}$     D. $\left\{-\frac{1}{2}, 0, \frac{1}{2}\right\}$

3. 若函数 $f(x) = a + \frac{4}{3^x - 1}$ 为奇函数，则实数 $a =$ ( ) 【答案：B】
A. $-1$     B. $1$     C. $2$     D. $4$

4. 若 $\exists x \in \{x \mid -3 \leq x \leq 3\}$，使得 $x - 4a - 13 < 0$ 成立，则实数 $a$ 的取值范围是 ( ) 【答案：B】
A. $(-\infty, 3)$     B. $(-4, +\infty)$     C. $(-3, +\infty)$     D. $(-\infty, -4)$

5. 随着生态环境的改善，每年来某地湖泊繁育幼鸟的各种鸟类越来越多，鸟类众多、比较集中，且各种鸟类的数量在 1500 及以上的时间称为鸟类繁育“旺季”。第 $k$ 个月，当地湖泊中各种鸟类的数量 $f(k)$ 可近似用函数 $f(k) = 2000 \cos\left(\frac{k\pi}{6} + \frac{2\pi}{3}\right) + 2500$ 来表示，那么一年中是“旺季”的月份有 ( ) 【答案：B】
A. 3个     B. 4个     C. 5个     D. 6个

6. 已知 $x \geq y \geq 0$，则 $3x + \frac{1}{2x - y} + \frac{1}{x + y}$ 的最小值为 ( ) 【答案：B】
A. $4\sqrt{3}$     B. $6$     C. $9$     D. $12$

7. 已知函数 $f(x) = \sin\left(\omega x + \frac{\pi}{4}\right)$ ($\omega > 0$)，若 $f(x) \leq \left|f\left(\frac{\pi}{4}\right)\right|$ 且函数 $f(x)$ 的最小正周期 $T$ 满足 $T \in \left(\frac{\pi}{5}, \frac{\pi}{3}\right)$，则 $T =$ ( ) 【答案：B】
A. $\frac{2\pi}{7}$     B. $\frac{2\pi}{9}$     C. $\frac{12\pi}{5}$     D. $\frac{27\pi}{10}$
