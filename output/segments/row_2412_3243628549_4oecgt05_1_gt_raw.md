7. 已知不等式 $ax^2 - bx - 1 > 0$ 的解集是 $\left\{ x \mid -\frac{1}{2} < x < -\frac{1}{3} \right\}$，则不等式 $x^2 - bx - a < 0$ 的解集是 ( ) 【答案：A】
A. $$\left\{ x \mid 2 < x < 3 \right\}$$
B. $$\left\{ x \mid x < 2 \text{ 或 } x > 3 \right\}$$
C. $$\left\{ x \mid \frac{1}{3} < x < \frac{1}{2} \right\}$$
D. $$\left\{ x \mid x < \frac{1}{3} \text{ 或 } x > \frac{1}{2} \right\}$$

8. 在数学上称函数 $y = [x]$ 为高斯函数，其中 $[x]$ 表示不大于 $x$ 的最大整数，如 $[1.5] = 1$, $[-2.3] = -3$, $[3] = 3$。那么使不等式 $4[x]^2 - 12[x] + 5 \leqslant 0$ 成立的充分不必要条件是 ( ) 【答案：B】
A. $$x \in \left\{ x \mid \frac{1}{2} \leqslant x \leqslant \frac{5}{2} \right\}$$
B. $$x \in \left\{ x \mid 1 \leqslant x \leqslant 2 \right\}$$
C. $$x \in \left\{ x \mid 1 \leqslant x < 3 \right\}$$
D. $$x \in \left\{ x \mid 1 \leqslant x \leqslant 3 \right\}$$

D. “$a+b=0$”的充要条件是“$\frac{a}{b}=-1$”
