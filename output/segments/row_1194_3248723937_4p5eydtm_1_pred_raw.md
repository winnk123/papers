1. 设等比数列 $\{a_n\}$ 的前 $n$ 项和为 $S_n$，且满足 $a_6 = 8a_3$，则（　　）
A. $S_6 = 3S_3$
B. 数列 $\{a_n\}$ 的公比为 2
C. $\frac{S_6}{S_3} = 8$
D. $|a_n|$ 的最小值为 1

【答案：B】

A. $$S_6 = 3S_3$$     B. $$\text{数列 } \{a_n\} \text{ 的公比为 } 2$$     C. $$\frac{S_6}{S_3} = 8$$   D. $$|a_n| \text{ 的最小值为 } 1$$

2. 已知等差数列 $\{a_n\}$ 的前 $n$ 项和为 $S_n$，$a_1 = 11$，$a_5 = 3$，则（　　）
A. $S_3 = 35$
B. $a_n = 13 - 2n$
C. $S_n = \frac{n}{2}(a_1 + a_n)$
D. $a_{10} = 1$

【答案：B】

A. $$S_3 = 35$$     B. $$a_n = 13 - 2n$$     C. $$S_n = \frac{n}{2}(a_1 + a_n)$$   D. $$a_{10} = 1$$

14. 已知数列 $\{a_n\}$ 满足 $a_1 = 3$, $a_{n+1} - a_n = 2n + 1$，则 $a_n =$ ______。
