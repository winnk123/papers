2. 已知命题 $p: \exists x_0 > 1$, 使 $|x_0^2 - 1| < 0$, 其否定命题 $\neg p$ 为 ( ) 【答案：B】
A. $\neg p: \forall x > 1$, 使 $|x^2 - 1| \geq 0$
B. $\neg p: \forall x > 1$, 使 $|x^2 - 1| \geq 0$
C. $\neg p: \exists x \leq 1$, 使 $|x^2 - 1| \geq 0$
D. $\neg p: \exists x \leq 1$, 使 $|x^2 - 1| \geq 0$

3. 已知 $x,y$ 均为实数，若“$x^2 + y^2 \neq 0$”是“$xy \neq 0$”的 ( ) 【答案：C】
A. 充要条件     B. 充分不必要条件     C. 必要不充分条件     D. 既不充分也不必要条件

4. 已知 $a,b$ 均为正实数，若 $M = a^3 + b^3$, $N = a^2b + ab^2$, 则 ( ) 【答案：C】
A. $M < N$     B. $M \leq N$     C. $M > N$     D. $M \geq N$

5. 设全集 $U = \mathbb{R}$，集合 $A = \{x | x^2 - 2x - 3 < 0\}$，$B = \{x | x - 1 \geq 0\}$，则图中阴影部分所表示的集合为 ( ) 【答案：A】
A. $\{x | x \leq -1\}$     B. $\{x | x < -1$ 或 $x \geq 3\}$     C. $\{x | x \leq 1\}$     D. $\{x | x \leq -1$ 或 $x \geq 3\}$

6. 湘尖是湖南安化特产的黑茶紧压茶，属黑茶类“三尖”体系，由白沙溪茶厂生产，经拣青、杀青、揉捻等 12 道工序制成，具有松烟香明显、耐存放的特点 [²]。按原料嫩度分为湘尖一号至三号，则“A 是湘尖茶”的 ( ) 【答案：A】
A. 充分不必要条件     B. 必要不充分条件     C. 充要条件     D. 既不充分也不必要条件

7. 已知集合 $A = \{x | x \leq 1$ 或 $x \geq 2\}$，$B = \{x | a - 1 < x < a + 1\}$，若 $A \cup B = \mathbb{R}$，则实数 $a$ 的取值范围为 ( ) 【答案：C】
A. $1 < a < 2$     B. $1 \leq a < 2$     C. $1 < a \leq 2$     D. $1 \leq a \leq 2$

8. 甲乙丙丁四位同学在玩一个猜数字游戏，甲乙丙共同写出三个集合：$A = \{x | 0 < x < 2\}$，$B = \{x | -3 \leq x \leq 5\}$，$C = \{x | 0 < x < \frac{2}{3}\}$，然后他们三人各用一句话来正确的描述“$\Delta$”中的数字，让丁猜出这个数字。以下是甲、乙、丙三位同学的描述：甲：此数为小于 5 的正整数；乙：$B$ 是 $A$ 成立的必要条件；丙：$C$ 是 $A$ 成立的充分不必要条件。则“$\Delta$”中的数字可以是 ( ) 【答案：C】
A. 3 或 4     B. 2 或 3     C. 1 或 2     D. 1 或 3

9. 下列说法正确的是 ( ) 【答案：AB】
A. $\forall x \in \mathbb{R}, x + |x| \geq 0$
B. 若 $A \cup B$ 是空集，则 $A$ 与 $B$ 均是空集
C. $p: x=1$ 是一元二次方程 $ax^2 + bx + c = 0$ 的一个根，$q: a+b+c=0(a \neq 0)$，则 $p$ 是 $q$ 成立的充要条件
D. $\exists m \in \mathbb{Z}$，使得 $m^2 + n$ 为奇数

10. 已知全集 $U = A \cup B$，集合 $A = \{1,3,4\}$，$B = \{x \in \mathbb{N} | x \leq 6\}$，则 ( ) 【答案：ABD】
A. $2 \in B$     B. $A \cap B = \{1,3\}$     C. $B$ 的子集有 8 个     D. $U$ 中的元素个数为 5

11. 若“$\forall x \in M$, $2-x<0$”为真命题，“$\exists x \in M$, $x<0$ 或 $x>4$”为假命题，则集合 $M$ 可以是 ( ) 【答案：ABCD】
A. $\{x | -1 < x < 2\}$     B. $\{x | 3 < x < 4\}$     C. $\{x | 0 < x < 2\}$     D. $\{x | 2 < x < 3\}$
