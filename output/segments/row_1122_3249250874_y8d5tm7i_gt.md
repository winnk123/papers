19. (17分) 数学中的“对称性”与“等效性”不仅体现在几何和函数中，还体现在概率问题中，例如：某班级共有 N 名学生（ N>1 ），其学号记为 1,2,3,⋯,N ，老师使用 AI 点名软件进行随机点名。规则为：每次点名时，软件会从 N 名学生中随机抽取一名学生对应的学号（每次点名后，学号复位，即认为有放回抽取），且每次抽取相互独立。老师随机进行两次点名，记 Xᵢ 为第 i 次抽取的学号（ i=1,2 ）， X = |X₁ - X₂| 。
(1) 计算 P(X=0) 与 P(X₁ < X₂) ；（用 N 表示）
(2) 已知随机变量的期望具有线性可加性，即对于随机变量 xi, eta ，有 E(xi ± eta) = E(xi) ± E(eta) 。记 Y = max{X₁, X₂} ， Z = min{X₁, X₂} ，其中 max{a,b} 表示 a, b 中的最大者， min{a,b} 表示 a, b 中的最小者。
(i) 计算 E(Y+Z) 的值；（用 N 表示）
(ii) 证明 max{E(X), E(Y), E(Z)} = E(Y) 。
（参考公式： 1² + 2² + ⋯ + n² = (n(n+1)(2n+1))/(6) ）
