4. 已知 $x \in R$，则使得 $x^2 - 2x < 0$ 成立的一个充分不必要条件是 ( ) 【答案：D】
A. $-1 < x < 0$
B. $0 < x < 3$
C. $0 < x < 1$
D. $0 < x < 2$

5. 设 $m + n > 0$，则关于 $x$ 的不等式 $(m - x)(n + x) > 0$ 的解集是 ( ) 【答案：A】
A. $\{x | x < -n, \text{ 或 } x > m\}$
B. $\{x | -n < x < m\}$
C. $\{x | x < -m, \text{ 或 } x > n\}$
D. $\{x | -m < x < n\}$

6. 已知 $a > 0, b > 0$，若 $\frac{1}{4a + b} + \frac{1}{a + 2b} = 3$，则 $5a + 3b$ 的最小值为 ( ) 【答案：D】
A. 2
B. 4
C. $\frac{2}{3}$
D. $\frac{3}{4}$

7. 设 $a, b \in R$，且 $a < b < 0$，则 ( ) 【答案：D】
A. $\frac{1}{a} < \frac{1}{b}$
B. $\frac{b}{a} > \frac{a}{b}$
C. $\frac{a + b}{2} > \sqrt{ab}$
D. $\frac{b}{a} + \frac{a}{b} > 2$

8. 已知关于 $x$ 的不等式 $x^2 - 2ax - 8a^2 < 0$ ($a > 0$)，其解集为 $\{x | x_1 < x < x_2\}$，且 $x_2 - x_1 \leq 12$，则 $a$ 的取值范围为 ( ) 【答案：D】
A. $a > -5$
B. $a < 2$
C. $-5 < a \leq 10$
D. $0 < a \leq 2$

9. 已知集合 $A = \{x | x^2 - 1 = 0\}$，$Q$ 为有理数集，则下列式子表示正确的是 ( ) 【答案：A】
A. $\sqrt{2} \notin A$
B. $\{-1\} \notin A$
C. $A \subseteq Q$
D. $\{1, -1\} \subseteq A$

10. 已知关于 $x$ 的不等式 $ax^2 + bx + c > 0$ 的解集为 $\{x | x < -2 \text{ 或 } x > 3\}$，则下列选项中正确的是 ( ) 【答案：B】
A. $a < 0$
B. 不等式 $bx + c > 0$ 的解集是 $\{x | x < -6\}$
