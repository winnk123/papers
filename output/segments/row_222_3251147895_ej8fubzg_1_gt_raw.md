一、选择题
1. 复数 $z$ 满足 $z(2+i)=i$，则 $|z|=$ ( ) 【答案：D】
A. $$\sqrt{3}$$     B. $$\sqrt{5}$$     C. $$\frac{\sqrt{5}}{3}$$     D. $$\frac{\sqrt{5}}{5}$$

2. 已知向量 $\vec{a}=(-1,3)$，$\vec{b}=(2,-1)$，若 $(2\vec{a}-\vec{b}) \parallel (\vec{a}+k\vec{b})$，则 $k=$ ( ) 【答案：B】
A. $$\frac{1}{2}$$     B. $$-\frac{1}{2}$$     C. $$-\frac{5}{3}$$     D. $$\frac{5}{3}$$

3. 已知集合 $A=\{x \mid -1 < x \leq a\}$，$B=\{x \mid 0 < x < 2\}$，若 $A \cap B = B$，则实数 $a$ 的取值范围是 ( ) 【答案：A】
A. $$a \geq 2$$     B. $$a < 2$$     C. $$a > 2$$     D. $$a \leq 2$$

4. 下列命题正确的是 ( ) 【答案：C】
A. $$\forall x \in \mathbb{N},\ x^3 > x^2$$
B. $$\exists x_0 \in \mathbb{R},\ x_0^2 + 2x_0 + 3 = 0$$
C. “$x > 1$”是“$x^2 > 1$”的充分且不必要条件
D. 若 $a > b$，则 $a^2 > b^2$

5. 已知命题 $p$: $\forall x > 2$, $x^2 - 1 > 0$，则 $\neg p$ 是 ( ) 【答案：C】
A. $$\exists x \leq 2,\ x^2 - 1 > 0$$
B. $$\exists x > 2,\ x^2 - 1 < 0$$
C. $$\exists x > 2,\ x^2 - 1 < 0$$
D. $$\forall x \leq 2,\ x^2 - 1 \leq 0$$

6. 不等式 $\frac{3x-1}{x} \leq 1$ 的解集为 ( ) 【答案：B】
A. $$\left\{x \mid 0 \geq x \leq \frac{1}{2}\right\}$$
B. $$\left\{x \mid 0 < x \leq \frac{1}{2}\right\}$$
C. $$\left\{x \mid x \leq \frac{1}{2}\right\}$$
D. $$\left\{x \mid x > 0\right\}$$

7. 已知 $a,\ b > 0$，且 $ab = a + b + 3$，则 $a + 4b$ 的最小值是 ( ) 【答案：C】
A. $$6$$
B. $$9$$
C. $$13$$
D. $$7 + 4\sqrt{3}$$
