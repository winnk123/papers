一、选择题

1. 已知 $A=\left\{x \left| \frac{3-x}{x+1} \geq 0 \right.\right\}$, $B=\{-2, -1, 0, 1, 2\}$, 则 $A \cap B=$ ( )
A. $\{-2, -1, 0, 1, 2\}$     B. $\{-1, 0, 1, 2\}$     C. $\{0, 1, 2\}$     D. $\{0, 1, 2, 3\}$
【答案：C】

2. 在 $\triangle ABC$ 中, 若满足 $\sin A : \sin B : \sin C = 2 : 3 : \sqrt{7}$, 则 $\triangle ABC$ 为 ( )
A. 锐角三角形     B. 直角三角形     C. 钝角三角形     D. 等腰三角形
【答案：C】

3. 下列函数是偶函数且值域为 $[1, +\infty)$ 的是 ( )
A. $y=\cos x$    B. $y=e^|x|+1$   C. $y=|\tan x| + 1$          D. $y=x^2-2x+2$
【答案：D】

4. 在 $\triangle ABC$ 中, $AB=AC=2$, $BC=2\sqrt{3}$, 设 $\overrightarrow{AB}=\vec{a}$, $\overrightarrow{AC}=\vec{b}$, 则向量 $\vec{b}$ 在向量 $\vec{a}$ 方向上的投影向量是 ( )
A. $\frac{1}{2}$     B. $\frac{1}{2}\vec{a}$     C. $-\frac{1}{2}$     D. $-\frac{1}{2}\vec{a}$
【答案：C】

C. 若 $|\vec{a}| + |\vec{b}| = |\vec{a} + \vec{b}|$，则 $\vec{b} = \lambda \vec{a}$
