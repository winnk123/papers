19. (17 分)
若 aᵢ > 0 ， bᵢ > 0(i=1,2,⋯,n) ， m>0 ，则不等式 (a₁ᵐ⁺¹)/(b₁ᵐ) + (a₂ᵐ⁺¹)/(b₂ᵐ) + ⋯ + (aₙᵐ⁺¹)/(bₙᵐ) ≥ (a₁ + a₂ + ⋯ + aₙ)/(b₁ + b₂ + ⋯ + bₙ) 成立，当且仅当 (a₁)/(b₁) = (a₂)/(b₂) = ⋯ = (aₙ)/(bₙ) 时，等号成立。这个不等式叫做权方和不等式， m 称为该不等式的指数。
(1) 若 a₁ > 0 ， b₁ > 0(i=1,2) ，证明二维形式的权方和不等式： (a₁²)/(b₁) + (a₂²)/(b₂) ≥ ((a₁ + a₂)²)/(b₁ + b₂) ；
(2) 已知 xᵢ > 0(i=1,2,⋯,5) ， x₁ + 2x₂ + 3x₃ + 4x₄ + 5x₅ = 30 ，求 x₁² + 2x₂² + 3x₃² + 4x₄² + 5x₅² 的最小值；
(3) 某同学运用权方和不等式解决下列问题，指出这种解法是否正确，并说明理由：
已知正数 x ， y 满足 x² + y² - 3x + 2 = 0 ，求 ((x+2y)²)/(3x-2) 的最大值。
解：由权方和不等式得 ((x+2y)²)/(3x-2) = ((x+2y)²)/(x² + y²) ≤ ((x+2y)²)/((x² + y²)) = 5 ，
所以 ((x+2y)²)/(3x-2) 的最大值是 5。
