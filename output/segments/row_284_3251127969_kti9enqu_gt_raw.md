一、选择题
1. 直线 $x + \sqrt{3}y - 2 = 0$ 的斜率为 ( )
A. $\sqrt{3}$     B. $\sqrt{3}$     C. $-\frac{\sqrt{3}}{3}$     D. $-\sqrt{3}$
【答案：C】

2. 点 $P(1,2,-3)$ 关于 $Oxz$ 平面对称的点的坐标是 ( )
A. $(1,2,3)$     B. $(1,-2,-3)$     C. $(-1,2,-3)$     D. $(-1,-2,3)$
【答案：B】

3. 在平行六面体 $ABCD-A_1B_1C_1D_1$ 中，$\overrightarrow{AB} + \overrightarrow{AD} + \overrightarrow{AA_1} =$ ( )
A. $\overrightarrow{CA_1}$     B. $\overrightarrow{AC_1}$     C. $\overrightarrow{DB_1}$     D. $\overrightarrow{BD_1}$
【答案：B】

4. 设 $x,y \in \mathbb{R}$，向量 $\vec{a} = (0,1)$，$\vec{b} = (2,y,2)$，$\vec{c} = (-3,6,-3)$，且 $\vec{a} \perp \vec{b}, \vec{b} // \vec{c}$，则 $|\vec{a} - \vec{b}| =$ ( )
A. $2\sqrt{2}$     B. $3$     C. $\sqrt{29}$     D. $2\sqrt{6}$
【答案：C】

5. 已知 $\vec{n_1} = (-1,9,1)$，$\vec{n_2} = (m,-3,2)$，若 $\langle \vec{n_1}, \vec{n_2} \rangle$ 不能构成空间的一个基底，则 $m =$ ( )
A. $1$     B. $3$     C. $5$     D. $7$
【答案：A】

6. 已知 $a > 0$, $b > 0$，直线 $(a-1)x + 2y + 3 = 0$ 与直线 $x + by - 1 = 0$ 垂直，则 $\frac{1}{a} + \frac{1}{b}$ 的最小值是 ( )
A. $1$     B. $6$     C. $2 + \sqrt{2}$     D. $3 + 2\sqrt{2}$
【答案：D】

7. 若直线 $l$ 过定点 $P(1,0)$，且与以 $A(-1,2), B(2,\sqrt{3})$ 为端点的线段相交（包括端点），则其倾斜角的取值范围是 ( )
A. $\left[\frac{\pi}{3}, \frac{3\pi}{4}\right]$     B. $\left[\frac{\pi}{6}, \frac{3\pi}{4}\right]$     C. $\left(0, \frac{\pi}{6}\right] \cup \left[\frac{3\pi}{4}, \pi\right)$     D. $\left(0, \frac{\pi}{3}\right] \cup \left[\frac{3\pi}{4}, \pi\right]$
【答案：A】

8. 在等腰直角三角形 $ABC$ 中，$AB = AC = 4$，点 $P$ 是边 $AB$ 上异于 $A,B$ 的一点，光线从点 $P$ 出发经 $BC, CA$ 反射后又回到点 $P$，若光线 $QR$ 经过 $\triangle ABC$ 的重心，则 $\triangle PQR$ 的周长等于 ( )
A. $\frac{2\sqrt{37}}{3}$     B. $4\sqrt{15}$     C. $\frac{5\sqrt{3}}{3}$     D. $\frac{8\sqrt{2}}{3}$
【答案：C】

9. 关于空间向量，以下说法正确的是
A. 若空间中任意一点 $O$，有 $\overrightarrow{OP} = \frac{1}{3}\overrightarrow{OA} + \frac{1}{6}\overrightarrow{OB} + \frac{1}{2}\overrightarrow{OC}$，则 $P,A,B,C$ 四点共面
B. 若空间向量 $\vec{a} = (1,0,1)$，$\vec{b} = (0,1,-1)$，则 $\vec{a}$ 在 $\vec{b}$ 上的投影向量为 $\left(0, -\frac{1}{2}, \frac{1}{2}\right)$
C. 若空间向量 $\vec{a}, \vec{b}$ 满足 $\vec{a} \cdot \vec{b} < 0$，则 $\vec{a}$ 与 $\vec{b}$ 夹角为钝角
D. 若 $l_1 \perp l_2$，则 $a = \frac{3}{2}$
【答案：BC】

10. 已知直线 $l_1: ax + 3y + 4 = 0, l_2: x + (a-2)y + a^2 - 5 = 0$，在每小题给出的选项中，有多项符合题目要求。全部选对得 6 分，部分选对得 3 分，有选错的得 0 分。
A. 若 $a = 1$，则 $l_1$ 的一个方向向量为 $(3, -1)$
B. 若 $l_1 \perp l_2$，则 $a = \frac{3}{2}$
C. 若 $l_1 // l_2$，则 $a = -1$ 或 $a = 3$
D. 若 $l_1$ 不经过第二象限，则 $a \leq 0$
【答案：BD】
