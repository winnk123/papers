1. 已知命题 $p: \exists x>0, x^3 \leq x$，则命题 $p$ 的否定为 ( ) 【答案：D】
A. $\exists x>0, x^3 \leq x$     B. $\exists x\leq0, x^3 \geq x$     C. $\forall x>0, x^3 \leq x$     D. $\forall x>0, x^3 \geq x$

2. 若集合 $A=\{x|x\leq3\}, B=\{x|x\leq1\}$，则 $A \cap (\complement_{\mathbb{R}}B) = $ ( ) 【答案：A】
A. $\{x|1<x\leq3\}$     B. $\{x|1\leq x\leq3\}$     C. $\{x|x>1\}$     D. $\{x|x\leq1\}$

3. 设 $x\in\mathbb{R}$，则“$0<x<5$”是“$|x-1|<1$”的 ( ) 【答案：D】
A. 充分而不必要条件     B. 必要而不充分条件     C. 充要条件     D. 既不充分也不必要条件

4. 不等式 $\frac{3x-1}{x} \leq 1$ 的解集为 ( ) 【答案：B】
A. $\left\{x|0\leq x\leq\frac{1}{2}\right\}$     B. $\left\{x|0<x\leq\frac{1}{2}\right\}$     C. $\left\{x|x\leq\frac{1}{2}\right\}$     D. $\{x|x>0\}$

5. 已知 $a>b>0$，下列不等式中正确的是 ( ) 【答案：D】
A. $\frac{c}{a}>\frac{c}{b}$     B. $\frac{1}{a-1}<\frac{1}{b-1}$     C. $ab<b^2$     D. $-a^2<-ab$

6. 已知 $x>-1$，当 $x=a$ 时，$x-4+\frac{9}{x+1}$ 取得最小值为 $b$，则 $a+b=$ ( ) 【答案：C】
A. $-3$     B. $2$     C. $3$     D. $8$

7. 集合 $A=\{-4,2a-1,a^2\}, B=\{9,a-5,1-a\}$，若 $A\cap B=\{9\}$，则 $a=$ ( ) 【答案：A】
A. $-3$     B. $3$ 或 $-3$     C. $3$     D. $3$ 或 $-3$ 或 $5$

8. 已知 $x>0, y>0$，且 $x+y+xy=3$，若不等式 $x+y\geq m^2-m$ 恒成立，则实数 $m$ 的取值范围为 ( ) 【答案：B】
A. $-2\leq m\leq1$     B. $-1\leq m\leq2$     C. $m\leq-2$ 或 $m\geq1$     D. $m\leq-1$ 或 $m\geq2$

9. 如果集合 $A=\{x|ax^2-2x-1=0\}$ 只有一个元素，则 $a$ 的值可以是 ( ) 【答案：C】
A. $0$     B. $1$     C. $-1$     D. $2$
