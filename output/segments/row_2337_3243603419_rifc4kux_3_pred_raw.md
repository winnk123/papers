[条件缺失]

10. 已知集合 $A = \{x | ax = \frac{1}{2}, x \in R\}$, $B = \{x | x^2 - x - 56 = 0\}$, 若 $A \subseteq B$，则实数 $a$ 的值可以是 ( ) 【答案：A】
A. $\frac{1}{8}$
B. $\frac{1}{7}$
C. $\frac{1}{6}$
D. $\frac{1}{5}$

11. 已知正数 $a$, $b$ 满足 $a + 2b = 2ab - 3$，则 ( ) 【答案：B】
A. $a > b$
B. $a < b$
C. $a = b$
D. $a + b = 3$

12. 集合 $A = \{x | x \leq 5 \text{ 且 } x \neq 1\}$ 用区间表示 ( ) 【答案：D】
A. $(\frac{1}{2}, +\infty)$
B. $(-\infty, \frac{1}{2}] \cup [\frac{1}{2}, 5]$
C. $(-\infty, 5]$
D. $(-\infty, 1) \cup (1, 5]$

13. 已知 $0 < x < \frac{4}{3}$，则当 $x =$ ( ) 时，$x(4 - 3x)$ 取最大值为 ( ) 【答案：$\frac{2}{3}$】
A. $\frac{1}{3}$
B. $\frac{2}{3}$
C. $\frac{3}{4}$
D. $\frac{4}{5}$

14. 若命题“$\exists x_0 \in R$, $(m - 1)x_0^2 + (m - 1)x_0 + 1 \leq 0$”是假命题，则实数 $m$ 的取值范围是 ( ) 【答案：$(1, +\infty)$】
A. $(1, +\infty)$
B. $(0, 1)$
C. $(1, 2)$
D. $(2, +\infty)$

15. (13分) 已知集合 $A = \{x | x^2 - 4x + 3 < 0\}$, $B = \{x | 2m < x < 1 - m\}$.
(1) 当 $m = -1$ 时，求：① $A \cup B$；② $A \cap (\complement_R B)$；③ $(\complement_R A) \cap B$。
(2) 若 $A \subseteq B$，求实数 $m$ 的取值范围。

16. (15分) 已知 $a > 0$, $b \geq 0$.
(1) 求证：$a^3 + b^3 \geq ab^2 + a^2b$；
(2) 若 $ab = 2ab$，求 $ab$ 的最小值。
