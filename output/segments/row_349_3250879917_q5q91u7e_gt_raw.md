一、选择题

1. 若椭圆 $\frac{x^2}{25} + \frac{y^2} = 1$ 上一点 $P$ 到两焦点一个焦点的距离为 3，则 $P$ 到另一个焦点的距离为 ( )
A. 2     B. 5     C. 7    D. 8
【答案：C】

2. 若方程 $\frac{x^2}{6-m} = 1-\frac{y^2}{m-4} $ 表示焦点在 $y$ 轴上的椭圆，则实数 $m$ 的取值范围是 ( )
A. [4,5]     B. (4,6)     C. $[5,+\infy)$     D. (5,6)
【答案：D】

3. 已知椭圆方程 $\frac{x^2}{4} + \frac{y^2}{3} = 1$，过左焦点 $F_1$ 的直线与椭圆交于 $A,B$ 两点，连接 $AF_2, BF_2$，则三角形 $ABF_2$ 的周长为 ( )
A. 8     B. 10    C. 12     D. 14
【答案：D】

4. 方程 $x^2 + y^2 - 2x + my + 5 = 0$ 表示圆，则 $m$ 的取值范围是 ( )
A. $(-\infty, -4) \cup (4, +\infty)$
B. $(-\infty, -2) \cup (2, +\infty)$
C. $(-\infty, -2\sqrt{3}) \cup (2\sqrt{3}, +\infty)$
D. $(-\infty, -2\sqrt{5}) \cup (2\sqrt{5}, +\infty)$
【答案：A】

5. 已知直线 $l: 3x - 4y + 7 = 0$ 与直线 $l_1: 6x - (m+1)y + 1 - m = 0$ 平行，则 $m$ 的值为 ( )
A. 3     B. $-\frac{11}{2}$     C. 7     D. 8
【答案：C】

6. 从点 $P(m, 2-m)$ 向圆 $(x+2)^2 + (y+2)^2 = 1$ 引切线，则切线长的最小值为 ( )
A. 5     B. $\sqrt{19}$     C. $\sqrt{17}$     D. $\3\sqrt{2}$
【答案：C】

7. 若圆 $C: (x+a)^2 + (y+b)^2 = r^2$ 与直线 $l: x - y + 2 = 0$ 相切，且存在点 $P$，过点 $P$ 可以作两条相切直线与圆 $C$ 相切，则 $r$ 的取值范围是 ( )
A. $(0, \sqrt{2}]$     B. $(1, \sqrt{2}]$     C. $[\sqrt{2}, 2)$     D. $[\sqrt{2}, +\infty)$
【答案：D】

8. 已知实数 $a, b, c, d$ 满足 $\frac{a}{b} = \frac{c-1}{d-3} = \frac{4}{3}$，则 $(a-c)^2 + (b-d)^2$ 的最小值为 ( )
A. $\frac{121}{25}$
B. $\frac{81}{25}$
C. $\frac{64}{25}$
D. $\frac{49}{25}$
【答案：B】

9. 椭圆 $C: \frac{x^2}{9} + \frac{y^2}{5} = 1$ 的左、右焦点分别为 $F_1, F_2$，点 $P$ 为 $C$ 上的任意一点，则下列说法错误的是 ( )
A. 椭圆 $C$ 的长轴长为 3     B. 椭圆 $C$ 的离心率为 $\frac{2}{3}$     C. $|PF_1|$ 的最大值为 5     D. 存在点 $P$，使得 $PF_1 \perp PF_2$
【答案：BC】

10. 下列命题错误的是 ( )
A. 任意一条直线有且只有一个倾斜角     B. 当直线平行于 $x$ 轴时，直线的倾斜角是 $180^\circ$     C. 直线 $y - y_0 = k(x - x_0)$ 可以表示所有直线     D. 直线 $Ax + By + C = 0$（其中 $A, B$ 不同时为 0）可以表示所有直线
【答案：AD】

11. 已知直线 $l: 3x + 2y + m = 0$，圆 $C: x^2 + y^2 + 4x - y + \frac{1}{4} = 0$，则下列说法错误的是 ( )

D. 若 $m > 5$，圆 $C$ 上有且仅有两个点到 $l$ 的距离为 $1$，则 $5 + \sqrt{13} < m < 5 + 3\sqrt{13}$
