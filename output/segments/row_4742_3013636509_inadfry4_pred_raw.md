17. 在平行四边形 $ABCD$ 中，$AB = 4, AD = 6, \angle BAD = \frac{\pi}{3}, F$ 是线段 $AD$ 的中点，点 $E$ 在直线 $DC$ 上，且 $\overrightarrow{DE} = \lambda \overrightarrow{DC} (-1 \leq \lambda \leq 1)$.
(1) 当 $\lambda = \frac{1}{3}$ 时，求 $\overrightarrow{AE} \cdot \overrightarrow{BF}$ 的值；
(2) 当 $\lambda = \frac{1}{2}$ 时，$AE$ 与 $BF$ 交于点 $N$, $\overrightarrow{AN} = x \overrightarrow{AB} + y \overrightarrow{AD}$，求 $x - y$ 的值.

18. 在 $\triangle ABC$ 中，角 $A, B, C$ 所对的边分别为 $a, b, c$, $2a - c = 2b\cos C$.
(1) 求 $B$;
(2) 若点 $D$ 为边 $BC$ 的中点，点 $E, F$ 分别在边 $AB, AC$ (包括顶点) 上，$\angle EDF = \frac{\pi}{6}, b = c = 2$.
设 $\angle BDE = \alpha$，将 $\triangle DEF$ 的面积 $S$ 表示为 $\alpha$ 的函数，并求 $S$ 的取值范围.

(1) 证明：$SE \perp$ 平面 $AED$;
(2) 若 $AE = \sqrt{3}$，求二面角 $E-SA-D$ 的余弦值;
