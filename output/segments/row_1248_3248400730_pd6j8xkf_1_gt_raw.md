一、选择题

9. 下列结论正确的是 ( )
A. 命题“若 $x=5$，则 $x^2-6x+5=0$”为真命题
B. “$x=5$”是“$x^2-6x+5=0$”的充分不必要条件
C. 已知命题 $P$：“若 $m \leq 0$，则方程 $x^2+x-m=0$ 有实数根”，则命题 $P$ 的否定为真命题
D. 命题“若 $m^2+n^2=0$，则 $m=0$ 且 $n=0$”为真命题
【答案：ABD】

10. 已知全集 $U=\{1,2,3,4,5,6\}$，集合 $A \subseteq U, B \subseteq U$，且满足：$A \cap B = \{3,5\}, (\complement_U A) \cap (\complement_U B) = \{2,4\}$，则下列说法正确的为 ( )
A. $4 \in A$
B. $6 \in A \cup B$
C. 集合 $A$ 可能是 $\{1,3,5,6\}$
D. $(\complement_U A) \cup (\complement_U B) = \{1,2,4,6\}$
【答案：BC】

11. 对于集合 $M = \{a | a = x^2 - y^2, x \in \mathbb{Z}, y \in \mathbb{Z}\}$，则下面结论正确有 ( )
A. 如果 $P = \{b | b = 2n + 1, n \in \mathbb{Z}\}$，那么 $P \subseteq M$
B. 如果 $c = 4n + 2, n \in \mathbb{Z}$，那么 $c \in M$
C. 如果 $a_1 \in M, a_2 \in M$，那么 $a_1 a_2 \in M$
D. 如果 $a_1 \in M, a_2 \in M$，那么 $a_1 + a_2 \in M$
【答案：AC】

二、填空题
12. 设 $\alpha: -1 \leq x < 3, \beta: x < a$，若 $\alpha$ 是 $\beta$ 的充分条件，则实数 $a$ 的取值范围是 ______。
【答案：$a \geq 3$】

13. 若命题 $p$: “$\exists x \in \mathbb{R}, x^2 + 2ax + 3a < 0$”是假命题，命题 $q$: “$\forall x \geq 0, x + a \geq 2$”是真命题，则实数 $a$ 的取值范围是 ______。
【答案：$[2, 3]$】

14. 已知集合 $A = \{a, b, c, d\}, B = \{a^2, b^2, c^2, d^2\}$，其中 $A \subset \mathbb{N}^*, B \subset \mathbb{N}^*, a < b < c < d, A \cap B = \{a, d\}$，$A \cup B$ 中元素之和为 124，且 $a + d = 10$，用列举法写出集合 $B = $ ______。
【答案：$\{1, 9, 25, 81\}$】
