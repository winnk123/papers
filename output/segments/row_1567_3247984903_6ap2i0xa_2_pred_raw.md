[条件缺失]

7. 条件 $p: x^2 - 4x - 5 < 0$ 是条件 $q: x^2 + 6x + 5 \geq 0$ 的 ( )
A. 充分不必要条件
B. 必要不充分条件
C. 充要条件
D. 非充分又非必要条件
【答案：D】

8. 若正数 $x, y$ 满足 $xy - 2x - y = 0$，则 $x + \frac{y}{x}$ 的最小值是 ( )
A. $2\sqrt{2}$
B. $4$
C. $4\sqrt{2}$
【答案：C】

9. 下列与集合 $M = \left\{(x,y) \middle| \begin{cases} x+y=1 \\ x-y-3=0 \end{cases} \right\}$ 表示同一个集合的有 ( )
A. $\{(2,-1)\}$
B. $\{2,-1\}$
C. $\{(x,y) | x=2, y=-1\}$
D. $\{x=2, y=-1\}$
【答案：C】

10. 不等式 $ax^2 + bx + c \geq 0$ 的解集是 $\{x | -1 \leq x \leq 2\}$，则下列结论正确的是 ( )
A. $b < 0$
B. $a+b+c > 0$
C. $c < 0$
D. $a+b=0$
【答案：D】

11. (多选) 下列命题为真命题的是 ( )
A. “$A \cap B \neq \varnothing$”是“A ⊆ B”的必要不充分条件
B. “$x > \sqrt{2}$”是“$x > 1$”的充分不必要条件
C. “$m \neq 0$”是“$mn \neq 0$”的充分条件
D. “$a^2 + b^2 + c^2 = ab + bc + ac$”是“$a=b=c$”的充要条件
【答案：B D】

12. 下列命题为真命题的是 ( )
A. 若 $a^2 + b^2 = 2$，则 $a+b$ 的最大值为 $\sqrt{2}$；
B. 若 $0 \leq x+y \leq 2$, $-1 \leq x-y \leq 1$，则 $-\frac{3}{2} \leq 2x+y \leq \frac{9}{2}$；
C. 不等式 $\frac{1-2x}{3x+1} > 1$ 的解集是 $\left\{x \middle| \frac{1}{3} < x < 0\right\}$；
D. 当且仅当 $a, b$ 均为正数时，$\frac{a+b}{ab} \geq 2$ 恒成立。
【答案：D】

13. 已知集合 $A = \{1, 3, 4m-3\}, B = \{3, m^2\}$，若 $B \subseteq A$，则实数 $m=$ ______
【答案：1 或 -1】

14. 已知“$\exists x \in (0,1], mx^2 - x > 0$”为假命题，则 $m$ 的最大值为 ______
【答案：0】
