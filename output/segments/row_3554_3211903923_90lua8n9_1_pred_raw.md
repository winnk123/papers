1. $i$ 为虚数单位，复数 $z=i(i+2)$ 在复平面内对应的点在 ( )  
A. 第一象限   B. 第二象限   C. 第三象限   D. 第四象限  
【答案：B】  

2. 已知 $f(x)=\lg \frac{x-b}{1+x}, b\in \mathbb{R}$，则“$b=1$”是“$f(x)$ 是奇函数”的 ( )  
A. 充分不必要条件   B. 必要不充分条件  
C. 充分必要条件   D. 既不充分也不必要条件  
【答案：A】  

3. 在 $\triangle ABC$ 中，点 $D$ 满足 $\overrightarrow{CD}=\frac{1}{2}(\overrightarrow{CA}+\overrightarrow{CB})$，且 $AB=2CD$，则 $\angle ACB=$ ( )  
A. $\frac{\pi}{6}$   B. $\frac{\pi}{4}$   C. $\frac{\pi}{3}$   D. $\frac{\pi}{2}$  
【答案：C】  

4. 已知 $\frac{\cos \alpha}{2\cos \alpha+\sin \alpha}=\frac{1}{4}$，则 $\tan \left(\alpha+\frac{\pi}{4}\right)=$ ( )  
A. -3   B. -5   C. 5   D. 3  
【答案：B】  

5. 已知某圆锥放置于半径为 $\sqrt{2}$ 的球 $O$ 内，当该圆锥的体积取得最大值时，该圆锥的高为 ( )  
A. $\frac{4\sqrt{2}}{3}$   B. $\sqrt{2}$   C. $\frac{2\sqrt{2}}{3}$   D. $\frac{1}{2}$  
【答案：C】  

6. 定义在 $\mathbb{R}$ 上的偶函数 $f(x)$ 满足 $f(x+4)=f(x)$，且 $x\in (-2,0)$ 时，$f(x)=2^x-\frac{3}{5}$，则 $f(20+\log_{2}25)=$ ( )  
A. $\frac{1}{25}$   B. $-\frac{1}{5}$   C. $\frac{7}{15}$   D. $\frac{5}{9}$  
【答案：B】  

7. 现有某种细胞 1 千个，其中约有占总数一半的细胞每小时分裂一次，即由 1 个细胞分裂成 2 个细胞，按这种规律，1 小时后，细胞总数约为 $\frac{1}{2}\times1000+\frac{1}{2}\times1000\times2=\frac{3}{2}\times1000$，2 小时后，细胞总数约为 $\frac{1}{2}\times\frac{3}{2}\times1000+\frac{1}{2}\times\frac{3}{2}\times1000\times2=\frac{9}{4}\times1000$，问当细胞总数超过 $10^{18}$ 个时，所需时间至少为 ( ) （参考数据：$\lg 3=0.477$, $\lg 2=0.301$）  
A. 36 小时   B. 38 小时   C. 40 小时   D. 42 小时  
【答案：C】  

8. 已知正三棱锥 $A-BCD$ 的底面边长为 6，二面角 $A-BC-D$ 的余弦值为 $\frac{\sqrt{3}}{4}$，则正三棱锥 $A-BCD$ 外接球的表面积为 ( )  
A. $\frac{256\sqrt{13}}{13}\pi$   B. $\frac{625\sqrt{13}}{13}\pi$   C. $\frac{625}{13}\pi$   D. $\frac{256}{13}\pi$  
【答案：B】
