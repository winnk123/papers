[条件缺失]

9. 已知 $a \in \mathbb{R}$, 集合 $M = \{x | a < x < a + 2\}, N = \{x | x < 0 \text{ 或 } x > 1\}$, 则下列结论中正确的是 ( ) 【答案：BCD】
A. 若 $a = 0$, 则 $M \cap N = \{x | 1 < x < 2\}$
B. 若 $a \geqslant 1$, 则 $M \subseteq N$
C. 若 $M \cup N = \mathbb{R}$, 则 $-1 \leqslant a \leqslant 0$
D. 不存在实数 $a$, 使得 $M \cap N = \varnothing$

10. 下列说法正确的是 ( ) 【答案：BC】
A. 若关于 $x$ 的不等式 $(mx - 1)(x - 2) > 0$ 的解集为 $\left(\frac{1}{m}, 2\right)$, 则 $m > 2$
B. 若函数 $f(x)$ 和 $g(x)$ 在区间 $[a,b]$ 上均单调递增, 则函数 $f(x) \cdot g(x)$ 在区间 $[a,b]$ 上不一定单调递增
C. 已知直线 $y = x + m$ ($m \in \mathbb{R}$) 是曲线 $f(x) = e^{x-n}$ 的一条切线, 则 $m + n = 1$
D. 若定义在 $\mathbb{R}$ 上的偶函数 $f(x)$ 满足 $f(x+2) = \frac{1}{x^2}$, 且当 $x \in [-2, 0]$ 时, $f(x) = \frac{1}{x}$, 则 $f(\log_2 25) = 25$

11. 已知函数 $f(x) = x e^x$, $g(x) = x \ln x$, 则下列说法正确的是 ( ) 【答案：ACD】
A. 函数 $f(x), g(x)$ 有相同的极小值
B. 若方程 $g(x) - m = 0$ 有唯一的实根, 则 $m$ 的取值范围为 $[0, +\infty)$
C. 若 $x_1 > 0$, $x_2 > 0$, $f(x_1) = g(x_2) = t$, 则 $x_1 x_2 = t$
D. 当 $a > 1$ 时, 不等式 $\frac{g(x)}{e^x} \leqslant -x + a$ 恒成立

12. 若函数 $f(x) = \ln(ax - 6)$ 在区间 $[1,3]$ 上单调递增, 则实数 $a$ 的取值范围是 ______ 【答案：$(2, +\infty)$】

13. 已知正实数 $x, y$ 满足 $x + y = 2$, 则 $\frac{y}{x} + \frac{1}{y}$ 的最小值为 ______ 【答案：$\frac{5}{2}$】

14. 若函数 $f(x) = \begin{cases} x e^x, & x \geqslant 1 \\ a - 2x, & x < -1 \end{cases}$ 有且仅有两个零点, 则实数 $a$ 的取值范围为 ______ 【答案：$(-\infty, 0)$】
