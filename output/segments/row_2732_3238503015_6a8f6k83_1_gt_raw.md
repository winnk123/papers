

一、单项选择题 (本题共8小题, 每小题5分, 共40分. 在每小题给出的四个选项中, 只有一项是符合题目要求的)

1. 不等式 $(x+3)^2 < 1$ 的解集是 (C)
   [A] $\{x | x > -2\}$
   [B] $\{x | x < -4\}$
   [C] $\{x | -4 < x < -2\}$
   [D] $\{x | -4 \leq x \leq -2\}$

2. 设 $a > 0, b > 0$, $A = \sqrt{a} + \sqrt{b}$, $B = \sqrt{a + b}$, 则 $A, B$ 的大小关系是 (B)
   [A] $A \leq B$
   [B] $A > B$
   [C] $A < B$
   [D] $A = B$

3. 若 $a > b$, 则下列不等式成立的是 (B)
   [A] $a^2 > b^2$
   [B] $a - b > 0$
   [C] $|a| > |b|$
   [D] $a + b > 0$

4. 若 $0 < a < b$, 则下列不等式成立的是 (C)
   [A] $\sqrt{ab} < a < \frac{a+b}{2} < b$
   [B] $\sqrt{ab} < \frac{a+b}{2} < a < b$
   [C] $a < \sqrt{ab} < \frac{a+b}{2} < b$
   [D] $a < \frac{a+b}{2} < \sqrt{ab} < b$

5. 已知不等式 $ax^2 + bx + 1 > 0$ 的解集为 $\left\{x \middle| -\frac{1}{2} < x < \frac{1}{3}\right\}$, 则不等式 $x^2 - bx - a \geq 0$ 的解集为 (C)
   [A] $\{x | 2 \leq x \leq 3\}$
   [B] $\{x | -3 \leq x \leq -2\}$
   [C] $\{x | x \leq -3 \text{ 或 } x \geq -2\}$
   [D] $\{x | x \leq -2 \text{ 或 } x \geq 3\}$

6. 已知 $x > 0, y > 0$, 且 $2x + y = 1$, 若 $\frac{m}{xy} \leq x + 2y$ 恒成立, 则实数 $m$ 的取值范围为 (C)
   [A] $m < 1$
   [B] $m > \frac{9}{2}$
   [C] $m \leq 1$ 或 $m \geq \frac{9}{8}$
   [D] $m < 1$ 或 $m > \frac{9}{8}$

7. 若 $a, b$ 为正实数, 且 $\frac{1}{a} + \frac{1}{b} + \frac{1}{2a + 4b} = 1$, 则 $a + b$ 的最小值为 (C)
   [A] $\frac{4}{5}$
   [B] $\frac{2}{3}$
   [C] 1
   [D] 2

二、多项选择题 (本题共3小题, 每小题6分, 共18分. 在每小题给出的选项中, 有多项符合题目要求. 全部选对的得6分, 部分选对的得部分分, 有选错的得0分)

8. 已知关于不等式 $x^2 + (a+1)x + a \leq 0$ 成立的任意一个 $x$, 都满足不等式 $3x - 1 \leq 0$, 则实数 $a$ 的取值范围为 (C)
   [A] $\left\{a \middle| a \leq -\frac{1}{3}\right\}$
   [B] $\left\{a \middle| a \geq -\frac{1}{3}\right\}$
   [C] $\left\{a \middle| a \leq -\frac{1}{3}\right\}$
   [D] $\left\{a \middle| a \geq \frac{1}{3}\right\}$

9. 已知 $1 \leq a \leq 2, 2 \leq b \leq 3$, 则下列说法正确的是 (AC)
   [A] $4 \leq a + b \leq 5$
   [B] $2 \leq \frac{a}{b} \leq \frac{2}{3}$
   [C] $3 \leq ab \leq 10$
   [D] $-2 \leq a - b \leq 1$

10. 已知实数 $a, b$ 满足 $(a-1)(b-1) = 1$, 则下列选项正确的是 (BCD)
    [A] $\frac{1}{a} + \frac{1}{b} = 1$
    [B] $\frac{1}{a} + \frac{1}{b} \geq 2$
    [C] $a + b \geq 4$
    [D] $a^2 + b^2 \geq 8$

三、填空题 (本题共3小题, 每小题5分, 共15分)

11. 已知关于 $x$ 的不等式 $x^2 - 4ax + 3a^2 < 0 (a < 0)$ 的解集为 $\{x | x_1 < x < x_2\}$, 则不等式 $x^2 - 4ax + 3a^2 < 0 (a < 0)$ 的解集为 $\{x | 3a < x < a\}$

    [B] $x_1 + x_2 + x_1x_2$ 的最小值为 $\frac{4}{3}$
    [C] $x_1 + x_2 + x_1x_2$ 的最小值为 $-\frac{4}{3}$

12. 若命题“$\exists x \in \mathbb{R}, 4x^2 - 2x + m - 9 = 0$”为假命题, 则实数 $m$ 的取值范围为 $m > 9$。

13. 已知 $0 < a < 1$, 则函数 $f(x) = \frac{a^x - 1}{a^x + 1}$ 的最小值是 $-1$。
