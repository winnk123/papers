一、选择题

9. 已知全集 $U = A \cup B$，集合 $A = \{1, 3, 4\}$，$B = \left\{x \in \mathbb{N} \middle| \frac{6}{x} \in \mathbb{N}\right\}$，则 ( ) 【答案：C】
A. $2 \in B$  
B. $A \cap B = \{1, 3\}$  
C. $U$ 中的元素个数为 5  
D. 若集合 $C = \{x \mid x \text{ 是质数}\}$，则 $C \cap B$ 的子集有 8 个

10. 已知不等式 $ax^2 + bx + c > 0$ 的解集是 $\{x \mid -2 < x < 3\}$，则下列说法正确的是 ( ) 【答案：BD】
A. $a > 0$  
B. $a + b + c > 0$  
C. 不等式 $ax^2 + bx + a < 0$ 的解集是 $\left\{x \middle| -\frac{1}{3} < x < \frac{1}{2}\right\}$  
D. 当 $c = 6$ 时，关于 $x$ 的不等式 $\frac{(x+a)(x+b)}{x+c} > 0$ 的解集为 $\{x \mid -6 < x < -1 \text{ 或 } x > 1\}$

11. 已知 $a, b \in \mathbb{R}, a > 1 > b > 0$，则 ( ) 【答案：AB】

B. $\sqrt{a^2 + b} > \sqrt{a + b^2}$  
C. $a + \frac{1}{a - b} > 2$  
D. $\left(\frac{a-1}{b} > \frac{1-b}{a}\right)$

二、填空题

12. 已知 $p: |x - 1| < 1$，$q: x < a$，若 $p$ 是 $q$ 的充分不必要条件，则实数 $a$ 的最小值为 ______ 【答案：$2$】

13. 若 $a > 0, b > 0$ 且 $a + \frac{1}{b} = 1$，则 $b + \frac{4}{a}$ 的最小值为 ______ 【答案：$5$】

14. 已知 $m \in \mathbb{R}$，集合 $A = \{x \mid m < x < 3m - 1\}$ 中的元素恰有 2 个整数，则 $m$ 的取值范围是 ______ 【答案：$\frac{4}{3} \leq m \leq 2$】
